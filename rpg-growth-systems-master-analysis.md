### 핵심 성장 레이어 구조 (공통 패턴)

```
[가챠 RPG 표준 성장 스택]
레이어 7: 중복 캐릭터 가치 시스템 (별자리/성흔/공명쇄)     ← 과금 최상위
레이어 6: 장비 옵션 최적화 (룬/유물/에코/디스크 부옵션 RNG)  ← 엔드게임
레이어 5: 장비 세트/티어 강화                               ← 중반~후반
레이어 4: 스킬 레벨업 (요일 던전 파밍)                      ← 중반
레이어 3: 돌파/각성/인사이트 (레벨캡 해제)                  ← 초중반
레이어 2: 전용 무기/광추/W엔진 강화                         ← 중반
레이어 1: 기본 레벨링                                       ← 초반
```

### 가챠 RPG 성장 시스템 3대 혁신

**① 서머너즈 워 룬 시스템 (2014)**
- 세트 효과 + 주옵/부옵 랜덤이라는 표준 공식을 업계 최초 확립
- 이후 에픽세븐, Dislyte, 스타레일, ZZZ까지 모두 이 공식의 변주
- "파밍의 목적 = 완벽한 세트 + 완벽한 옵션"이라는 엔드게임 구조의 원형

**② FEH 스킬 계승 (2017)**
- "캐릭터 고정 스킬"에서 "자유 커스터마이징"으로의 패러다임 전환
- 이후 수많은 게임이 유사한 빌드 커스터마이징 시스템을 도입
- "같은 캐릭터인데 내 빌드가 남과 다르다"는 소유감의 혁신

**③ AFK아레나 공명수정 → 니케 싱크로 디바이스 (2019~2022)**
- "상위 N인 레벨 = 전체 팀 레벨"이라는 동기화 시스템
- 신캐릭터를 즉시 전투에 투입할 수 있는 편의성
- "내가 투자한 모든 캐릭터가 팀에 기여한다"는 수집 만족감

## 1-2. 방치형/Idle RPG (22종 분석)

### 5세대 진화 흐름

| 세대 | 기간 | 대표 게임 | 성장 구조 | 누적 매출 |
|------|------|-----------|-----------|-----------|
| 1세대 | 2014~2015 | Clicker Heroes, Tap Titans, 중년기사 김봉식 | 탭+영웅+프레스티지 (3~4층) | 장르 원형 확립 |
| 2세대 | 2016~2017 | Idle Heroes, TT2, Grow Castle | 합성/강화+스킬트리+가챠 (5~6층) | Idle Heroes 첫 해 $100M+ |
| 3세대 | 2019 | AFK Arena, MLA, Ulala | 시간 게이트+어센션+콘텐츠 해금 (7층+) | AFK Arena 누적 $14.5억 |
| 4세대 | 2021~2022 | Cookie Run Kingdom, Survivor.io, Blade Idle | 장르 하이브리드화 | Survivor.io 누적 $500M+ |
| 5세대 | 2023~2024 | Legend of Mushroom, Capybara Go, AFK Journey | 극단적 단순화 OR 극단적 복잡화 | Legend of Mushroom 6개월 $270M |

### 프레스티지/환생 시스템의 진화

| 시기 | 방식 | 대표 | 특징 |
|------|------|------|------|
| 2014 | 전면 리셋 + 아티팩트 | Clicker Heroes, Tap Titans | 리셋의 공포가 핵심 긴장감 |
| 2016 | 합성 게이트 | Idle Heroes | 프레스티지 없이 "합성 벽"이 성장 게이트 |
| 2019 | 시간 게이트 | AFK Arena | 과금으로 살 수 없는 시간 자원이 핵심 |
| 2024 | 시즌 리셋 | AFK Journey | 4개월 주기 강제 리셋으로 파워 인플레이션 제어 |
| 2023 | 리셋 없음 | Legend of Mushroom | 단순 장비 루프로 프레스티지 자체를 제거 |

### 방치형 RPG의 성공 공식

**AFK Arena 모델 (3세대 황금 공식)**
```
핵심 원리: "과금으로 살 수 없는 자원 = 시간"
- 오프라인 방치 보상이 유일한 성장 재화
- 어센션 → 공명수정 → 시그니처아이템 → 가구 순차 해금
- 각 단계가 해금될 때마다 새로운 목표 제시
- 결과: DAU 100만, 누적 $14.5억
```

**Legend of Mushroom 모델 (5세대 역발상)**
```
핵심 원리: "극도의 단순함도 실행이 완벽하면 승리"
- 매직 램프 → 장비 드롭 → 장착/판매 의 단일 루프
- 복잡한 다층 구조 없이 단 4개 레이어
- 결과: 6개월 $270M+
```

## 1-3. 액션/로그라이트 RPG (23종 분석)

### 메타 프로그레션 설계의 4세대 진화

| 세대 | 기간 | 방식 | 대표 |
|------|------|------|------|
| 1세대 | 2016~2018 | 스탯 인플레이션 | Honkai Impact 3rd, Soul Knight |
| 2세대 | 2019~2020 | 옵션 확장형 | Archero, Dead Cells, Hades |
| 3세대 | 2021~2023 | 이중 루프 (런 내 기하급수 + 런 간 미세 성장) | Vampire Survivors, Survivor.io |
| 4세대 | 2022~2024 | 복합 레이어 + 라이브 서비스 | Diablo Immortal, Torchlight Infinite |

### 런 기반 vs 영구 성장 밸런스 스펙트럼

```
[퓨어 로그라이크] ←——————————————————→ [퓨어 RPG]
  Brotato    Dead Cells   Hades   Vampire S.   Archero   Survivor.io   Diablo I.   Honkai 3rd
  (메타 무)  (옵션 확장)  (커스텀)  (미세 스탯)  (장비+탤런트) (장비 중심)  (파라곤+젬)  (가챠+강화)
```

**최적 밸런스 4대 원칙**
1. **"제로 진행 런 제거"**: 실패한 런에서도 골드/해금/업적 중 최소 하나
2. **"메타 < 런 내 선택"**: 메타 파워가 런 내 빌드 선택보다 임팩트가 작아야 함
3. **"옵션 확장 > 스탯 인플레이션"**: 새 무기/스킬/경로 해금이 직접 스탯 증가보다 건강
4. **"시작점 미세 조정"**: 메타 증가분은 미미하지만 누적은 유의미

### 모바일 로그라이트 세션 최적화

| 세션 길이 | 대표 | 결과 |
|-----------|------|------|
| 5~10분 | Archero | 출퇴근 최적화. 누적 $2.6억 |
| 10~15분 | Survivor.io, Soul Knight | 모바일 골든 타임. Survivor.io $500M+ |
| 15~20분 | 20 Minutes Till Dawn | 집중 세션. 인디 히트 |
| 20~30분 | Vampire Survivors | PC/모바일 겸용 성공 |

**핵심 결론**: 10~15분이 모바일 로그라이트의 스위트 스팟. 이 범위 내에서 "런 시작 → 파워 성장 → 클라이맥스 → 보상"의 완전한 감정 아크 완성.

### MMO 성장 시스템 10년 변화

**2017~2019 (단순 이식기)**
- PC MMO를 모바일에 그대로 이식
- 리니지M이 "변신/마법인형"이라는 모바일 전용 성장 축 추가
- 확률형 강화 + 장비 파괴가 표준 모델

**2020~2021 (레이어 팽창기)**
- 8~12개 레이어로 급증
- "성장할 것이 많다 = 컨텐츠가 풍부하다"는 마케팅 논리
- 미르4의 Play to Earn 실험

**2022~2024 (반성과 전환기)**
- 로스트아크: 초월/엘릭서 삭제 결정 → **"레이어 과잉의 폐해" 공식 인정**
- 천애명월도M, 블소 NEO: 의도적 간소화
- 아이온2: "확률형 아이템 미판매" 선언
- MMO 비중 하락 → 방치형, 서브컬처가 매출 잠식

## 트렌드 1. 성장 레이어의 역U자 곡선

```
레이어 수
   10│              ╭─────╮
    9│           ╭─╯ MMO  ╰─╮
    8│        ╭─╯  가챠RPG   ╰─╮
    7│     ╭─╯                  ╰───
    6│  ╭─╯
    5│─╯
    4│
    3│
     └────────────────────────────→ 연도
     2014  2016  2018  2020  2022  2024
```

- **2014~2015**: 기본 구조 확립 (3~6층)
- **2018~2021**: 레이어 팽창 극대화 (7~12층)
- **2022~2024**: 반성과 수렴 (5~7층으로 수렴)

**핵심 데이터**: 로스트아크가 2024년 초월/엘릭서 삭제 결정 → "레이어 과잉은 결국 삭제"라는 업계 학습

---

## 트렌드 2. 실패해도 보상받는 시스템의 보편화

모든 장르에서 "완전한 실패"를 제거하는 방향으로 진화:

| 장르 | 방식 | 진화 방향 |
|------|------|-----------|
| 가챠 RPG | 가챠 천장 시스템 (보장 확률) | 없음 → 소프트 천장 → 하드 천장 |
| 방치형 | 프레스티지 영구 보상 | 전면 리셋 → 아티팩트 보상 → 일부만 리셋 |
| 로그라이트 | "제로 진행 런 제거" | 런 실패 = 아무것도 없음 → 항상 재화/해금 |
| MMO | 아티산 에너지 (실패 누적 → 확정) | 파괴형 → 유지형 → 천장형 |
| 전술 RPG | 관대한 가챠/무과금 파밍 | 가챠 의존 → 무과금 파밍 대안 제공 |

## 트렌드 3. 랜덤 장비 파밍 = 엔드게임의 불변 법칙

서머너즈 워(2014) 룬 시스템 이후 10년간, **모든 RPG 장르에서 랜덤 옵션 장비 파밍이 엔드게임의 핵심**으로 자리잡음:

```
[계보 추적]
서머너즈 워 룬 (2014)
   ├→ 에픽세븐 장비 (2018)
   ├→ Dislyte 유물 (2022)
   ├→ RAID Shadow Legends 장비 (2018)
   └→ 원신 성유물 (2020)
         ├→ 붕괴: 스타레일 유물 (2023)
         └→ ZZZ 디스크 드라이브 (2024)
              └→ 명조 에코 (2024)
```

**왜 이 시스템이 10년간 살아남았는가?**
1. 무한한 목표: "완벽한 옵션"은 영원히 추구 가능
2. 간헐적 강화 스케줄: 좋은 옵션이 나왔을 때의 쾌감이 지속 동기 부여
3. 과금 연계: 스태미나 구매, 즉시 파밍으로 자연스러운 과금 유도
4. 커뮤니티: "내 장비 스펙" 공유 문화로 소셜 참여 유도

## 트렌드 4. 성장 시스템의 목적 분화

**초기 (2014~2017)**: 성장 = 숫자 올리기 (수직 파워 인플레이션)

**현재 (2022~2024)**: 성장의 목적이 4가지로 분화

| 목적 | 설명 | 대표 |
|------|------|------|
| 파워 증가 | 전통적 수직 성장 | MMO RPG, 가챠 RPG |
| 선택지 확대 | 빌드 다양성 | Hades, FEH, Arknights |
| 감정적 유대 | "좋아하는 캐릭터를 키우는" 경험 | FGO, 블루아카이브, Eversoul |
| 컨텐츠 접근권 | 다음 스테이지/레이드를 위한 스탯 | MMO, 수집형 RPG |

## 트렌드 5. 세션 길이와 성장 시스템 복잡도의 상관관계

```
성장 레이어 복잡도
    높음 │                        ●Diablo I.
        │                   ●로스트아크
        │            ●RAID   ●검은사막M
        │       ●에픽세븐  ●원신
        │   ●AFK Arena  ●스타레일
        │●명일방주  ●Survivor.io
        │●Archero
낮음    │●Brotato  ●VS
        └───────────────────────→ 평균 세션 길이
         3분    10분   20분   30분+
```

**법칙**: 세션 길이가 길수록 복잡한 성장 시스템이 허용됨. 모바일에서 10분 이하 세션을 타겟하면 성장 시스템은 3~4층으로 제한해야 함.

## 트렌드 6. P2W에서 "경험 다양화"로의 전환 (2022~2024)

**한국 MMO의 전환**:
- 리니지식 "파괴형 강화 + 무한 P2W" → 천장형/보장형 강화
- 확률형 아이템 → 구독/패스/코스메틱 중심

**글로벌 가챠 RPG의 전환**:
- 중복 캐릭터 = 분쇄 재료 → 중복 = "별자리/성흔" (파워 아닌 경험 확장)
- "무조건 최강템" → "나만의 빌드"로 다양화

**방치형 RPG의 전환**:
- 과금 = 시간 단축 → 과금 = 시즌 진행 가속 (AFK Journey 시즌 리셋 모델)

# Part 3. 데이터로 본 "좋은 성장 시스템"의 공식

---

## 공식 1: 성장 레이어 최적 수

| 장르 | 최적 레이어 수 | 근거 |
|------|--------------|------|
| 가챠 RPG | 5~7층 | 스타레일/ZZZ 5~7층이 "역대 가장 세련됨" 평가. 에픽세븐 7~8층은 장비 RNG 불만 |
| 방치형 | 4~5층 | Legend of Mushroom 4층 $270M vs AFK Arena 7층+ $14.5억 → 둘 다 성공. 단 단순함이 진입장벽 낮춤 |
| 로그라이트 | 3~4층 | Archero($2.6억)/Survivor.io($5억+) 3~4층이 최적. Diablo I. 5~6층은 하드코어 타겟 |
| MMO | 7~8층 | 5~6층은 깊이 부족. 10층+ 는 반드시 삭제 필요 발생 (로스트아크 사례) |
| 턴제/전술 | 4~5층 | Arknights/NIKKE 4~5층이 $1B+ 달성. WotV 7층+는 신규 진입장벽 높음 |

**범용 결론: 5~7층이 모바일 RPG의 황금 구간**

---

## 공식 2: 반드시 존재해야 하는 4가지 요소

**① 실패 보상 메커니즘 (Anti-Failure)**
- 가챠: 천장/보정 확률
- 강화: 실패 누적 아티산/경험치
- 로그라이트: 런 실패시 재화/해금
- 없으면: 이탈률 급증. 투자 손실감이 이탈의 직접 원인

**② 명확한 다음 목표 (Next Goal Clarity)**
- 유저가 "지금 뭘 해야 하는지" 항상 알아야 함
- AFK Arena: 어센션 → 공명수정 → 시그니처 아이템 → 가구 (단계별 명확한 목표)
- Arknights: 스테이지 클리어 → 승진 재료 파밍 → 마스터리 → 다음 스테이지
- 없으면: "뭘 해야 할지 모르겠다" = 최악의 UX 상태

**③ 장기 목표 + 단기 목표의 이중 구조 (Dual Goal)**
- 장기: 최고 등급 캐릭터 달성, 최강 장비 완성
- 단기: 오늘 일일 퀘스트, 이번 주 이벤트
- Genshin Impact 사례: 장기(전설 캐릭터+최종 무기) + 단기(수지 파밍 + 이번 이벤트)

**④ 성장이 파워만 아닌 선택지를 확장시켜야 함 (Options > Stats)**
- 레벨업 = 스킬 선택지 증가 (로그라이트)
- 강화 = 새 전략 가능 (명일방주 승진 → 새 스킬 특성)
- 장비 완성 = 새 컨텐츠 접근 가능 (MMO 레이드 입장)
- 아닌 경우: "수치만 오르는 성장"은 레벨 60 이후 급격히 보람을 잃음

---

## 공식 3: 반드시 피해야 하는 4가지 함정

**① 성장 레이어 10층 이상 = 삭제 예정 시스템**
- 로스트아크 초월/엘릭서: 2024년 삭제 결정
- 검은사막M 연금석: 신등급 추가 → 기존 투자 가치 폐기 → 유저 분노
- 교훈: 레이어를 추가할 때마다 향후 삭제/정리 비용을 고려해야 함

**② 완전 파괴형 강화 = 장기적 독**
- 리니지2M 장비 파괴: 단기 과금 극대화에 효과적이나 장기적으로 생태계 파괴
- 2024년 이후 신작에서 파괴형 강화를 채택하지 않는 추세
- 교훈: 도박적 재미는 천장/보장형으로 대체 가능

**③ 신규/복귀 유저 진입장벽 방치**
- 서비스 기간이 길어질수록 성장 레이어가 누적됨
- 해결책: 따라잡기 시스템, 스타터 패키지, 캐릭터 동기화 (NIKKE 싱크로)
- 없으면: 코어 유저만 남는 "노후화" 가속

**④ "내 생각엔 재밌을 것 같다" = 근거 없음**
- 성장 시스템의 레이어 추가, 확률 설정, 파밍 효율 등은 반드시 레퍼런스 데이터로 검증
- 유저 리뷰에서 "강화 확률이 너무 낮다", "파밍 효율이 너무 낮다" 같은 피드백은 수치적으로 조정 가능한 신호

## 공식 4: 장르별 성장 시스템 설계 우선순위

### 수집형 가챠 RPG
```
우선순위:
1. 가챠 천장/보정 시스템 (없으면 현대 시장에서 생존 불가)
2. 캐릭터 감정 유대와 성장의 연동 (FGO/블루아카 모델)
3. 랜덤 장비 파밍 (서머너즈 워 룬 계보)
4. 중복 캐릭터 가치 (분쇄→별자리로 진화)
5. 따라잡기/동기화 시스템 (신규 유저 진입 지원)
```

### 방치형 RPG
```
우선순위:
1. 오프라인 방치 보상의 핵심 재화화 (AFK Arena 모델)
2. 명확한 어센션/합성 게이트 (장기 목표)
3. 성장 레이어 순차 해금 (정보 과부하 방지)
4. 단기 이벤트 로테이션 (Capybara Go 성장 이벤트 모델)
5. 극도의 단순함 OR 의미 있는 깊이 (중간은 없음)
```

### 액션/로그라이트 RPG
```
우선순위:
1. "제로 진행 런 없음" 보장 (실패해도 재화/해금)
2. 런 내 빌드 선택 임팩트 > 메타 파워
3. 10~15분 세션 내 완전한 감정 아크
4. 옵션 확장형 메타 (스탯 인플레이션 최소화)
5. 3~4층 성장 레이어로 직관성 유지
```

### MMO RPG
```
우선순위:
1. 천장/보장형 강화 시스템 (실패 누적 보상)
2. P2W 격차 상한선 설정 (무과금 생태계 유지)
3. 7~8층 레이어 고수 (10층 이상 절대 금지)
4. 성장 목적과 연결된 컨텐츠 (레이드/PvP 목표성)
5. 신규 유저 따라잡기 시스템
```

# Part 4. 결론: 10년이 말해주는 7가지 교훈

---

## 교훈 1. "5~7층이 황금 구간이고, 10층 이상은 반드시 후회한다"

- 2014~2021 사이 모든 장르에서 레이어 팽창 실험이 있었음
- 2022~2024 모든 장르에서 "적정 수준으로 수렴" 또는 "과잉 레이어 삭제" 추세
- 로스트아크 초월/엘릭서 삭제는 업계 전체가 배운 교과서적 사례

## 교훈 2. "실패 보상 없는 성장 시스템은 2024년에 출시할 수 없다"

- 천장 시스템, 아티산 에너지, 제로 진행 런 제거, 프레스티지 보상
- 어떤 방식이든 "투자가 보존된다"는 심리적 안전망이 없으면 이탈률 급증
- 로스트아크 아티산 에너지가 2019년 이후 업계 표준으로 자리잡음

## 교훈 3. "랜덤 장비 파밍은 10년 뒤에도 엔드게임의 핵심일 것이다"

- 서머너즈 워(2014) 룬 시스템이 확립한 "랜덤 옵션 파밍" 구조
- 2024년 ZZZ 디스크, 명조 에코까지 동일한 구조
- 간헐적 강화 스케줄 + 무한한 완성 목표 = 영원한 리텐션 구조

## 교훈 4. "단순함도 실행이 완벽하면 복잡한 시스템을 이긴다"

- Legend of Mushroom 4층 → 6개월 $270M
- Brotato 2층 → 인디 히트
- Clash Royale 2층 → $3.3B+
- 복잡한 성장 시스템이 필수가 아님. 단순하되 핵심 루프가 중독적이면 충분

## 교훈 5. "성장 시스템은 파워 증가보다 선택지 확대를 목표로 해야 한다"

- Hades: 옵션 확장형 메타가 수십만 명을 매일 복귀시킴
- FEH: 스킬 계승이 "내 빌드"를 만들게 해서 $1B+ 달성
- Arknights: 강화해도 파워보다 "전략적 선택지"가 넓어지는 설계로 $1.1B+
- 파워 인플레이션이 심하면 "강해도 즐길 것 없다"는 후기 이탈 가속

## 교훈 6. "세션 길이에 맞는 성장 피드백 속도가 있다"

- 5분 세션 게임에서 "한 달 뒤에 강해집니다"는 이탈 유발
- 30분+ 세션 게임에서 "매 세션마다 완성"은 깊이 부족
- 성장 시스템의 복잡도 = 세션 길이에 비례해야 함

## 교훈 7. "P2W 격차는 무과금 생태계가 살아있어야 과금 유저도 살아있다"

- 검은사막M 40:1 무쌍: 무과금 이탈 → 과금 유저도 이탈 (생태계 붕괴)
- 알비온 온라인: 풀룻 PvP로 과금 장비 영구화 방지 → 장기 운영 성공
- 교훈: 무과금 유저는 과금 유저의 "먹잇감"이 아니라 게임 생태계 유지의 필수 요소
