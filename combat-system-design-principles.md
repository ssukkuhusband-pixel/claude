## 원칙 0. 전투의 본질 — "캐주얼 메커닉 + RPG/수집"이 역대 최고 공식

> 모바일 역대 최고 수익 전투 시스템은 전부 "전 세계가 아는 캐주얼 행위를 RPG 전투로 전환한 것"이다.

### 데이터

| 게임 | 캐주얼 메커닉 → 전투 | 누적 수익 |
|------|-------------------|----------|
| Monster Strike | 핀볼/당구 → 몬스터 충돌 | **$11.4B** |
| Puzzle & Dragons | 매치3 퍼즐 → 속성 공격 | **$7B** |
| Clash of Clans | 드롭 배치 → 기지 공격 | **$6B+** |
| Genshin Impact | 오픈월드 액션 → 원소 반응 | **$5B+** |
| Clash Royale | 카드 배치 → 실시간 전략 | **$4.9B** |

**법칙**: "익숙한 조작 + 새로운 맥락(RPG/수집)" = 진입장벽 0 + 깊이 무한.

## 원칙 1. 10초 이해 법칙 — 설명 필요하면 실패

### 데이터

참신함의 임계점을 넘은 전투 vs 못 넘은 전투:

| 구분 | 넘은 전투 | 못 넘은 전투 |
|------|---------|-----------|
| 유저 반응 | "처음 봤는데 바로 이해됨" | "독특한데 XX랑 뭐가 다르지?" |
| 설명 필요 | **10초 이내** | 튜토리얼 필수 |
| 비교 대상 | "비슷한 게임이 없다" | "XX보다 완성도 낮다" |

### 원칙

- 전투 메커닉을 **한 문장으로 설명할 수 없으면** 참신함의 임계점 미달
- 예: "오브를 드래그해서 매칭" (P&D), "캐릭터를 튕겨서 적에게 충돌" (Monster Strike)
- 반례: "중력을 조작하면서 팀 기반 FPS" (LawBreakers) → "뭐?" → 실패

### 체크리스트

```
□ 전투 메커닉을 한 문장으로 설명할 수 있는가?
□ 첫 전투에서 10초 안에 "이렇게 하는 거구나"를 이해시키는가?
□ 유저가 "이런 게임 본 적 없는데 바로 이해됨"이라고 반응할 수준인가?
```

## 원칙 2. 한 손 조작 = 시장 규모 결정

### 데이터

| 조작 스탠스 | 유저 비율 | 설계 대상 |
|-----------|---------|---------|
| 양손 엄지 (게이머) | 소수 코어 | 미드코어/코어 |
| **한 손+다른 손 탭 (캐주얼)** | **대다수** | **시장 극대화** |
| 한 손 엄지 (이동 중) | 이동 중 | 방치형/턴제 |

- Angry Birds, Fruit Ninja, P&D, Monster Strike = 전부 한 손 기본 조작 가능
- 최소 탭 타깃: **44~48px** (Apple HIG / Google Material Design)

### 원칙

- **한 손으로 기본 조작 가능한 전투 = 시장 규모 극대화**
- 양손 필수 = 코어 유저만 남음 (Genshin급 IP/퀄리티가 아니면 위험)
- 정밀 멀티핑거 커맨드(격투게임 236P) = 모바일에서 **구조적 불가능**

### 체크리스트

```
□ 핵심 전투 조작이 한 손(엄지)으로 가능한가?
□ 탭 타깃이 44px 이상인가?
□ 이동 중(지하철)에서도 플레이 가능한가?
```

## 원칙 3. 1전투 = 1~3분, 세션 = 5~8분

### 데이터

- GameAnalytics 2025: 모바일 세션 중앙값 **5~6분**, 상위 25% **8~9분**
- 하루 평균 **4세션**
- 1전투 적정: **1~3분** (턴제는 5분까지 허용)

### 조작 복잡도별 수익 패턴

| 복잡도 | 조작 | 성공 사례 | BM 연결 |
|--------|------|---------|--------|
| 1 (원터치) | 탭/방치 | 쿠키런, 방치RPG | 광고+경량 가챠 |
| 2 (드래그) | 오브 드래그, 스와이프 | P&D, Fruit Ninja | 가챠+스태미나 |
| **3 (탭+편성)** | **턴제+팀 편성** | **SW, AFK Arena** | **가챠+BP+PvP (최적)** |
| 4 (실시간) | 조이스틱+스킬 | Genshin, MOBA | 코스메틱+BP |
| 5 (콘솔급) | 커맨드 입력 | 모바일 사례 없음 | 불가 |

**복잡도 3 = BM 연결 최적 지점**: 캐주얼 접근성 + 전략 깊이 + 가챠/PvP 자연 연결

### 체크리스트

```
□ 1전투가 1~3분 범위인가?
□ 1세션(3~4전투)이 5~8분에 완료되는가?
□ 조작 복잡도가 장르/BM 목표에 맞는가?
□ 전투 중 방해(전화/알림) 시 복귀가 자연스러운가?
```

## 원칙 4. 질리지 않는 전투 = 숙련도 층위의 다층 설계

### 데이터 — 10년+ 서비스 게임의 비밀

| 게임 | 서비스 기간 | 핵심 메커닉 변화 | 질리지 않는 이유 |
|------|-----------|---------------|-------------|
| P&D | 12년+ | 오브 드래그 **불변** | 초보 "최대한 매칭" ↔ 고수 "한 드래그 6콤보" = 숙련도 무한 확장 |
| SW | 10년+ | 턴제 자동 **불변** | 23종 룬 × RNG 서브스탯 = 무한 최적화 목표 |
| CoC | 12년+ | 드롭 배치 **불변** | 배치 전략 깊이 + PvP 메타 순환 |
| Monster Strike | 11년+ | 핀볼 **불변** | 벽 반사 궤적 숙련 + 멀티플레이 변수 |

### 원칙

- **핵심 입력 메커닉은 10년 불변** — 바꾸는 게 아니라 주변을 확장
- **같은 메커닉에서 숙련도 층위를 다층 설계** — 초보와 고수가 같은 전투에서 다른 경험
- 깊이는 **팀빌딩, 장비, PvP 메타, 시너지**에서 확장

### 체크리스트

```
□ 초보와 고수가 같은 전투에서 다른 경험을 하는가?
□ 숙련도 올라갈수록 전략 선택지가 넓어지는가?
□ 핵심 메커닉을 바꾸지 않고도 깊이를 확장할 수 있는 구조인가?
```

## 원칙 5. 시각/청각 피드백(Juice) > 밸런스

### 데이터

- **타격감 > 전투 밸런스** (체감 재미 기준)
- 크리티컬 타격음 → 도파민 분비 → 반복 동기 강화
- Vampire Survivors의 성공: 밸런스 엉망이지만 화면 가득 이펙트 = $57M+

### 원칙

- 전투의 "느낌"이 전투의 "균형"보다 유저 만족에 더 큰 영향
- 모든 공격에 **시각 이펙트 + 타격음 + 화면 흔들림** 최소 3가지 피드백
- 큰 데미지 = 큰 숫자 + 큰 이펙트 + 큰 사운드 (과장이 정답)
- **"미쳤다!" 순간의 피드백**: 화면 가득 이펙트, 슬로우 모션, 특수 사운드

### 체크리스트

```
□ 모든 공격에 시각+청각 피드백이 있는가?
□ 크리티컬/대형 데미지에 차별화된 과장 피드백이 있는가?
□ 전투가 "소리를 끄면 재미가 반감되는" 수준인가?
```

## 원칙 6. 전투와 BM의 자연스러운 연결 — 3축 설계

### 3축

```
① 전력 판매: 강한 캐릭터/장비/룬 → 가챠
② 시간 판매: 스태미나/소탕/배속 → 편의 과금
③ 편의 판매: 슬롯 확장/자동 품질 → 구독/패스
```

### 원칙

- 전투에서 **"이 캐릭터/장비가 확실히 다르다"는 체감**이 있어야 가챠 동기 발생
- 체감 없이 수치만 다르면 → "아무거나 써도 똑같네" → 과금 동기 소멸
- **오토 전투는 BM 도구**: 배속 과금, 소탕 티켓 제한, 패스에 배속 포함 = 자연스러운 수익
- 전략적 도전(첫 클리어/PvP/보스)에서는 오토 OFF 유도 → 전투 깊이 유지

### 체크리스트

```
□ 전투에서 캐릭터/장비 차이가 체감되는가?
□ BM 3축(전력/시간/편의) 중 최소 2개가 전투와 연결되는가?
□ 오토 전투와 수동 전투의 역할이 분리되어 있는가?
```

## 원칙 9. 전투 수명 — 핵심 불변, 주변 진화, 소셜 필수

### 10년+ 서비스 공통 패턴

1. **핵심 입력 메커닉 불변** — 오브 드래그는 오브 드래그
2. **깊이는 주변에서 확장** — 팀빌딩, 룬, PvP 메타, 콜라보
3. **소셜/PvP 필수** — D90+ 장기 리텐션은 소셜 없이 불가
4. **초기 투자 가치 보존** — 2014년 캐릭터가 2024년에도 통용

### 체크리스트

```
□ 핵심 메커닉이 10년 뒤에도 변하지 않을 구조인가?
□ 주변 시스템(장비/스킬/PvP)으로 깊이를 확장할 여지가 있는가?
□ 소셜/PvP 레이어가 전투와 연결되어 있는가?
```
