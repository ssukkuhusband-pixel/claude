## 원칙 1. 세션 길이 — 1판 1~3분, 세션 10~20분

### 장르별 적정

| 장르 | 1판 | 세션 총 |
|------|-----|--------|
| 하이퍼캐주얼 | 30초~1분 | 3~5분 |
| 캐주얼 퍼즐 | **1~3분** | 10~20분 |
| 머지 | 2~5분 | 10~20분 |
| 하이브리드 캐주얼 | 5~15분 | 15~30분 |

- 게임 즐거움 피크: 세션 **15~20분** 구간
- 1판은 짧게, 자동 재시작으로 세션 연장

### 레벨/스테이지 수

- 론칭 시 최소 **200~300레벨**
- **5레벨마다 새 요소** 투입 (절대 수보다 다양성이 핵심)
- Candy Crush: 15,850+ 레벨 (주 2~3레벨 추가)

### 체크리스트

```
□ 1판이 1~3분(퍼즐) / 30초~1분(하이퍼) / 5~15분(하이브리드)?
□ 론칭 시 200~300레벨 확보?
□ 5레벨마다 새 요소가 등장하는가?
```

## 원칙 2. 난이도 — 니어 미스(Near Miss)가 전환율의 비밀

### Candy Crush 니어 미스 설계

- 실패 직전 1~2수 부족 상황을 **의도적으로** 설계
- "당첨 직전 슬롯머신"과 동일한 심리 반응
- 이 순간에 "추가 이동 구매" $0.99 제시 → **역대 최고 전환 장치**

### 난이도 패턴 3가지

1. **완만한 상승형** — 초반 200레벨 점진적 (학습)
2. **파도형 (Hard-Easy-Hard)** — 어려운 3개마다 쉬운 1개 → "한판만 더"
3. **벽-돌파형** — 보스 레벨 → 돌파 시 강한 성취감 + BM 연결

### "한 판만 더" 3요소

1. **진행 가시화** — 레벨 진행 바 항상 표시
2. **변수 보상** — 매 판 약간 다른 결과 → 도파민 지속
3. **손실 회피** — 라이프 시스템 → "남은 라이프" 긴장감

### 체크리스트

```
□ 니어 미스가 의도적으로 설계되어 있는가?
□ 난이도 패턴이 "파도형"인가?
□ "한 판만 더" 3요소가 모두 존재하는가?
```

## 원칙 3. BM — 광고 vs IAP는 전략적 선택

### 장르별 수익 구조

| 모델 | 대표 | 수익 | 광고 |
|------|------|------|------|
| **IAP 중심(광고 0)** | Royal Match | **$5.8B** | 0% |
| IAP + 광고 하이브리드 | Solitaire Grand Harvest | $1B | 50% |
| 광고 100% | 하이퍼캐주얼 | $5~20M | 100% |
| 유료 판매 | Mario Run | $60M(실패) | 0% |

### Royal Match "광고 제로" 전략

- 광고 없음 → 유저 경험 품질 상승 → 고ARPU 유저 유입
- RpD(다운로드당 수익) **$55** — 광고 수익 포기 대신 IAP로 교환
- 매치3 전체 IAP의 **51% 독점** (2024)

### Candy Crush "추가 이동" 판매 5단계

1. 투자 심리 (5분 투자)
2. 니어 미스 (2~3수 부족)
3. 합리적 가격 ($0.99)
4. 손실 프레이밍 ("진행 잃고 싶지 않다")
5. 즉각 해결 (구매 즉시 클리어)

### 하이퍼캐주얼 ATT 이후 (법칙 27)

- 2021 ATT → eCPM 30~40% 감소 → 하이퍼캐주얼 공식 붕괴
- 생존 경로: **하이브리드 캐주얼로 전환** (IAP 추가)
- 2023~2024: 하이브리드 캐주얼 수익 **+30%**

### 체크리스트

```
□ BM 전략이 장르에 맞게 선택되어 있는가?
□ 하이퍼캐주얼이라면 하이브리드로 전환 계획이 있는가?
□ 니어 미스 직후 과금 제안이 배치되어 있는가?
□ 유료 모델을 선택하고 있지 않은가? (모바일 부적합)
```

## 원칙 4. 소셜 — 무료 UA 엔진이자 리텐션 엔진

### Monopoly GO $6B의 비밀

```
Monopoly IP(노스탤지어) + 주사위+보드게임(친숙도)
+ 소셜 공격/방어(도파민) + 스티커 트레이딩(수집+소셜)
+ 라이브 이벤트(FOMO) = $6B
```

4대 소셜 메커닉:
1. **공격/방어 루프** — 친구 보드 공격 → 복수심 → 재접속
2. **스티커 앨범 트레이딩** — Facebook 그룹 자발 형성
3. **파트너 이벤트** — "친구가 기다린다" → 의무감
4. **주사위 링크 공유** — 무료 UA 바이럴

### 캐주얼 소셜 ROI

| 소셜 기능 | 효과 |
|---------|------|
| 친구 순위표 | 이탈률 감소 + 바이럴 (Candy Crush 초기) |
| 팀 이벤트 | 수익 +25% (Royal Match Hidden Temple) |
| 실시간 PvP | RPD 2~5배 (Match Masters $5 vs 일반 $1~2) |
| 스티커 트레이딩 | 게임 외 소셜 생태계 (Monopoly GO) |

### 체크리스트

```
□ 소셜 기능이 최소 1개 설계되어 있는가?
□ 공유/바이럴 메커닉이 있는가?
□ 소셜이 BM과 연결되어 있는가?
```

## 원칙 5. 머지(Merge) 장르 — "새로운 매치3"

### 중독성 4가지 원리

1. **완성 욕구** — A+A=B, B+B=C... 다음 단계가 항상 보임
2. **공간 관리 긴장** — 제한 보드, 지우지 않으면 막힘 → 지속 행동
3. **즉각 피드백** — 머지 성공 즉시 시각/청각 효과
4. **발견의 재미** — 무엇이 나올지 모름 → 가챠적 요소

### 메타 서사 결합 = 수익 극대화

- Merge Dragons($780M+): 순수 머지 (메타 없음)
- Merge Mansion($600M+): 할머니 미스터리 스토리 → 정서적 투자 → 수익 증가

### 체크리스트

```
□ 머지 게임이라면 4가지 중독성 원리가 모두 있는가?
□ 메타 서사/꾸미기가 결합되어 있는가?
```

## 원칙 6. 힐링/시뮬레이션 — 소규모 팀의 무기

### 성공 공식

- **아트 품질 = 핵심**. 게임성보다 시각적 만족이 우선
- zero pressure design — 강제/타이머 없음 → 자발적 복귀
- 수집 완성욕 (도감 채우기)
- SNS 공유 = 유기적 UA

### 구조적 한계

| 항목 | 현실 |
|------|------|
| ARPU | 매우 낮음 ($0.5~1) |
| 수익 천장 | $10~30M 수준 (어비스리움 추정) |
| 수명 | 바이럴 버즈 소멸 후 급락 |
| 적합 대상 | 소규모 팀, 저비용 개발 |

### 체크리스트

```
□ 아트 품질이 "이것만으로도 가치 있다" 수준인가?
□ 강제 요소 없이 자발적 복귀 구조인가?
□ 수집/도감 시스템이 있는가?
□ SNS 공유 메커닉이 있는가?
□ 대규모 투자 회수를 기대하고 있지 않은가? (ARPU 한계 인지)
```
