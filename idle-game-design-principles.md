## 원칙 0. 방치형의 본질 — "최소 노력으로 최대 성취감"

> 유저가 원하는 것은 "아무것도 안 해도 되는 게임"이 아니라 "최소의 노력으로 최대의 성취감"이다.

결정을 내리는 순간이 재미의 핵심이다. 방치 구간은 "결정의 결과가 쌓이는 시간"이고, 접속은 "다음 결정을 내리는 순간"이다.

---

## 원칙 1. 오프라인 보상 — 6~12시간 상한이 황금 구간

### 데이터

| 게임 | 상한 | 복귀 주기 |
|------|------|---------|
| AFK Arena | 12시간(기본), VIP 36시간 | 하루 2~3회 |
| Nonstop Knight | 2층 진행 (보스 차단) | 강제 복귀 |
| Clicker Heroes | 무제한 | 복귀 강요 없음 |

- **최적: 6~12시간** — 하루 2~3회 자연 복귀 유도
- 너무 짧으면(2시간): "숙제 게임" 불만
- 너무 길면(72시간+): 유저가 게임 존재 망각

### 복귀 트리거 5가지

1. **보상 상한 알림** — "오프라인 보상이 꽉 찼습니다!"
2. **시간 제한 이벤트** — 24~72시간 이벤트, FOMO
3. **진행 차단** — "보스 처치 필요" (직접 해결 요구)
4. **소셜 알림** — 길드원 활동, 랭킹 변화
5. **데일리 미션** — 일일 퀘스트 리셋 → 습관

### 체크리스트

```
□ 오프라인 보상 상한이 6~12시간인가?
□ 복귀 트리거가 최소 2개 설계되어 있는가?
□ "너무 방치적"이지 않은가? (접속 없이 아무것도 안 막히면 위험)
```

## 원칙 2. 프레스티지 — 방치형의 핵심 엔진

### 프레스티지 3원칙

1. **잃는 것을 최소화** — 손실 적을수록 리셋 빈번 = 빠른 성장 체감 = 높은 리텐션
2. **획득은 반드시 쌓이기** — "더 강하게 시작" 보장 없으면 리셋 거부
3. **첫 프레스티지 = 1~3일 내** — 너무 빠르면 이해 전 리셋, 너무 늦으면 의문 발생

### 프레스티지 사이클 주기

| 단계 | 주기 |
|------|------|
| 초반 첫 프레스티지 | 24~72시간 |
| 중반 | 수일~1주 |
| 후반 | 수주~1개월 |

### 체크리스트

```
□ 프레스티지(또는 시즌 리셋) 시스템이 있는가?
□ 첫 프레스티지가 1~3일 내에 경험되는가?
□ 리셋 후 "더 강하게 시작"이 체감되는가?
```

## 원칙 3. 수익 결정 공식 — 메타게임 레이어가 100배 격차를 만든다

### 법칙 30 (396종 분석)

```
방치형 수익 = 가챠 깊이 × PvP 긴장감 × 소셜 레이어
```

| 구조 | 대표 | 수익 |
|------|------|------|
| 순수 방치+광고 | AdVenture Capitalist | $1~5M |
| 방치+IAP | Clicker Heroes류 | $10~50M |
| 방치+가챠 | Idle Heroes | $100~500M |
| **방치+가챠+PvP+소셜** | **AFK Arena** | **$1B+** |

### 광고 활용 (방치형 특화)

리워드 광고 4가지 배치:
1. **오프라인 보상 2배** — 복귀 시 "30초 광고 = 보상 2배"
2. **진행 속도 부스트** — "광고 = 2시간 추가 방치 자원"
3. **특별 아이템 언락** — 수집 욕구 자극
4. **프레스티지 보너스** — 리셋 직전 고가치 타이밍

### 시간 판매의 심리학

> "기다림의 고통 × 진행 욕망 = 지불 의향"

- AFK Arena 역설: 핵심 자원은 오직 방치로만 획득 → 과금으로 직접 구매 불가 → 오히려 "공정하다" 인식 → 소과금 전환율 증가

### 체크리스트

```
□ 메타게임 레이어가 최소 2개(가챠/PvP/소셜 중) 있는가?
□ 리워드 광고가 방치형 특화 타이밍에 배치되어 있는가?
□ 시간 판매가 "삶의 질 개선"으로 프레이밍되어 있는가?
□ 무과금도 충분히 진행 가능한가? (느리게라도)
```

## 원칙 4. 성장 설계 — 지수 곡선 + 숙련도 층위 + 질적 변화

### 수학 모델 (Pecorella GDC 2016)

- 비용: 지수 함수 (n^x) — 반드시 지수 증가
- 생산: 선형/다항 (x^k) — 비용보다 느리게
- 이 갭이 "벽"을 자연 생성

### 성장 쾌감 유지 4가지 도구

1. **순차적 레이어 해금** — 레벨업→어센션→시그니처→가구 (AFK Arena)
2. **빠름-느림-빠름 리듬** — 저항이 있어야 가속이 쾌감
3. **빌드 다양성** — 내가 선택한 빌드 = 정체성 (Tap Titans 2 스킬 트리)
4. **질적 변화** — 수치 1000→1001이 아니라 외형/능력이 변해야 성장감

### 체크리스트

```
□ 비용이 지수, 생산이 선형/다항으로 설계되어 있는가?
□ 성장 레이어가 순차적으로 해금되는가?
□ 성장 과정에서 질적 변화(외형/능력)가 있는가?
□ 빌드 다양성이 존재하는가?
```

## 원칙 5. 소셜/PvP — 방치형에서의 비동기 설계

### 가능한 PvP 형태

| 유형 | 구현 | 대표 |
|------|------|------|
| 비동기 방어 AI | 내 팀이 AI로 방어 | AFK Arena |
| 스냅샷 랭킹 | 전투력 스냅샷 순위 | Idle Heroes |
| 시즌 리더보드 | 시즌 누적 점수 경쟁 | Capybara Go |
| 클랜 레이드 | 보스 DPS 경쟁 | Tap Titans 2 |

### 길드 효과

- Tap Titans 2: 클랜 레이드 참여 유저 D30 리텐션 **2.3배**
- 길드 = 접착제(의무감) + 목표 공유(복귀 이유 2배) + 소셜 증거(성장 동기)

### 체크리스트

```
□ 비동기 PvP가 설계되어 있는가?
□ 길드/클랜이 있는가? (D30 리텐션 2.3배 효과)
□ 구간 랭킹으로 무과금도 경쟁 가능한가?
```

## 원칙 6. 실패 패턴 4가지

| # | 패턴 | 진단 | 대응 |
|---|------|------|------|
| 1 | **너무 방치적** | 접속 없이 아무것도 안 막히면 유저가 망각 | 오프라인 상한 + 진행 차단 + 이벤트 |
| 2 | **성장 인플레이션** | 숫자가 무의미해짐, P2W 격차 1000배+ | 시즌 리셋(AFK Journey), 구간 랭킹 |
| 3 | **시장 포화** | 차별화 없는 방치RPG = 매몰 | 유니크 메커닉 결합 (로그라이트/건설/IP) |
| 4 | **P2W + 클론** | 복사본에 P2W = 15개월 종료 | 차별화 필수, P2W 상한 설계 |

### 체크리스트

```
□ "너무 방치적"이지 않은가?
□ 인플레이션 방지 장치(시즌 리셋/구간 랭킹)가 있는가?
□ 시장 내 차별화 포인트가 명확한가?
□ P2W 상한이 설계되어 있는가?
```

## 원칙 7. 방치+X 하이브리드 — 소규모 팀의 선택지

| 결합 | 대표 | 수익 | 소규모 팀 적합 |

| **방치+로그라이트** | Capybara Go | $109M | ✅ (콘텐츠 자동 생성) |

### 소규모 팀 최적 선택

**방치+로그라이트** = 콘텐츠 자동 생성 + 런 다양성 + 라이브 부담 최소

### 체크리스트

```
□ 방치+X에서 X가 라이브 부담을 줄이는 구조인가?
□ 소규모 팀이 6주 주기 업데이트를 감당할 수 있는가?

```
