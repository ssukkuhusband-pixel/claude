# 운수좋은날 (One Lucky Day)

## 개요
- **한 줄 설명**: 슬롯머신 스핀으로 전투하는 턴제 RPG
- **장르**: 슬롯라이크 (슬롯머신 + 턴제 RPG + 로그라이크 성장)
- **타겟 플랫폼**: 모바일 (iOS / Android)
- **타겟 시장**: 글로벌 (한국어/중국어 지원 확인)
- **퍼블리셔**: Habby (Habby Login 연동 예정)

## 핵심 유니크 포인트
5x3 슬롯 그리드를 스핀하여 4속성(화염/번개/자연/물) 심볼을 매칭하고, 축적된 속성 데미지로 1v1 턴제 전투를 벌이는 슬롯라이크. 스핀 결과에 전략적 개입(속성 변환, 특수 심볼, 반반 심볼)을 더해 운과 전략의 균형을 만드는 것이 핵심.

## 참고 게임
- **유니크 영역**: 슬롯 + RPG 전투 결합 (직접 개척)

## 확정 사항

### 전투 시스템 (프로토타입 검증 완료)
- **5x3 슬롯 그리드**, 4속성 체계 (화염/번개/자연/물)
- **매칭 규칙**: 가로/세로/대각선 3개 이상 연속 매칭 시 해당 속성 데미지 축적
- **하이브리드 심볼**: 두 속성 동시 매칭 가능한 반반 심볼
- **특수 패턴**: 삼각형, 역삼각형, 십자가 (전체 그리드 패턴 매칭)
- **1v1 턴제**: 스핀 데미지 적용 후, 적이 monsterTurn 주기마다 반격
- **상태이상 4계열**: 화상(DoT) / 어지러움(디버프) / 따가움(피해증가) / 저체온→빙결(행동불능)

### 스킬 시스템 (v1 확정 — 143종)
- **레벨업 시 3개 중 1개 선택** (로그라이크 드래프트)
- **등급**: B(20종) / A(53종) / S(44종) / SS(26종)
- **타입 14종**: 속성확률, 속성변환, 속성 특수심볼, 상태이상강화, 반반심볼, 이중속성 강화, 만능심볼, x2룬, 스탯, 조건부 스탯, 패턴배율, 콤보강화, 생존, 사제 이벤트
- 상세 목록: `skill_v1.csv`

### 펫 시스템 (프로토타입 기준)
- 36종: C(4) / B(8) / A(8) / S(8) / SS(8)
- 가챠 확률: C(120) / B(65) / A(10) / S(4) / SS(1)
- 최대 2마리 장착, 쿨다운 기반 액티브 + 패시브
- 합성으로 상위 등급 획득

### 장비 시스템 (프로토타입 기준)
- 6부위: 항아리, 목걸이, 귀걸이, 옷, 장갑, 신발
- 등급 11단계: C → B → A → S → S1 → S2 → SS → SS1 → SS2 → SS3 → SSS
- 3축 분류: 보호(PROTECT) / 공격(ATTACK) / 상태(STATUS)
- 일반/스타 2티어, 가챠 + 합성 승급

### 추억장식 시스템 (프로토타입 기준)
- 20종 × 5레벨, 4슬롯 장착
- Balatro 스타일 곱배율 시스템 (조건부 배율 적용)
- 등급: A / S / SS / SSS
- 조각 수집 → 레벨업 (10/20/30/40/50 조각)

### 컨텐츠 구성 (마일스톤 시트 기준)
- 메인 스테이지 100챕터 (챕터당 10스테이지)
- 인게임 스킬 200종 (v1 기준 143종 확정)
- 인게임 이벤트 5종 (룰렛, 악령구출 등)
- 던전: 미믹던전(최대 데미지 기반), 성당(뽑기 던전)
- 레벨업 100레벨

### BM 구조 (마일스톤 시트 기준)
- 시즌 패스 (2주 단위)
- 월정액 (일반/고급 2종)
- 광고 시스템 + 광고 제거 패키지
- 상점: 챕터 패키지 / 한정 패키지 / 일일 상점 / 보석 상점 / 골드 상점 / 팝업 패키지
