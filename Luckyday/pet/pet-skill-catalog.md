# 펫 스킬 메커니즘 카탈로그

> 참치김밥 작성 | 운수좋은날 전투 로직 + 인게임 스킬 143종 + 상태이상 4종 교차 분석
> 목적: 펫이 개입할 수 있는 모든 메커니즘을 빠짐없이 나열한 뒤, 등급별 배분의 재료로 사용

---

## 분석 기반

### 전투 루프
```
스핀 → 5x3 그리드에 심볼 배치 → 매칭 판정(가로/세로/대각 3+)
→ 속성별 데미지 축적 → 추억장식 곱배율 적용 → 적에게 데미지 → 적 턴(반격) → 반복
```

### 펫의 제약 조건
- ❌ 직접 데미지 (추억장식 뻥튀기 앞에서 무의미)
- ❌ 화상 직접 부여 (기준 화염데미지 없음)
- ❌ 출혈 직접 부여 (기준 자연데미지 없음)
- ✅ 저체온 부여 가능 (스택 기반)
- ✅ 어지러움 부여 가능 (스택 기반)
- ✅ 두두 버프/디버프 해제/슬롯 조작 등

### 상태이상 정의
| 상태이상 | 메커니즘 | 부여 경로 |
|---|---|---|
| 화상 | 화염 최종 데미지의 10% = 1틱 피해 | 불씨 심볼 체크로만 (펫 불가) |
| 출혈 | 자연 최종 데미지의 10% = 1틱 피해 | 가시 심볼 체크로만 (펫 불가) |
| 저체온 | 3스택 → 빙결 1턴. 스택 영구 유지 | 얼음 심볼 + 펫 가능 |
| 어지러움 | 1스택당 적 공격력 -10%, 빗맞음 확률 +10% | 천둥 심볼 + 펫 가능 |

### 펫 vs 장비 역할 분리 원칙
> **펫의 모든 패시브는 [액티브]와 연결되어야 한다.**
> 액티브 없이 독립적으로 작동하는 수치/효과 보너스는 장비의 영역이다.

## C. 등급별 설계 가이드라인

### C등급 (4종, 패시브 0)
- **역할**: 4가지 기본 역할 소개 (버프/보호/회복/디버프)
- **액티브 풀**: A1-1, A2-1, A2-2, A2-3, A3-1, A3-2 중 택
- **원칙**: 한 가지 기능만 심플하게. 10초 안에 이해.

### B등급 (8종, 패시브 1 — 강화 후 해금)
- **역할**: 기본 역할의 확장 + 새로운 카테고리 도입 (정화, 슬롯 조작)
- **액티브 풀**: C등급 + A2-4, A2-5, A2-6, A4-1, A4-2
- **패시브 풀**: B1(쿨타임 순환), B2-4(콤보 연동)
- **원칙**: 패시브1이 액티브와 기본적인 시너지 루프를 형성. 강화 후 "아, 이렇게 쓰는 거구나" 깨달음.

### A등급 (8종, 패시브 2 — 강화1, 강화2 해금)
- **역할**: 전문화 시작. 카테고리 내에서 세부 특성 분화.
- **액티브 풀**: B등급 + A1-3, A1-6, A3-6, A4-4, A4-5, A4-7
- **패시브 풀**: B1 전반, B2-1, B2-3, B2-7, B2-8, B2-10
- **원칙**: 패시브1 = 핵심 루프, 패시브2 = 조건부 강화. 인게임 스킬과의 시너지가 처음으로 명확해지는 등급.
- **C/B와의 차별점**: 새로운 심볼 종류(조커, 반반) 또는 새로운 역할(속성 특화, 적 약화)이 등장.

### S등급 (8종, 패시브 3 — 최초1, 강화1, 강화2 해금)
- **역할**: 빌드 정체성 확립. "이 펫 = 이 전략"이 명확.
- **액티브 풀**: A등급 + A1-4, A1-5, A1-7, A2-7, A2-8, A3-3, A3-5, A4-3, A4-6, A4-8, A4-10
- **패시브 풀**: B1 전반, B2 전반
- **원칙**: 최초 패시브1이 액티브의 핵심 루프를 완성. 강화할수록 루프가 촘촘해짐.
- **A등급과의 차별점**: 복합 효과 액티브 (가로+세로 부적, 정화+대회복, 복합CC 등). 단순 수치 스케일이 아닌 새로운 기능의 등장.

### SS등급 (8종, 패시브 4 — 최초2, 강화1, 강화2 해금)
- **역할**: 전설급. "이 펫이 있으면 전투의 규칙이 바뀐다."
- **액티브 풀**: S등급 + A4-8(홀드), A4-9(전용 심볼), A5-1(연쇄 스핀), A5-2(장식 증폭)
- **패시브 풀**: B1 전반 + B1-8(초기화), B2 전반 + B2-2/B2-5, B3-1(펫 시너지)
- **원칙**:
  - 액티브가 읽는 것만으로 "갖고 싶다"는 감정을 유발
  - S등급에는 없는 완전히 새로운 메커니즘이 최소 1개 포함
  - 패시브1+2(최초 보유)만으로 SS의 핵심 판타지가 작동
  - 패시브3+4(강화 해금)는 극한 최적화의 완성
- **S등급과의 차별점**:
  - 채택된 신규 메커니즘: 연쇄 스핀, 추억장식 2회 발동, 심볼 홀드, 펫 전용 특수 심볼(1~2종)
  - "규칙을 바꾸는" 수준의 임팩트

## D. 인게임 스킬과의 시너지 맵

### 빌드 축 0: 화상/출혈 빌드 (인게임 불씨/가시/상태이상강화 스킬)
- **시너지 펫 메커니즘**: A3-7(화상/출혈 지속턴 연장), A4-11(불씨/가시 특수 심볼 추가 등장)
- **예시 조합**: 불씨+업화+보호막녹이기 ← 펫이 불씨 추가 등장 + 화상 지속턴 연장 → 화상 DoT 극대화

### 빌드 축 1: 속성 집중 빌드 (인게임 속성확률/변환 스킬)
- **시너지 펫 메커니즘**: A1-3(속성 데미지 증가), A1-6(속성 확률 조작), A4-7(심볼 변환)

### 빌드 축 2: 빙결 빌드 (인게임 얼음/빙결 강화/영구동토)
- **시너지 펫 메커니즘**: A3-1(저체온), A3-3(복합CC), B2-6(방패→저체온)

### 빌드 축 3: 어지러움 빌드 (인게임 천둥/어지러움 강화/혼절)
- **시너지 펫 메커니즘**: A3-2(어지러움), B1-5(빗나감→쿨감)

### 빌드 축 4: 패턴 특화 빌드 (인게임 패턴배율, 사제 이벤트)
- **시너지 펫 메커니즘**: A4-1/2/3(부적), A4-6(x2룬)

### 빌드 축 5: 반반/이중속성 빌드 (인게임 반반심볼, 이중속성 강화)
- **시너지 펫 메커니즘**: A4-5(반반심볼 생성), A1-6(속성 확률 조작)

### 빌드 축 6: 크리티컬 빌드 (인게임 스탯, 조건부 스탯)
- **시너지 펫 메커니즘**: A1-2(크리 버프), A1-5(분노), B1-6(크리→쿨감)

### 빌드 축 7: 생존 빌드 (인게임 생존, 방패 계열)
- **시너지 펫 메커니즘**: A2 전반, B2-6(방패 소모 연동)

### 빌드 축 8: 콤보 빌드 (인게임 콤보강화)
- **시너지 펫 메커니즘**: A1-7(콤보 부스터), B1-1(콤보→쿨감), B2-4(콤보 연동)
