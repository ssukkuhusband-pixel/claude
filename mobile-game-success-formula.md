| **현황** | 순수 하이퍼캐주얼은 사실상 시장에서 퇴출 |

## 파트 3. 귀납적 결론: 성공과 실패의 법칙

---

### 법칙 1. 유니크 전략 — "갭 공식"의 실증

> **성공한 게임의 80% 이상은 기존에 없던 장르 조합을 했다.**

| 조합 | 게임 | 수익 |
|------|------|------|
| 오픈월드 RPG + 가챠 | Genshin Impact | $4B+ |
| 협동 핀볼 + RPG | Monster Strike | $11B+ |
| 타워디펜스 + 수집 RPG | Arknights | $1B+ |
| 수직 슈터 + 수집 RPG | NIKKE | $1B+ |
| 카드 덱빌딩 + 타워디펜스 + RTS | Clash Royale | $3.3B+ |
| 보드게임 + 소셜 공격/방어 | Monopoly GO! | $3B+ (473일 내) |
| 농장 + 도시 건설 | Township | $2.4B+ |
| 생존 재난 테마 + 4X SLG | Whiteout Survival | $2B+ |
| 육성 시뮬 + 경마 IP 미소녀화 | Uma Musume | $2.5B+ |
| 로그라이크 + 모바일 슈터 | Archero | $200M+ |

**결론**: "새 장르 조합"의 반대, 즉 "기존 게임 복제"는 거의 예외 없이 실패했다.

- Harry Potter: Wizards Unite → 포켓몬GO 복제 → 실패
- Concord → Overwatch 복제 → 14일 만에 종료
- C&C Rivals → Clash Royale 복제 → 실패
- 수십 개의 한국 MMORPG → 리니지M 복제 → 실패

---

### 법칙 2. 수익화 — F2P + 시즌패스가 10년 표준

> **2018년 이후 성공한 게임의 90%+는 F2P + 인앱결제 조합이다.**

**검증된 수익화 공식:**
```
핵심 수익 = 가챠(수집 욕구 자극) + 배틀패스/월정액(안정 수익) + 한정 이벤트(FOMO)
보조 수익 = 패키지 번들 + 광고(캐주얼 한정)
```

**실패 수익화 패턴:**
- **선결제**: Super Mario Run ($9.99) → 전환율 극히 낮음, Nintendo 스스로 실패 인정
- **극단적 P2W**: Diablo Immortal (완성 $100,000~$500,000) → 브랜드 파괴
- **이중 결제 장벽**: Artifact ($20 구매 + 카드 팩) → 출시 2개월 97.5% 유저 이탈
- **F2P 전환 지연**: LawBreakers → 이미 커뮤니티 붕괴 후 F2P 전환

**배틀패스의 표준화**: 2018년(Fortnite Battle Pass 성공) 이후 출시한 성공 게임의 거의 전부가 시즌패스를 도입. 안정적 월 수익 + 참여 유도의 핵심 도구.

---

### 법칙 3. 장기 리텐션 — 소셜 레이어가 핵심

> **1년 이상 Top Grossing을 유지한 게임의 공통점: 강력한 사회적 결속 구조**

| 메커니즘 | 대표 게임 | 효과 |
|----------|----------|------|
| 클랜/길드 전쟁 | CoC, CoC Royale, Summoners War | 소속감 + 이탈 시 죄책감 |
| 글로벌 PvP 랭킹 | Summoners War, FGO, Uma Musume | 경쟁심 + 지위 과시 |
| 실시간 협동 레이드 | Monster Strike, Epic Seven | 약속 이행 압박 |
| 동맹 시스템 | Clash of Clans, Whiteout Survival | 공동 목표 + 사회적 압력 |
| 커뮤니티 데이 | Pokémon GO | 현실 모임 + 소속감 |

**소셜 레이어 없이 실패한 게임들**: MapleStory 2, Dragalia Lost, 호연 → 엔드게임 부재 + 사회적 결속 부재 → 동시접속 99%+ 붕괴

---

### 법칙 4. IP 전략 — "IP ≠ 성공 보장"

> **IP가 있어도 장르 적합성과 원작 팬의 기대치를 충족 못 하면 오히려 역풍이 온다.**

**IP가 성공에 기여한 경우:**
- Pokémon GO: 포켓몬 수집 = 위치 기반 AR 수집과 완벽 매칭 → $8B+
- FGO: Fate IP + 스토리 가챠 → 일본 팬덤 정확히 공략 → $4B+
- Dragon Ball Dokkan: DB IP + 각성 연출 충격 → $3B+

**IP가 오히려 역풍을 초래한 경우:**
- Harry Potter: Wizards Unite → HP팬은 AR 위치기반이 아닌 마법 경험 원함 → 실패
- FF7: The First Soldier → FF7팬 ≠ 배틀로얄 팬 → 둘 다 확보 실패
- C&C: Rivals → C&C팬이 원하는 건 PC RTS, 모바일 Clash Royale 아님 → E3 싫어요 폭탄
- M&M: Era of Chaos → Heroes III 팬은 턴제 전략 원함, 가챠 RPG 아님 → IP 착취로 인식

**핵심 원칙**: **IP를 사용할 때 원작의 핵심 경험(Core Fantasy)을 유지해야 한다.** IP는 마케팅 비용을 줄여주는 도구지, 게임 품질을 대체하지 않는다.

---

### 법칙 5. 시장 타이밍 — 경쟁 지형을 읽어라

> **좋은 게임도 잘못된 타이밍에 출시하면 죽는다.**

**타이밍 실패 사례:**
- Battleborn (2016.05.03) vs. Overwatch (2016.05.24) → 3주 차이로 공간 자체 소멸
- LawBreakers (2017.08.08) → Overwatch Summer Games 복귀 시기와 겹침 → 출시 첫날 피크 3,000명
- Apex Legends Mobile → PC/콘솔 Apex와 크로스 진행 없어 기존 팬 이탈

**타이밍 성공 사례:**
- Arknights → FGO 독주 시대에 타워디펜스 RPG라는 새 장르로 진입 → 경쟁 없는 포지션 확보
- Whiteout Survival → 기존 SLG(CoC, LoK 등)의 "전쟁 피로도" 증가 시점에 생존 테마로 차별화
- Monopoly GO! → 모바일 소셜 카지노 게임의 IP 부재 공백을 Monopoly 브랜드로 메움

### 법칙 6. AR 게임 — 포켓몬GO만 살아남았다

> **2013~2025년 사이 출시된 AR 위치기반 게임 중 지속 생존한 것은 포켓몬GO가 유일하다.**

| 게임 | 결과 |
|------|------|
| Pokémon GO | ✅ 생존 ($8B+, 2024년도 $5.45억) |
| Harry Potter: Wizards Unite | ❌ 2022년 종료 |
| Minecraft Earth | ❌ 2021년 종료 |
| Ghostbusters World | ❌ 2020년 종료 |
| Walking Dead: Our World | ❌ 2023년 종료 |
| Jurassic World Alive | ⚠️ 극히 낮은 수익으로 유지 |

**포켓몬GO가 유일하게 생존한 이유:**
1. 포켓몬 수집 = AR 수집과 의미론적으로 완벽한 매칭
2. Niantic의 지도 인프라(Ingress 기반) 독점 우위
3. 커뮤니티 데이·레이드 등 오프라인 사교 활동 내재화
4. COVID 시기에도 홈플레이 모드 빠른 도입으로 생존

**결론**: AR 게임은 "포켓몬GO 복제" 시도가 아니라, 완전히 다른 수집·사회적 메커니즘을 설계할 때만 가능성이 있다.

### 법칙 7. 수익 효율 vs. 다운로드 규모의 이분법

> **다운로드 1위 ≠ 수익 1위. 두 전략은 완전히 다른 게임을 요구한다.**

**다운로드 챔피언 (광고 수익 중심):**
- Subway Surfers: 40억+ 다운로드, 광고 수익 중심
- Garena Free Fire: 10억+ 다운로드, 신흥시장 중심

**수익 효율 챔피언 (과금 유저 집중):**
- Arknights: 1,700만 다운로드로 $1B+ 달성 → 다운로드당 수익 **$40** (업계 평균의 4~10배)
- FGO: 3,200만 다운로드로 $4B+ 달성 → 극성 팬덤의 고과금

**시사점**: 소규모 팀이 선택해야 할 전략은 **수익 효율형**. 다운로드 경쟁은 마케팅 자본전쟁이지만, 수익 효율은 게임 설계의 문제임다.

---

### 법칙 8. Square Enix 교훈 — "타이틀 남발의 함정"

> **한 회사의 사례 연구: 10종 이상의 게임이 3년 내 종료된 단일 원인**

2021~2024년 Square Enix 모바일 부문 종료 타이틀:
NieR Reincarnation, Echoes of Mana, Engage Kill, FF Record Keeper (글로벌), Kingdom Hearts Union χ, Dragon Quest Dai, Fullmetal Alchemist Mobile, Final Fantasy VII: The First Soldier 등 **10종 이상**

**원인:**
1. 동시에 너무 많은 타이틀 운영 → 개별 게임 투자 분산
2. Genshin Impact 이후 높아진 기대치를 충족할 여력 없음
3. IP를 가챠 수익화 도구로만 사용 → 원작 팬덤 신뢰 붕괴
4. 퍼블리셔 브랜드 전체가 "곧 종료될 게임"으로 인식됨

**교훈**: **적게 출시하고, 하나에 집중하라.** 소규모 팀에서 이 원칙은 더욱 중요함다.

---

### 법칙 9. 마케팅 기대치 — 약속을 어기면 48시간 안에 붕괴

> **게임 품질 문제보다 '기대치 배신'이 더 빠르게 치명적이다.**

| 게임 | 약속 | 현실 | 결과 |
|------|------|------|------|
| The Day Before | 오픈월드 MMO 서바이벌 | 미완성 추출 슈터 | **4일** 만에 스튜디오 폐쇄 |
| Blade & Soul 2 | 과금 낮추겠다 | 리니지와 동일 과금 재포장 | 출시 5일 내 74% 유저 이탈 |
| Concord | AAA 히어로 슈터 | 2024년에 2016년 수준의 게임 | **14일** 만에 서비스 종료 |

**결론**: **마케팅이 설정한 기대치 = 게임이 지켜야 할 계약이다.** 이를 어기면 데이터 시대에 즉각 전파되고 복구 불가능하다.

### 법칙 10. 재미의 공식 — 성취감 × 의외성

> **성공한 게임들의 유저 리뷰에서 반복적으로 등장하는 두 가지 키워드**

**성취감 요소가 명확한 성공 사례:**
- CoC: 기지 업그레이드 → 클랜 전쟁 승리 → 명확한 성장 시각화
- Genshin Impact: 캐릭터 획득 → 스킬 숙련 → 나선비경 클리어 → 다음 캐릭터 가챠
- Summoners War: 룬 최적화 → PvP 순위 상승 → e스포츠 참가 가능성

**의외성 요소가 성공에 기여한 사례:**
- Monster Strike: 협동 플레이 결과가 매번 다름 (물리 엔진 기반 핀볼)
- Archero: 로그라이크 랜덤 스킬 선택 → 매 런마다 다른 빌드
- Coin Master: 주사위 결과의 랜덤성 → 공격/방어/이벤트 예측 불가

**실패 사례의 반대 패턴:**
- MapleStory 2 엔드게임: 극히 랜덤한 드롭 + 타임게이트 → 성취감 없는 반복 → 붕괴
- KartRider: Drift: 겉모습 캐주얼 + 실제 극난이도 → 성취감 달성 전 이탈

## 최종 결론: 모바일 게임 성공 공식

```
성공 = (유니크한 장르 조합) × (F2P + 가챠/배틀패스 수익화) × (소셜 결속 구조) × (적절한 시장 타이밍)

실패 = (IP만 믿고 장르 검증 생략) OR (극단적 P2W) OR (마케팅 약속 위반) OR (경쟁자 정면 복제)
```

### 소규모 팀을 위한 응용

1. **적게 출시하고 하나에 집중**: Square Enix의 반면교사
2. **유니크함을 15초 영상으로 전달**: Arknights("타워디펜스 + 미소녀 수집"), NIKKE("뒤에서 쏘는 수직 슈터"), Monster Strike("친구와 함께 핀볼 RPG") — 모두 한 문장 설명 가능
3. **수익 효율형 타겟**: 다운로드 경쟁이 아닌 충성 유저 소수의 깊은 몰입 설계
4. **소셜 레이어 초기 설계**: 나중에 추가하면 늦는다
5. **마케팅 기대치 = 출시 게임의 계약**: 약속 이하로 출시하는 것은 최악의 선택
