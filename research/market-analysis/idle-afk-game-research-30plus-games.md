## 방치 유형 분류 정의

| 코드 | 유형 | 설명 |
|------|------|------|
| P | 순수 방치 | 오프라인에서도 자원 축적, 최소 인터랙션 |
| C | 클리커 | 탭/클릭으로 가속, 적극적 인터랙션 가능 |
| R | 방치+RPG | 자동전투 기반 RPG, 수집/성장 메타 |
| G | 방치+수집(가챠) | 가챠 + 방치 진행 |
| L | 방치+로그라이트 | 런 기반 + 방치 성장 |
| T | 방치+타이쿤 | 경영/건설 시뮬레이션 + 방치 |
| H | 하이브리드 | 위 유형 2개 이상 결합 |

## 핵심 인사이트 (성공/실패 패턴 비교)

### 성공 패턴
1. **수집 + 시너지**: 영웅/캐릭터 수집에 조합 시너지 추가 → 팀빌딩 깊이 (AFK Arena, Summoners War)
2. **아트 퀄리티 + 현지화**: 고품질 아트 + 타겟 시장 맞춤 (Memento Mori 일본, Legend of Mushroom 한/일)
3. **방치+다른 장르 결합**: 방치 단독은 약함 → 로그라이트/MMORPG/타이쿤 결합 (Capybara Go, Ulala)
4. **BM 균형**: 비착취적 BM이 장기 LTV 높임 (Egg Inc Pro Permit 모델)
5. **라이브옵스**: 지속적 이벤트+신규 영웅 → 10년 장수 가능 (Summoners War, AFK Arena)

### 실패 패턴
1. **P2W 극단화**: Day7 이후 결제 없이 진행 불가 → 이탈 폭탄
2. **클론화 + 차별화 부재**: AFK Arena 그래픽만 복사 → UA 경쟁에서 패배
3. **P2E 도입**: 게임 경제를 토큰에 연동 → 2022 크립토 겨울에 전멸
4. **얕은 메타게임**: D1-D7 방치 자체는 쉬움, D30+ 리텐션 구조 없으면 생존 불가
5. **현지화 무시**: 일본/한국 특화 IP를 글로벌 그대로 → 문화 거부감 (Princess Connect 글로벌)
