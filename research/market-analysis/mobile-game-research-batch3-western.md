## 심층 분석 노트

### A. 프리미엄 유료 게임의 모바일 성공 조건

| 게임 | 가격 | 성공 요인 | 한계 |
|------|------|-----------|------|
| Monument Valley | $3.99 | 시각 예술성 + BAFTA 수상 = 미디어 커버리지 = 자연 UA | 후속작 없으면 소멸, 총 수익 $25M은 히트 F2P 대비 소규모 |
| Stardew Valley | $4.99 | 1인 개발 스토리 + 커뮤니티 열풍 + 노과금 구조 차별화 | 모바일만으로는 수익 파악 어려움, PC 중심 게임 |
| Alto's Adventure | $4.99 (iOS) | Apple 피처링 + 명상적 톤 + 단순 조작 | Android F2P 전환 불가피 → 수익 미미 |
| Florence | $2.99 | 수상 실적 + "30분 경험" 가성비 | 총 수익 $150만 추정 — 지속 개발 자금 부족 토로 |
| Geometry Dash | $1.99 | UGC 생태계로 콘텐츠 무한 공급 | 단가 낮음 + 무료판(Lite)과 경쟁 |

**결론:** 모바일 프리미엄 게임은 "어워드 수상 → 미디어 노출 → 피처링"의 선순환 없이는 UA 확보 불가. 수익 천장은 F2P 대비 10배 이상 낮음.

### C. 하이퍼캐주얼의 지속성 분석

| 게임 | 출시 | 현재 상태 | 지속 비결 |

| Temple Run | 2011 | 유지 운영 (연 $400만 수준으로 감소) | 브랜드 인지도 + 저사양 기기 지원 |
| Cut the Rope | 2010 | 프랜차이즈 유지 (물리 퍼즐 무한 반복) | 10억+ DL 브랜드, Om Nom 캐릭터 IP |

### D. 기타 주목 데이터 포인트

- **Beatstar의 수익 배분 모델:** 게임 수익의 일부를 음원사에 공유($16M/1년) → 음악 산업과 게임의 공생 모델 선례
- **Pikmin Bloom의 역주행:** 출시 초반 부진 → Pikmin 4 닌텐도 스위치 출시(2023) 이후 모바일 다운로드 2배 급증 → 2024년 +60% 성장, 2025년 최고 실적. 콘솔 출시가 모바일 견인
- **Wordle의 구독 퍼널 역할:** Wordle 앱 다운로드가 NYT 구독자 전환의 최대 채널로 기능 — 일 1,450만 플레이어, 2024년 연간 53억 플레이. 게임이 미디어 구독 비즈니스를 지탱
- **Legends of Runeterra의 실패 요인:** 2024년 PvP 모드 사실상 종료. "PvP 세트 개발비 > 수익" 구조적 한계 → Path of Champions PvE 전환. CCG 장르는 Hearthstone이 선점한 시장에서 후발 진입 한계 확인
- **MTG Arena의 전통 TCG 디지털 전환:** 2023년 모바일 플레이타임 비중 40%, 디지털 매출 $4억으로 전체 MTG 수익의 35% — 물리 카드게임의 성공적 디지털 전환 사례
