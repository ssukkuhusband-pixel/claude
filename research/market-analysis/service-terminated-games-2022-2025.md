## 2022년 종료 게임 15종

### 01. Brave Frontier (브레이브 프론티어 글로벌)

| 항목 | 내용 |
|------|------|
| **개발사** | Alim / gumi |
| **출시** | 2013년 (JP) / 2014년 (글로벌) |
| **종료일** | 2022년 4월 22일 |
| **서비스 기간** | 약 8년 (글로벌) |
| **장르** | 수집형 RPG |
| **추정 수익** | 전성기 연 $100M+ (2016~2017), 종료 시점엔 급감 |
| **종료 원인** | 장기 파워크리프 누적 → F2P 유저 이탈 → 고래만 남아 수익 하한선 붕괴 |
| **위반 법칙** | ❌ 법칙 3 (소셜 루프 약화), ❌ 법칙 25 (킬 기준 미설정 → 좀비 운영 장기화) |
| **법칙 따랐는데도 실패?** | 아님다. 법칙 위반이 명확함다. 8년 운영으로 파워크리프가 임계치 초과함 |
| **새로운 실패 패턴?** | **"장수 게임의 파워크리프 임계점"** — 3년+ 가챠 RPG는 수치 인플레가 필연적으로 F2P 접근성을 파괴함다 |

**법칙 체크:**
- [1] 유니크함: ⚠️ 출시 당시엔 있었으나 시장이 따라잡음
- [2] BM: ❌ 파워크리프로 P2W 도가 임계치 초과
- [3] 소셜: ⚠️ 길드 있었으나 의미 약화
- [4] 팀 규모: ✅ 대형 스튜디오
- [5] 업데이트: ⚠️ 양은 많았으나 신선도 하락

| **종료 원인** | EA의 Glu Mobile 인수(2021) 후 포트폴리오 정리. 하이퍼캐주얼 장르 자체가 ATT 이후 광고 수익 모델 붕괴 |
| **위반 법칙** | ❌ 법칙 27 (단일 수익원 — 광고 의존 → ATT 이후 수익 모델 붕괴), ❌ 법칙 17 (M&A 이후 포트폴리오 정리) |
| **법칙 따랐는데도 실패?** | 아님다. 법칙 27 명확한 위반 (ATT 이후 하이퍼캐주얼 전체 붕괴) |
| **새로운 실패 패턴?** | **"ATT 이후 광고 의존 하이퍼캐주얼의 집단 종료"** — Apple ATT(2021)로 IDFA 기반 타겟 광고 수익이 50~70% 감소. 법칙 27 위반의 가장 대규모 실증 사례 |
