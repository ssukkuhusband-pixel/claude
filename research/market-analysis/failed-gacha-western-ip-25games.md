## 섹션별 심층 분석

### 섹션 A: 스퀘어에닉스 반복 실패 패턴 (6종)

**SINoALICE / NieR Re[in]carnation / Bravely Default: Brilliant Lights / Mobius FF / Star Ocean: Anamnesis / Dissidia Opera Omnia**

Square Enix의 모바일 사업부는 2015~2024년 사이 6개 이상의 타이틀을 연속 종료하며 업계에서 "모바일 무덤"이라는 별명을 얻었다.

**공통된 구조적 문제:**

1. **글로벌 운영 역량 부재**: 일본 서버 대비 글로벌 서버 콘텐츠 공급 속도가 3~6개월 지연. 일본 유저가 이미 경험한 콘텐츠를 글로벌 유저에게 뒤늦게 공급하는 구조가 고착됨. SINoALICE의 경우 핵심 콘텐츠인 "Colosseum" 길드전이 글로벌 서버에서 기술적으로 제대로 작동하지 않아 유저 이탈을 야기.

2. **수익성 임계점 도달 즉시 종료**: 일본 서버가 수익을 유지하는 동안에도 글로벌 서버는 선제적으로 정리하는 패턴. Star Ocean Anamnesis(JP 2021 종료, 글로벌 2019 종료), SINoALICE(대만·홍콩 2022, 글로벌 2023) 등에서 반복.

3. **$140M 이상의 비용 처리(FY2024)**: Square Enix는 2024년 회계연도에 약 220억 엔(~$140M)의 특별손실을 계상. 이는 개발 취소 및 운영 중단 게임들의 비용 정리임. NieR Re[in]carnation, SINoALICE가 이 처리의 주요 대상.

4. **서로 다른 IP임에도 동일한 전략**: 각 게임마다 다른 IP와 장르임에도 불구하고 동일한 수익화 공식(스탠다드 가챠 + 정기 이벤트 + 시즌 패스)을 그대로 적용. 차별화 실패.

5. **2024년 구조조정 이후**: 대표이사 기리유 타카시(Takashi Kiryu)가 2023년 실적 발표에서 "모바일 포트폴리오를 줄이고 AAA에 집중"을 선언. 이후 남은 모바일 타이틀들도 투자 축소 상태.

### 섹션 B: 2세대 AR 게임 집단 실패

**Harry Potter: Wizards Unite / Ghostbusters World**

Pokemon GO(2016)가 AR 게임 시장을 창조한 후, 2018~2019년에 여러 IP를 얹은 "2세대 AR 게임"이 출시됐다. 결과는 집단 실패였다.

| | Pokemon GO | Harry Potter: Wizards Unite | Ghostbusters World |
|--|------------|---------------------------|-------------------|
| **출시년도** | 2016 | 2019 | 2018 |
| **누적 DL** | 10억+ | 2,030만 | 미공개(소규모) |
| **누적 매출** | $67억+(2024) | $3,940만 | 수백만 달러 추정 |
| **종료** | 현재도 운영 중 | 2022.01 | 2020.08 |

**왜 2세대가 실패했나:**

- **Pokemon의 핵심 루프 = IP 자체**: Pokemon GO가 성공한 이유는 AR 기술 때문이 아니라, "포켓몬을 포켓몬답게 잡는다"는 IP-메카닉 일체감 때문이다. Harry Potter IP에는 이에 필적하는 모바일 수집 루프가 없었다.
- **"마법사가 된 느낌" 미구현**: 주문을 외우고 마법을 쓰는 경험이 아닌 화면을 문지르는 동작. IP 팬들의 기대치를 충족하지 못함.
- **Ghostbusters World의 영화 타이밍 실패**: Ghostbusters: Afterlife 개봉을 2020년에서 2021년으로 연기하면서 바이럴 기회 상실. 게임 혼자서는 자생 불가능한 구조였음.
- **Pokemon GO와의 자원 경쟁**: 같은 Niantic이 만든 게임임에도 Pokemon GO에 비해 신규 기능 추가 속도가 현저히 느렸음. Niantic 내부에서도 Pokemon GO 우선 자원 배분.

### 섹션 C: 닌텐도 IP의 모바일 한계

**Dragalia Lost**

닌텐도는 2016년 DeNA와 파트너십을 체결하며 모바일 시장에 본격 진입했다. 결과는 예상보다 훨씬 복잡했다.

**닌텐도 모바일 성과 비교:**
- Fire Emblem Heroes: 누적 $1.2억+(성공)
- Pokemon GO(제3자, 닌텐도는 지분만 보유): $67억+
- Super Mario Run: $6,400만 — 닌텐도 자체 "실패"로 규정
- Dragalia Lost: 총 $1억 4,600만(2020년까지) → 이후 급감, 종료

**Dragalia Lost의 딜레마:**
1. 닌텐도의 기본 모바일 전략 = "기존 IP로 신규 유저를 Switch로 유도" → 오리지널 IP인 Dragalia Lost는 이 전략과 구조적으로 맞지 않음
2. 비P2W 설계 철학 → 수익 창출 한계 → Fire Emblem Heroes 대비 ARPU 현저히 낮음
3. Genshin Impact 출시(2020.09) 이후 무료 오픈월드 RPG 대안이 생기면서 유저 이탈 가속

결론: 닌텐도는 Dragalia Lost 종료 이후 사실상 오리지널 모바일 IP 개발을 중단. 향후 모바일 전략은 기존 IP 활용으로만 제한될 가능성이 높다.

### 섹션 D: 서구 IP 모바일화 실패 패턴

**Command & Conquer: Rivals / Elder Scrolls: Blades / Battlefield Mobile / Star Wars 시리즈**

**EA의 반복 실수:**
- C&C: Rivals — E3에서 팬 야유를 받은 채 출시. Clash Royale 장르에 C&C IP를 억지로 끼워넣은 결과. 핵심 팬 니즈 무시.
- Elder Scrolls: Blades — P2W 게이트가 처음부터 문제. OpenCritic 37점은 AAA 스튜디오가 모바일에서 받은 최저 수준의 점수 중 하나. 7년 방치 후 2026년 6월 종료 예정.
- Battlefield Mobile — Battlefield 2042 실패의 불똥. Apple IDFA 정책 변경이라는 외부 충격. 스튜디오 120명 해산.

**Star Wars IP의 연속 실패:**
- Uprising(2016): 1년 만에 종료
- Force Arena(2019): 2년 만에 종료
- 배경: Disney/Lucasfilm이 모바일 게임 IP 라이선스를 여러 중소 개발사에 분산 허가 → 품질 관리 부재 → IP 브랜드 훼손 누적

**Behaviour Interactive의 특이 케이스:**
- Westworld Mobile: 자사 코드 무단 유용 소송(Bethesda) → 합의 후 강제 종료. 개발 역량 문제가 아닌 법적 문제로 종료된 유일한 케이스.
- Game of Thrones: Beyond the Wall: GOT 시즌8 실망감이 IP 가치를 훼손한 상태에서 출시. "낮은 퀄리티 머니 그랩" 첫인상을 끝내 극복하지 못함.

**Family Guy: The Quest for Stuff / South Park: Phone Destroyer / The Simpsons: Tapped Out**

이 세 게임은 각각 TV 애니메이션 IP를 활용한 빌더·카드 게임으로, 출시 초기 성공 이후 "좀비 상태(기술적으로 운영되나 신규 콘텐츠/투자 없음)"로 수년간 방치된 공통점이 있다.

| 게임 | 출시 | 최종 상태 | 말년 월 매출 |
|------|------|----------|------------|
| Simpsons: Tapped Out | 2012 | 2025.01 종료 | $수만 달러 |
| Family Guy: Quest for Stuff | 2014 | 2025.01 종료 | ~$5만 |
| South Park: Phone Destroyer | 2017 | 2022 후 방치 | ~$6만 |

**좀비 상태의 경제학**: 광고·잔여 과금으로 월 수만 달러를 유지하는 동안 서버 유지 비용이 그보다 낮으면 종료 명분이 없다. 그러나 이는 사실상 유저를 과거 콘텐츠로 가두는 비윤리적 방치와 다르지 않다.

## 25종에서 공통 발견되는 패턴 (5가지)

### 패턴 1: IP 인지도 ≠ 모바일 성공 (가장 중요)

Harry Potter, Star Wars, Game of Thrones, The Simpsons, Transformers — 세계 최대급 IP를 가졌음에도 연속 실패했다. 핵심 이유는 **IP의 핵심 경험을 모바일 메카닉으로 번역하지 못했기 때문**이다. Harry Potter 팬은 "마법사가 되는 경험"을 원했지만, Wizards Unite는 Pokemon GO 수집 루프에 Harry Potter 스킨만 입혔다. IP 라이선스를 취득한 순간 개발팀이 "어떻게 이 IP의 핵심 감정을 30초 세션에 담을 것인가"라는 질문을 포기하는 경향이 있다.

### 패턴 2: 글로벌 운영은 현지화가 아닌 별도 사업이다

Square Enix의 6종 게임 모두 일본-글로벌 서버 분리 운영에서 구조적 문제가 발생했다. 콘텐츠 공급 속도 차이, 길드전 시스템의 기술적 불안정, 글로벌 서비스 지원 팀의 부재가 반복됐다. Chain Chronicle 역시 Sega-Japan 서버와 Gumi 퍼블리싱 글로벌 서버의 괴리가 이탈을 만들었다. **"일본 서버를 영어로 번역해서 내보내는 것"은 글로벌 운영이 아니다.**

### 패턴 3: 출시 시점의 IP 건강도가 게임 생존을 결정한다

- Game of Thrones: Beyond the Wall — 시즌8 엔딩 실망(2019) 직후 출시(2020). IP 브랜드 붕괴 상태에서 시작.
- Ghostbusters World — 영화 연기로 IP 모멘텀 상실.
- Westworld Mobile — TV 시리즈 자체의 시청률 하락기에 출시.
- Battlefield Mobile — Battlefield 2042 역대 최악 반응 직후 개발 중.

반면 Pokemon GO(2016)는 시리즈 20주년과 맞물려 IP 히트 절정기에 출시됐다. **IP 게임의 성패에서 "언제 나왔는가"는 "무엇을 만들었는가"만큼 중요하다.**

### 패턴 4: 모바일 과금 설계 실수는 AAA 브랜드를 파괴한다

- Elder Scrolls: Blades: OpenCritic 37점. "Bethesda가 이런 게임을 냈다"는 브랜드 손상.
- Command & Conquer: Rivals: E3 야유 → C&C IP 자체의 부정적 이미지 강화 → 이후 C&C 리마스터 시리즈 판매에도 악영향.
- Diablo Immortal(참고): $10만 완전 강화 논란 → Blizzard 브랜드 손상.

**모바일 게임의 실패는 해당 타이틀로 끝나지 않는다. 모기업의 AAA IP 브랜드를 훼손한다.** EA, Bethesda, Square Enix 모두 모바일 실패가 콘솔/PC 시장에서의 선호도에 부정적 영향을 미쳤다.

### 패턴 5: "좀비 상태"와 공식 종료의 경계는 유저에게 더 잔인하다

The Simpsons: Tapped Out(2020~2025 좀비), South Park: Phone Destroyer(2022~ 방치), Valkyrie Connect(포럼 폐쇄 후 방치), Fantasica(2016년 기술 오류 방치) 등은 공식 서비스 종료 발표 없이 수년간 투자를 중단했다. 유저는 게임이 살아있다는 착각 속에 과금을 계속하지만, 개발팀은 이미 다른 프로젝트로 이동한 상태다. **좀비 상태는 공식 종료보다 더 무책임한 운영 방식이다. 적어도 공식 종료는 유저에게 마무리할 시간과 환불 기회를 준다.**
