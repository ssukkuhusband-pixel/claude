## 실패/종료 원인 상세 분석

### 그룹 A: NFT/블록체인 논란 → 신뢰 붕괴

#### 12. Ni no Kuni: Cross Worlds
- **출시 첫 2주 $100M 돌파** — 스튜디오 지블리 IP 파워 + Netmarble 마케팅 화력이 맞물린 폭발적 출발
- **실패 핵심**: MARBLEX 블록체인 생태계 도입 발표 → 최고급 장비를 크립토 샵에서만 구매 가능한 구조 → P2W 극단화
- 봇 침범으로 서버 큐 폭발(정상 유저 몇 시간 대기) → Netmarble의 해결책이 "현금 결제" 유도 → 여론 폭발
- App Store/Google Play 별점 조작 논란(인센티브 제공으로 별점 끌어올리기)
- 결과: 개발사-유저 신뢰 완전 파괴. 수익은 유지되나 게임성 평판 회복 불가 상태
- **교훈**: 강한 IP도 투명성 파괴되면 순식간에 역풍. NFT 도입은 비(非)크립토 게임 유저에게 거부감이 치명적

- **글로벌 첫 달 $80~100M** — "Genshin 킬러"로 홍보하며 대규모 유입
- **실패 핵심**: 오픈월드 MMO와 가챠 수집형의 정체성 혼재 — Genshin 팬도, MMORPG 팬도 메인 타깃으로 삼기 어렵
- 표절 논란: Genshin 리뷰를 자사 App Store 리뷰로 유용했다는 의혹
- 초기 버그/서버 불안정 + 중급 기기 최적화 실패
- 캐릭터 정체성 부재: 플레이어들이 2B(NieR), Hololive 버추얼 유튜버 스킨으로 자신을 표현하는 현상 → 고유 캐릭터 IP 부재 노출
- **교훈**: "경쟁작 킬러" 마케팅은 단기 유입 후 역풍. 독자 포지션이 없으면 경쟁작 이탈 유저를 잡지 못함

### 그룹 C: 라이브서비스 개발 역량 부족 → 조기 종료

#### 14. Exos Heroes
- 3년 운영 후 2023.05 종료
- Season 5 업데이트에서 기존 시스템 대폭 변경 → 숙련 유저 이탈 + 커뮤니티 반발
- 개발팀 자체 고백: **"빠른 업데이트 속도를 유지하면서 안정적인 구조를 만드는 것이 불가능했다"**
- 결과: 라이브서비스 개발 체력 고갈 → 종료 선택
- **교훈**: 라이브서비스는 개발 속도와 구조 안정성의 균형이 생존 조건. 팀 규모 이상의 업데이트 속도는 기술 부채로 이어짐

#### 17. Echoes of Mana
- 출시 1년 만에 종료 — Square Enix 모바일 전략 실패의 전형
- 피크 이후 수익이 **피크의 절반에도 미치지 못하는** 급락 → 유저 재방문 설계 부실
- "Mana 시리즈 팬" 타깃이었으나 Mana IP 자체의 글로벌 팬덤 규모가 F2P 라이브서비스 유지에 불충분
- 경쟁작 대비 차별화 없는 2D 액션 + 가챠 구조
- **교훈**: IP 팬덤 규모가 라이브서비스 BM을 지탱하기에 충분한지 사전 검증 필수. "IP 있음"이 자동으로 장기 수익 보장하지 않음

#### 19. Overhit
- 글로벌 출시 후 1년 만에 종료(2019 출시 → 2020.07 글로벌 종료)
- 고품질 언리얼4 3D 그래픽이 중저가 기기 접근성 차단
- 아시아권(KR/JP)에서는 호평이었으나 글로벌 시장 타겟팅 부재
- Nexon 글로벌 게임 잇따른 조기 종료로 브랜드 신뢰도 하락(HIT, Overhit 연속 철수)
- **교훈**: 그래픽 품질과 모바일 접근성 사이의 균형. 아시아 시장 성공이 글로벌 성공을 보장하지 않음

### 그룹 D: IP 의존 + 라이브서비스 수익성 부재

#### 20. KonoSuba: Fantastic Days
- 애니메이션 팬덤 IP로 JP에서 중간 수준 성공 → 글로벌 확장 후 3.5년 운영
- 퍼블리셔 교체(Nexon → Sesisoft) 자체가 사업 불안정 신호
- 풀 보이스 이벤트 유지 비용 + 정기 업데이트 요구 → 비용 대비 수익 미달
- "불가피한 상황"으로만 종료 발표 → 명확한 이유 미공개
- **교훈**: 애니 IP 가챠는 IP 인기 사이클과 라이브서비스 수명이 불일치하는 리스크 내포. 팬덤이 이탈하면 무너지는 구조

#### 21. Mega Man X DiVE
- Capcom 분기보고서에서 모바일 매출 증가 기여 — 수익 기여는 있었음
- 그러나 신규 캐릭 밸런스 붕괴: **"0스타 신규 캐릭이 만렙 구캐를 압도"** → 가챠 강제 비판 폭발
- Mega Man Battle Network 리마스터 컬렉션의 흥행 → "Capcom이 Mega Man을 제대로 만드는 것을 봐버린" 유저 이탈
- 오프라인 버전(Mega Man X DiVE Offline)으로 전환하여 Steam 출시 → 라이브 종료 명분 확보
- **교훈**: 가챠 게임 밸런스 붕괴는 과금 유저를 포함한 전체 유저 이탈로 이어짐. "완전 유료 오프라인 전환"이 매력적인 대안이 될 수 있음

#### 22. Disgaea RPG
- 일본에서 수년 운영 — Disgaea 팬덤 기반 안정적 소규모 수익
- 기술적 문제: 출시 후 수개월간 서비스 중단 및 재론칭 이력
- 중국 서버 2022.12 종료 → 글로벌 2023.05 연이어 종료
- Boltrend(퍼블) 역량 한계 + ForwardWorks(원개발) 지원 부족
- Disgaea 주 팬층(하드코어 SRPG 팬)이 모바일 가챠 소비 성향과 미스매치
- **교훈**: IP 팬덤의 성향(하드코어 vs. 캐주얼)이 가챠 BM 소비 패턴과 일치하는지 검증 필수

### 그룹 E: 글로벌 시장 적합성 부재

#### 23. SINoALICE
- **누적 $340M+로 경제적으로는 상당히 성공한 게임** — "실패"보다 "글로벌 맞춤화 실패"에 가까움
- 일본 전용 핵심 게임플레이: **길드 전쟁 "Colosseum"이 실시간 동시접속 필수** → 시간대가 다른 글로벌 유저에게 치명적 진입장벽
- Square Enix 모바일 구조조정 시기와 겹쳐 글로벌 서버 우선 정리
- 일본 서버는 여전히 운영 중 → "글로벌화 실패"가 정확한 진단
- **교훈**: 일본 핵심 메타(실시간 동시 길드전)는 글로벌 시간대 다양성과 충돌. 글로벌 라이브서비스는 비동기 사회 콘텐츠 중심 설계가 더 유리

#### 24. World Flipper
- Cygames의 소규모 실험작 — 핀볼 메커닉의 독창성
- 일본 내에서는 안정적 소규모 팬덤 확보
- 글로벌 확장 후 JP 대비 낮은 수익 → JP+글로벌 동시 종료 결정
- 스토리 메인 퀘스트 챕터 12로 완결 후 오프라인 버전 제공 — 양호한 종료 처리
- **교훈**: 독창적 메커닉(핀볼)은 차별화 포인트이나 글로벌 수익성을 보장하지 않음. 장르적 생소함이 신규 유저 진입 장벽이 될 수 있음

#### 25. Dragalia Lost
- **첫 1년 $106M — 상업적으로 성공적인 출발**
- 오리지널 IP의 장기 팬덤 형성 실패: Fire Emblem Heroes(기존 IP)가 더 강한 Nintendo 모바일 성과를 냈음
- 비P2W 설계(칭찬받았으나) → 수익 창출 한계 → 서비스 유지 비용 미달
- 단말기 지원 종료(iOS 15 이상 요구) → 구형 기기 유저 이탈
- Cygames 경영진의 공개 반성: "단일 타이틀 의존도가 너무 높았다"
- **교훈**: 비P2W + 오리지널 IP의 조합은 유저 호감도는 높지만 장기 수익성 확보가 어려움. "착한 가챠"도 수익이 없으면 서비스 불가

## 성공 중소형 게임 핵심 교훈 분석

### 1. Archero 2 (성공)
- 전작 Archero의 방대한 유저베이스 + 수집형 요소 강화로 ARPPU 향상
- 단순 조작(로그라이트)에 복합 가챠 레이어 추가 → 다양한 소비 접점 확보
- **교훈**: 기존 게임의 "수집형 요소 부재"를 신작에서 보완 → 기존 IP 활용 + 새로운 BM 구조

### 2. Memento Mori (성공)
- **업계 평균 2배 가격 가챠** — 고가 전략이 성공한 이유: 소수 고과금 타깃 마케팅
- 조작 없는 완전 방치형 → 시간 투자 없이 과금만으로 즐길 수 있는 "바쁜 성인" 타깃
- JP 시장에서 광고비 $0으로 첫 달 $40M 돌파 — 입소문과 커뮤니티 확산
- **교훈**: 타깃을 "고소득 바쁜 성인"으로 좁히면 고가 BM이 통할 수 있음. 조작 부담 제거가 핵심

### 3. Another Eden (성공/생존)
- 완전 싱글 플레이어 JRPG + 가챠 조합 — 경쟁이 없는 틈새
- Masato Kato(Chrono Trigger 시나리오 작가) 시나리오 — 스토리 충성 팬 확보
- PvP/길드 없음 → 느린 속도로 즐기는 라이트 RPG 팬층 유지
- **교훈**: "소셜 없는 순수 싱글 RPG 가챠"는 경쟁이 덜한 틈새. 스토리 품질이 유저 충성도 유지 핵심

### 4. Path to Nowhere (성공/생존)
- Arknights와 유사한 전략 디펜스이나 다크/범죄 세계관 + 여성향 미형 캐릭으로 차별화
- **가장 관대한 가챠 중 하나**(2% 기본율, 80연 천장) → 과금 거부감 최소화
- 소셜 없음 → 복잡성 제거, 순수 전략+스토리 집중
- **교훈**: 경쟁작보다 관대한 가챠는 커뮤니티 호감도 → 자발적 홍보 → 신규 유입 선순환

## 성공 vs. 실패 비교 매트릭스

| 요인 | 성공 패턴 | 실패 패턴 |
|------|---------|---------|
| **IP 전략** | 자체 IP 구축 or 관대한 가챠로 팬덤 생성 | 원작 IP에만 의존, 게임 자체 경쟁력 부재 |
| **가챠 설계** | 80연 이하 천장, 2% 이상 기본율 | 120연 이상 천장, P2W 극단화, 봇/NFT 오염 |
| **전투 정체성** | 장르 내 명확한 포지션(방치/전략/액션 택1) | 오픈월드+MMO+가챠의 정체성 혼재 |
| **라이브서비스 역량** | 팀 규모에 맞는 지속 가능한 업데이트 주기 | 팀 한계를 초과하는 업데이트 부채 → 시스템 붕괴 |
| **소셜 설계** | 없거나 비동기(길드원 방치 배틀 등) | 실시간 동시접속 필수 PvP → 시간대 충돌 |
| **수익 모델** | 캐릭 수집 + 코스튬 + 시즌패스 복합 | 단일 가챠 의존, 또는 NFT/P2W 극단화 |
| **플랫폼 적합성** | 모바일 최적화, 저사양 호환 | 고사양 3D 강요로 접근성 제한 |
| **글로벌화** | 비동기 콘텐츠 중심 글로벌 설계 | JP 전용 메타(실시간 길드전)를 글로벌 강요 |

## 핵심 인사이트 요약

### 인사이트 1: "천장 높이"가 유저 신뢰의 첫 번째 신호
- 80연 이하 천장 게임(Path to Nowhere, Archero 2 등): 커뮤니티 호감 → 장기 생존
- 120연 이상 천장 게임(Dislyte 120연, Dragalia Lost 추정 120~200연): 수익 집중 구간이 길어 F2P 이탈 가속

### 인사이트 2: IP는 "초기 폭발"만 보장, "장기 수익"은 게임 자체가 결정
- Ni no Kuni(지블리 IP): 첫 2주 $100M → 이후 수익 급락
- Echoes of Mana(Mana IP): IP 팬덤 규모 자체가 F2P 라이브서비스 BM을 지탱하기에 부족
- KonoSuba(애니 IP): 애니 시즌 종료와 함께 신규 유입 소멸
- **결론**: IP는 CAC(고객획득비용)를 낮추지만 LTV(유저 생애가치)는 게임 자체 품질이 결정

### 인사이트 3: 완전 방치형은 "바쁜 고과금 성인"이라는 블루오션을 개척
- Memento Mori: 조작 없음 + 고가 BM = 일반 가챠 유저가 아닌 다른 세그먼트 공략
- 이 포지션은 Genshin/Arknights와 경쟁하지 않음 → 틈새 독점 가능

### 인사이트 4: NFT/블록체인은 2022~2023년 기준 "게임 가치 파괴 요소"
- 적용 사례 모두 유저 이탈 + 게임 평판 붕괴 → 비(非)크립토 게이머 시장에서 블록체인은 신뢰 파괴 신호

### 인사이트 5: 글로벌 시장에서 "실시간 동시접속 PvP"는 양날의 검
- JP 국내 성공작의 실시간 길드전 메타(SINoALICE, 일부 KR MMORPG)는 글로벌 시간대 분산 환경에서 치명적 진입장벽
- 글로벌 성공한 PvP는 비동기(아레나 자동전 결과) 또는 지역별 서버 분리 방식
