# 전투 메커닉 실패/외면 사례 분석

> 조사일: 2026-03-30 | 조사 대원: 참치김밥+땡초김밥 (Sonnet 에이전트)

## 모바일에서 "절대 작동 안 하는" 전투 5유형

1. **정밀 멀티핑거 커맨드** (격투게임 236P류) — 인식률 60% 이하
2. **20분+ 세션 전략 전투** (정통 MOBA, RTS) — 모바일 세션 5~8분과 불일치
3. **AR 실시간 전투** — 배터리/햇빛/이동위험/타인시선 4중 장벽
4. **드로잉/제스처 인식 전투** — 인식 실패 1회 = 재미 초기화
5. **에이전시 0% 완전 자동** — "전투"로 포장하면 기대 불일치로 이탈

## 카테고리별 실패 사례

### 드로잉/제스처 기반
- Infinity Blade: 기술적 성공이나 생태계 소멸 ($30M+ 후 앱스토어 삭제)
- Arcane Legends: 정밀 드래그 → 모바일 환경 불일치
- Arx Fatalis: 룬 드로잉 인식률 문제 → "게임 최악의 요소"

### AR 기반
- Walking Dead Our World: AR 전투 = "서서 핸드폰 들기"의 불편함
- Witcher Monster Slayer: 복잡한 준비 시스템 + AR = 2023 종료
- Jurassic World Alive: 수집은 AR, 전투는 일반 → 경험 분열

### MOBA 이식 실패
- Vainglory: 좋은 전투, 세션 20분 = 모바일 불일치 → 서버 위기
- Solstice Arena: 5분 MOBA 맞는 방향 + 에너지 BM이 파괴
- LOVA: MOBA+카드 = 양쪽 모두 애매

### 격투/액션
- Street Fighter IV Mobile: 커맨드 입력 = 터치 물리적 불가
- Mortal Kombat Mobile: 과도한 단순화 → "격투인 척 카드게임"
- LawBreakers: 참신한 전투 + 진입장벽 + Overwatch 타이밍 = 즉사

### Genshin 클론
- Tower of Fantasy: "Genshin 킬러" = 비교에서 짐 (법칙 19)
- Noah's Heart: 전투 OK + 라이브 운영 부재 → 2024 종료
- Dragalia Lost: 전투 좋음 + 수집 동기 약함 → 2022 종료

## 실패 6법칙

```
A: 전투 참신함은 "10초 이해" 필수
B: 세션 길이 ↔ 전투 복잡도 반드시 매칭
C: BM이 전투 리듬 파괴 → 유저는 "전투가 재미없다"고 인식
D: "더 나은 경쟁자"는 없음. "다른 게임"이어야 함
E: 드로잉/정밀 제스처/AR = 현재 기술 구조적 제약
F: 전투 참신함 ≠ 수집/성장 동기 (Dragalia Lost 교훈)
```

## 참신함의 임계점: 넘은 전투 vs 못 넘은 전투

| 구분 | 넘은 전투 | 못 넘은 전투 |
|------|---------|-----------|
| 유저 반응 | "처음 봤는데 바로 이해됨" | "독특한데 XX랑 뭐가 다르지?" |
| 설명 필요 | 10초 이내 | 튜토리얼 필수 |
| 차별화 | "비슷한 게임이 없다" | "XX보다 완성도 낮다" |
| BM 연결 | 전투 자체가 수집 욕구 자극 | 전투와 BM이 분리 |
