# 캐주얼 게임 설계 원칙 조사

> 조사일: 2026-03-30
> 출처: Deconstructor of Fun, Naavik, GameRefinery, PocketGamer, Sensor Tower, Liftoff, Tenjin

## 핵심 원칙 요약

### 코어 루프
- 1판 1~3분(퍼즐), 30초~1분(하이퍼캐주얼), 5~15분(하이브리드)
- 난이도: 플로우 이론 + 니어 미스(Near Miss) 설계 = Candy Crush 핵심
- "한 판만 더" 3요소: 진행 가시화, 변수 보상, 손실 회피
- 론칭 시 200~300레벨, 5레벨마다 새 요소
- Playrix 공식: 코어 퍼즐 + 메타 서사 = 장기 리텐션

### BM
- Royal Match 광고 제로 → RpD $55, 매치3 IAP 51% 독점
- Candy Crush "추가 이동" = 역대 최고 전환 장치 (니어 미스 직후)
- 하이브리드 캐주얼: IAP 50~60% + 광고 40~50%
- ATT 이후 하이퍼캐주얼 eCPM 30~40% 감소 → 하이브리드 전환 필수

### 소셜
- Candy Crush 초기: Facebook 연동 → 무료 UA 바이럴 엔진
- Monopoly GO $6B: 공격/방어 + 스티커 트레이딩 + 파트너 이벤트 + 주사위 링크 공유
- Royal Match 팀 기능 + Hidden Temple → 수익 25% 증가
- Match Masters PvP → RPD $5 (일반 캐주얼 2~5배)

### 힐링 게임
- 아트 품질이 핵심, BM은 약함 (ARPU $0.5~1)
- 소규모 팀 무기, 대형 투자 부적합
- 트렌드 의존적 수명 (바이럴 버즈 소멸 후 급락)

### 머지 장르
- Merge Dragons $780M+ — 머지 장르 1위
- 중독성 4요소: 완성 욕구, 공간 관리 긴장, 즉각 피드백, 발견의 재미
- 메타 서사 결합 시 수익 크게 증가 (Merge Mansion)

### 2024~2025 트렌드
- Monopoly GO $6B: 보드게임 IP + 소셜 = 폭발
- Triple Match 3D: 2D→3D 시각적 새로움만으로 성공
- AI 개인화: Habby LTV +90%, 리텐션 +33%
- 수집 요소 결합: 도감, 앨범 등 "모으는 것" = 리텐션
