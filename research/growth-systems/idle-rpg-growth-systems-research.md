# 방치형/AFK RPG 모바일 게임 성장 시스템 연구 (2014-2024)

## 연구 개요

2014년부터 2024년까지 주요 방치형(Idle)/AFK RPG 모바일 게임 25종 이상의 성장 시스템을 시간순으로 분석한다. 프레스티지/환생 시스템, 오프라인 vs 온라인 성장 밸런스, 성장 깊이의 점진적 확장, 그리고 장르의 진화 과정에 초점을 맞춘다.

---

---

### 1. Clicker Heroes (2014)

| 항목 | 내용 |
|------|------|
| 출시 | 2014년 8월 (Flash, 이후 모바일) |
| 개발사 | Playsaurus |
| 매출/다운로드 | 전 플랫폼 누적 1억 회 이상 플레이 |

**핵심 성장 시스템:**
- **영웅 고용 시스템**: 54명의 고유 영웅을 골드로 고용/업그레이드. 영웅들이 자동으로 DPS를 생성하여 오프라인에서도 진행
- **존(Zone) 진행**: 점진적으로 어려워지는 존을 클리어하며 보스 처치로 진행
- **길드(Gild) 시스템**: 특정 영웅에 길드를 집중시켜 DPS 효율 극대화

**방치 전용 성장 메카닉:**
- 오프라인 중 영웅들이 자동 공격하여 골드/경험치 축적
- 접속 시 누적된 오프라인 보상 일괄 수령
- 초기에는 오프라인 진행 속도가 온라인 대비 매우 제한적

**프레스티지 시스템 (2단계 구조 - 장르 최초):**

1. **어센션(Ascension) - 1차 프레스티지**: 영웅과 존 진행도를 리셋하고 "히어로 소울(Hero Souls)"을 획득. 히어로 소울 1개당 DPS 10% 보너스. 히어로 소울로 "에인션트(Ancients)"를 소환하여 골드 보너스, 방치 보너스 등 영구 패시브 획득
2. **트랜센던스(Transcendence) - 2차 프레스티지**: 에인션트, 길드, 히어로 소울까지 모두 리셋하고 "에인션트 소울(Ancient Souls)" 획득. 에인션트 소울로 "아웃사이더(Outsiders)" 업그레이드 가능. 아웃사이더는 소울 획득량, 방치 보너스, 골드 수입, 전체 속도를 매크로 레벨에서 증가

**성장 레이어 스택:**
```
[레이어 4] 아웃사이더 (트랜센던스 보상) - 메타 레벨 패시브
    ↑ 트랜센던스 리셋 (에인션트 소울 획득)
[레이어 3] 에인션트 (히어로 소울로 구매) - 영구 패시브 버프
    ↑ 어센션 리셋 (히어로 소울 획득)
[레이어 2] 영웅 레벨/길드 - DPS 생산
[레이어 1] 골드 축적 → 영웅 고용/업그레이드
```

**유저 반응:**
- 2단계 프레스티지 시스템은 방치형 게임의 표준 공식이 됨
- 트랜센던스 도입 후 장기 플레이어들의 목표 의식이 크게 강화됨
- 다만 후반부에 어센션 사이클이 지나치게 반복적이라는 피로감도 존재

### 2. Tap Titans (2014)

| 항목 | 내용 |
|------|------|
| 출시 | 2014년 12월 |
| 개발사 | Game Hive (캐나다) |
| 매출/다운로드 | Google Play/App Store 500회 이상 피쳐링, Game Hive 전체 1.8억 다운로드 중 상당 비중 |

**핵심 성장 시스템:**
- **탭 기반 전투**: 화면을 탭하여 타이탄을 공격, 빠른 탭핑이 더 높은 데미지
- **영웅 고용**: 골드로 동료 영웅 고용 및 업그레이드, 자동 DPS 생산
- **4가지 재화 시스템**: 골드, 크리스탈(하드 커런시), 아티팩트, 렐릭

**방치 전용 성장 메카닉:**
- 오프라인 시 고용된 영웅들이 자동 전투
- 오프라인 골드 축적 (온라인 대비 제한적)

**프레스티지 시스템:**
- 프레스티지 시 게임 진행도 리셋, 크리스탈/아티팩트/렐릭은 유지
- 렐릭 획득량은 영웅 레벨에 비례
- 렐릭으로 아티팩트 구매 (랜덤 획득 - 선택 불가)
- 아티팩트가 영웅/동료의 스탯을 영구 증가

**성장 레이어 스택:**
```
[레이어 3] 아티팩트 (렐릭으로 구매) - 영구 스탯 증가
    ↑ 프레스티지 리셋 (렐릭 획득)
[레이어 2] 영웅 레벨 - 자동 DPS 생산
[레이어 1] 탭 데미지 + 골드 축적
```

**유저 반응:**
- Clicker Heroes와 함께 모바일 방치 RPG 장르의 기틀을 마련
- 단순하지만 중독성 있는 탭-프레스티지 루프가 캐주얼 유저를 강하게 끌어들임
- "아티팩트 랜덤 획득"에 대한 불만 존재 (운에 따라 진행 속도 차이)

### 3. 중년기사 김봉식 (2015)

| 항목 | 내용 |
|------|------|
| 출시 | 2015년 5월 2일 (Google Play), 7월 1일 (App Store) |
| 개발사 | MAF Games (한국) |
| 매출/다운로드 | 누적 다운로드 약 460만, 누적 매출 약 23억 원 |
| 개발 기간 | 4주 |

**핵심 성장 시스템:**
- **완전 방치형 던전 돌파**: 김봉식이 알아서 던전을 돌파하는 순수 방치형
- **플레이어 역할**: 게임 외적 퀘스트 수행, 금액 관리, 열쇠/보석 활용하여 김봉식 강화
- **장비 강화**: 무기/방어구 수집 및 강화 시스템

**방치 전용 성장 메카닉:**
- 100% 방치 기반 - 플레이어 직접 조종 불필요
- 자동 던전 진행 및 골드 수집
- 접속 시 축적된 보상 수령

**한국 방치형의 원형:**
- 한국 시장에서 "원조 방치형 근본겜"으로 불림
- 4주 개발이라는 초단기 개발 사이클로 성공
- 2015년 10월 일본 진출 (Polaris X 사 퍼블리싱)
- 이후 한국 방치형 RPG 붐의 시발점

**유저 반응:**
- 극도로 단순한 조작이 한국 모바일 게임 시장에서 통함
- "출퇴근길에 켜놓기만 하면 되는 게임"으로 직장인 타겟 적중
- 성장 시스템 자체는 단순하나, 한국형 방치 RPG의 공식을 확립

### 4. Idle Heroes (2016)

| 항목 | 내용 |
|------|------|
| 출시 | 2016년 6월 |
| 개발사 | DroidHang Games (DHGames, 중국) |
| 매출/다운로드 | 출시 3개월 일평균 $1,075 → 2018년 1월 일평균 $264,663. 2021년 7월 기준 월 $3M, 7만 다운로드. 첫 해 $100M 이상 달성 |

**핵심 성장 시스템:**
- **영웅 수집/가차**: 아시아 메타게임 디자인 기반. 영웅을 수집하고 합성
- **별(Star) 등급 시스템**: 별 등급이 최대 레벨을 결정. 5성부터 다른 캐릭터를 희생(가차 퓨전)하여 승급
- **골드 + 승급석(Promotion Stones)**: 캐릭터 업그레이드 이중 재화

**방치 전용 성장 메카닉:**
- 오프라인 중 소프트 커런시(골드, 경험치) 지속 축적
- 접속 시 방치 보상 일괄 수령 → 긍정적 감정 유도
- 방치 보상이 유일한 레벨업 리소스 (구매로 대체 불가한 시간 리소스)

**성장 레이어 스택:**
```
[레이어 5] 가차 합성 (5성+ 영웅 제작)
[레이어 4] 장비/아티팩트 시스템
[레이어 3] 영웅 승급 (별 등급 상승)
[레이어 2] 영웅 레벨링 (골드 + 승급석)
[레이어 1] 방치 보상 축적 (골드/경험치/장비)
```

**핵심 설계 원칙 - "모든 것이 사용된다":**
- 가차에서 나온 모든 것을 나중에 활용 가능 (합성 재료)
- 이 깊은 시스템이 유저 과금을 유도하는 핵심 동력
- Idle Heroes의 수익 모델이 이후 방치 RPG의 표준이 됨

**유저 반응:**
- 방치 보상으로만 성장 가능한 구조가 "과금 없이도 즐길 수 있다"는 인식 형성
- 동시에 빠른 성장을 원하면 가차 과금으로 합성 재료 확보 필요
- 중국/글로벌 시장에서 방치 RPG = Idle Heroes 공식 확립

### 5. Tap Titans 2 (2016)

| 항목 | 내용 |
|------|------|
| 출시 | 2016년 12월 |
| 개발사 | Game Hive |
| 매출/다운로드 | 전작 대비 대폭 성장, 중국 시장 진출 성공 (Yodo1 퍼블리싱) |

**핵심 성장 시스템:**
- **소드 마스터 레벨**: 프레스티지를 위해 레벨 600 필요 (약 스테이지 100)
- **6개 스킬 트리**: Knight(탭 데미지), Summoner(펫 데미지), Warlord(영웅/클랜십 데미지), Sorcerer(마나/쉐도우 클론), Rogue(치명타/단검), Alchemist(골드 멀티플라이어)
- **아티팩트 시스템**: 스테이지 60 이상에서 프레스티지 렐릭으로 구매/업그레이드. 해금 비용 기하급수적 증가
- **펫 시스템**: 전투 보조 + 패시브 보너스
- **클랜 시스템**: 클랜 레이드, 클랜십 데미지

**프레스티지 시스템 (전작 대비 진화):**
- 프레스티지 시 렐릭 획득 → 아티팩트 해금/업그레이드
- 스킬 포인트 보상 → 스킬 트리 투자
- 매 프레스티지마다 더 높은 스테이지 도달 가능

**성장 레이어 스택 (6중 레이어):**
```
[레이어 6] 클랜 보너스 (협동 레이드)
[레이어 5] 스킬 트리 (6개 빌드 경로)
[레이어 4] 아티팩트 (렐릭으로 업그레이드)
    ↑ 프레스티지 리셋
[레이어 3] 펫 보너스 (패시브)
[레이어 2] 영웅 레벨/스킬
[레이어 1] 소드 마스터 탭 데미지 + 골드
```

**유저 반응:**
- 전작의 단순 탭-프레스티지에서 6중 성장 레이어로 대폭 진화
- 스킬 트리 도입으로 "빌드 다양성" 확보 → 장기 리텐션 향상
- 클랜 시스템이 소셜 접착제 역할
- 다만 후반 프레스티지 사이클이 반복적이라는 피드백 존재

### 6. Nonstop Knight (2016)

| 항목 | 내용 |
|------|------|
| 출시 | 2016년 6월 |
| 개발사 | Flaregames (독일) |
| 매출/다운로드 | 출시 초기 상위권 기록 |

**핵심 성장 시스템:**
- **자동 던전 진행**: 영웅이 자동으로 던전을 달리며 전투
- **3가지 스킬 버튼**: 유일한 플레이어 인터랙션 (특수 공격 타이밍 조절)
- **장비 가차**: 보스 처치 시 상자 드롭. 무기(공격력)/방어구(방어력)/망토(스킬 파워), 1~3성 등급 + 추가 속성
- **3층마다 보스**: 보스전은 반드시 접속 상태에서만 진행 가능

**방치 전용 성장 메카닉:**
- 오프라인 시 잡몹 자동 처치 + 골드 수집 (최대 2층 진행)
- 보스전 필수 접속 → 오프라인 진행에 명확한 상한선
- Quick Battle: 2시간 분량의 그라인드를 5분으로 압축 + ATK 10% 보너스

**프레스티지 시스템:**
- 비용 상승 속도 > 골드 획득 속도 → 자연스러운 프레스티지 유도
- 첫 회차 30층 → 다음 45층 → 1주 후 150층으로 점진적 진행
- 프레스티지 시 영구 스탯 업그레이드 획득

**유저 반응:**
- 액션 RPG + 방치형의 성공적 결합
- "보스전 필수 접속" 설계가 방치형에서의 능동적 참여 밸런스를 잘 잡음
- 장비 가차의 랜덤성이 진행의 변화를 줌
- 후속작 Nonstop Knight 2 출시되었으나 원작만큼의 성공은 못 거둠

### 7. Grow Castle (2016)

| 항목 | 내용 |
|------|------|
| 출시 | 2016년 4월 |
| 개발사 | RAON Games (한국) |
| 매출/다운로드 | Google Play 1억+ 다운로드, 장기 운영 |

**핵심 성장 시스템:**
- **타워 디펜스 + 방치 하이브리드**: 성을 방어하며 웨이브를 격퇴
- **120+ 영웅 시스템**: 각 영웅 고유 스킬, 성 타워 각 층에 배치
- **타운 아처**: 업그레이드할수록 점진적으로 강력해지는 핵심 유닛
- **콜로니 시스템**: 식민지 건설 + 노동자 고용 → 골드 수입 증가

**방치 전용 성장 메카닉:**
- 오프라인 중 리소스 축적
- 콜로니의 자동 골드 생산
- 접속 시 축적된 자원으로 영웅 업그레이드/해금

**성장 레이어 스택:**
```
[레이어 4] 아티팩트 시스템
[레이어 3] 영웅 승급/스킬 해금 (120+ 영웅)
[레이어 2] 성 업그레이드 + 유닛 배치 전략
[레이어 1] 웨이브 방어 → 골드 축적 + 콜로니 수입
```

**유저 반응:**
- 한국 개발 + 글로벌 인기의 롱런 방치 타워 디펜스
- 전략적 유닛 배치 + 방치 성장의 조합이 중독적
- 단순하지만 깊은 시너지 시스템 (유닛 조합에 따른 효율 차이)
- 꾸준한 업데이트로 10년 가까이 운영 중

### 8. Almost a Hero (2017)

| 항목 | 내용 |
|------|------|

**핵심 성장 시스템:**
- **마법 반지(Ring) 시스템**: 반지를 선택하고 "쓸모없는 바보들"과 함께 적 호드에 맞섬
- **영웅 캐릭터 해금**: 여러 영웅 + 강력한 반지 + 펫(고유 능력) 해금
- **영구 아티팩트 업그레이드**: 다음 회차를 조금씩 덜 치명적으로 만듦
- **룬(Rune) 시스템**: 반지에 장착하는 아이템. 스테이지 진행(고정 룬) 또는 에픽 상자(랜덤 룬)로 획득

**프레스티지 시스템 (관대한 설계):**
- 프레스티지 시 잃는 것: 코인, 현재 반지/캐릭터 레벨, 스테이지 진행도
- 프레스티지 시 유지되는 것: 스크랩, 룬, 아이템, 반지, 영웅, 미스스톤, 젬, 해금 내역
- 프레스티지가 캐릭터 스킬 포인트 재분배 기회 제공
- Compass 아이템으로 이전 진행 지점까지 빠르게 복귀 가능

**성장 레이어 스택:**
```
[레이어 5] 영구 아티팩트 (미스스톤으로 업그레이드)
[레이어 4] 룬 (반지 강화)
    ↑ 프레스티지 리셋 (거의 손실 없음)
[레이어 3] 반지 레벨/효과
[레이어 2] 영웅 스킬 트리 + 레벨
[레이어 1] 스테이지 진행 → 코인/경험치
```

**유저 반응:**
- "가장 관대한 프레스티지 시스템" - 리셋 시 손실이 거의 없어 부담감 최소화
- 유머러스한 "쓸모없는 영웅들" 컨셉이 캐주얼 유저에게 어필
- 인크리멘탈 게임 커뮤니티에서 높은 평가
- 프레스티지의 부담 없음 → 자주 프레스티지 → 빠른 성장 체감 → 높은 리텐션

### 9. AFK Arena (2019)

| 항목 | 내용 |
|------|------|
| 출시 | 2019년 (소프트 런치 2018년 6월) |
| 개발사 | Lilith Games (중국) |
| 매출/다운로드 | 소프트 런치 기간 140만 DL / $2.6M → 2020년 $14.5억 매출(전년 대비 124% 성장). 총 4,847만 DL, 1억 명 이상 플레이. 2020년 F2P 매출 8위, RPG 매출 1위 |

**핵심 성장 시스템 (방치형 RPG의 교과서):**

1. **어센션(Ascension) 시스템**: 영웅의 복사본을 합성하여 등급 상승
   - Elite → Legendary → Mythic → Ascended 단계
   - 각 등급이 레벨 상한선을 결정
   - "어센션 게이트"가 과금의 핵심 동기 → 모든 영웅이 레벨 캡이고 어센션 재료가 부족할 때

2. **공명 수정체(Resonating Crystal)**: 상위 5명의 영웅 레벨이 나머지 전체 영웅에 적용
   - Ascended 등급 영웅만 레벨 240 돌파 가능
   - Ascended 영웅 1명당 레벨 캡 +5 증가

3. **장비 시스템**: 영웅 클래스 기반 장비, 장비 업그레이드가 동일 클래스 전체에 적용

4. **아케인 래비린스(Arcane Labyrinth)**: 로그라이크 던전 + 에센스 보상

5. **킹스 타워(King's Tower) + 팩션 타워**: 단계적 난이도 상승, 일회성/주기적 보상

**방치 전용 성장 메카닉 (혁신적 설계):**
- AFK 메카닉이 유일한 핵심 리소스 공급원
- 히어로 에센스, 골드, 경험치 등 레벨업 자원은 오직 방치 보상으로만 획득
- 과금으로 이 리소스를 직접 구매할 수 없음 → "시간"이 진정한 재화
- 결과: 과금만으로는 팀 전투력을 올릴 수 없고, 방치 시간이 반드시 필요

**성장 레이어 스택 (다중 레이어):**
```
[레이어 7] 시그니처 아이템 + 가구(Furniture) 시스템
[레이어 6] 킹스 타워/팩션 타워 (난이도 별 보상)
[레이어 5] 아케인 래비린스 (로그라이크 + 보상)
[레이어 4] 공명 수정체 (Ascended 영웅 수 기반 레벨 캡)
[레이어 3] 어센션 (영웅 복사본 합성)
[레이어 2] 장비 업그레이드 (클래스 공유)
[레이어 1] AFK 보상 축적 (방치 시간 = 핵심 자원)
```

**점진적 깊이 확장 (압도하지 않는 설계):**
- 초기: 영웅 수집 + 레벨업만 집중
- 중반: 어센션 게이트 등장 → 팩션 밸런스 고려 시작
- 후반: 공명 수정체 최적화, 시그니처 아이템, 가구 시스템 순차 해금
- 각 시스템이 특정 진행 단계에서 자연스럽게 해금되어 정보 과부하 방지

**유저 반응:**
- "방치형인데 전략적" - 캐주얼과 하드코어 양쪽 만족
- 과금으로 시간을 살 수 없는 구조가 F2P 친화적이라는 인식 형성
- 어센션 게이트에서의 과금 유도가 자연스러움
- 장기 리텐션이 매우 높음 (2020년 거의 100만 DAU)

|------|--------|----------------|------|

| 1세대 (2014) | Tap Titans | 1단계 프레스티지 | 단순 리셋 → 아티팩트 획득 |

**핵심 진화 트렌드:** 초기 "전면 리셋 → 보상" 구조에서, 점차 "리셋 없이 레이어 추가" 또는 "시즌 기반 강제 리셋"으로 양극화

### 성장 깊이(Depth)의 점진적 확장 방식

**1단계 - 단순 스택 (2014-2015)**
- Tap Titans/Clicker Heroes: 탭 → 골드 → 영웅 → 프레스티지 → 아티팩트
- 레이어 수: 3~4개
- 깊이 추가: 프레스티지 단계 추가 (트랜센던스)

**2단계 - 다중 축 성장 (2016-2017)**
- Idle Heroes/TT2: 영웅 합성 + 장비 + 스킬 트리 + 가차 + 펫
- 레이어 수: 5~6개
- 깊이 추가: 각 축이 독립적으로 성장 → 플레이어 선택지 증가

**3단계 - 게이트 기반 설계 (2019)**
- AFK Arena/MLA: 어센션 게이트 + 시간 게이트 + 콘텐츠 해금 게이트
- 레이어 수: 7개 이상
- 깊이 추가: 진행 단계별 시스템 순차 해금으로 정보 과부하 방지

**4단계 - 하이브리드 (2021-2022)**
- Cookie Run Kingdom: RPG + 킹덤 빌딩 + 생산 방치
- Survivor.io: 로그라이크 런 + 영구 성장
- 깊이 추가: 장르 결합으로 플레이 스타일 다양화

**5단계 - 극단의 양극화 (2023-2024)**
- Legend of Mushroom: 극단적 단순화 (단일 루프)
- AFK Journey: 극단적 복잡화 (시즌 리셋 + 오토체스 + 오픈월드)

### 결론: 방치형 RPG 성장 시스템의 핵심 교훈

1. **프레스티지의 진화**: "전면 리셋 → 보상"에서 "합성 게이트" → "시간 게이트" → "시즌 리셋"으로 변화. 유저에게 리셋의 고통을 줄이면서 장기 목표를 제공하는 방향으로 진화

2. **오프라인 진행의 황금 비율**: AFK Arena가 증명한 "과금으로 살 수 없는 시간 자원" 모델이 F2P 유저 리텐션의 최적해. 다만 최근에는 세션 기반(Survivor.io)과 극단적 방치(Legend of Mushroom) 양극화 추세

3. **깊이의 점진적 해금**: 7개 이상의 성장 레이어를 한 번에 노출하지 않고, 진행 단계에 따라 순차 해금하는 것이 핵심. AFK Arena의 "어센션 → 공명 수정체 → 시그니처 아이템 → 가구" 순차 해금이 교과서적 사례

4. **단순함의 힘**: Legend of Mushroom($270M)과 Capybara Go($100M)가 증명한 것처럼, 극도로 단순한 단일 루프도 실행이 완벽하면 복잡한 다중 레이어 시스템을 매출에서 능가할 수 있음

5. **2024년 트렌드**: 시즌 리셋(AFK Journey), 성장 이벤트 로테이션(Capybara Go), 그리고 극단적 단순화(Legend of Mushroom)가 공존. 장르가 성숙하면서 "복잡한 깊이"와 "단순한 중독" 양극단 모두에서 성공 사례 등장
