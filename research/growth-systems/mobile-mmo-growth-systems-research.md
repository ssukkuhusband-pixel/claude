## 1. 연구 개요

### 분석 대상
2017년~2023년 사이 출시된 주요 모바일 MMO RPG 18종의 성장/강화 시스템을 연대순으로 분석한다. 한국 시장을 중심으로, 글로벌 서비스 타이틀도 포함하였다.

### 분석 프레임워크
각 게임별로 다음 항목을 조사한다:
- **기본 정보**: 게임명, 출시년도, 개발사, 대략적 매출/다운로드
- **핵심 성장 시스템**: 레벨 캡, 장비 강화, 각성/초월, PvP 랭킹, 길드 성장
- **MMO 고유 소셜/경쟁 성장 메커니즘**
- **성장 레이어 수와 복잡도**
- **유저 반응**: P2W 논란, 성장 벽 관련 평가

### 2-1. 리니지M (Lineage M, 2017)

| 항목 | 내용 |
|------|------|
| **출시** | 2017년 6월 (한국) |
| **개발사** | 엔씨소프트 (NCsoft) |
| **매출** | 출시 첫해 모바일 매출 약 9,953억원, NC 전체 매출 1조 7,587억원 견인. 리니지 3형제 글로벌 누적 매출 70억 달러 돌파 |
| **장르** | 모바일 MMORPG (PC 리니지1 IP 기반) |

#### 핵심 성장 시스템

**1) 장비 강화 시스템**
- 강화 주문서를 사용하여 무기/방어구의 수치를 상승시키는 전통적 구조
- 안전 강화 구간(+0~+6 정도)을 넘어서면 실패 확률 존재
- 실패 시 강화 수치 하락 또는 장비 파괴 가능 (축복 주문서 사용 시 파괴 방지, 대신 +0으로 초기화)
- 한국형 MMO의 전형적 "확률 강화+파괴" 공식을 모바일에 그대로 이식

**2) 변신(Transform) 시스템**
- 일반 / 고급 / 희귀 / 영웅 / 전설 등급으로 구분
- 합성 확률: 일반->고급 35%, 고급->희귀 17%, 희귀->영웅 12%, 영웅->전설 11%
- 변신에 따라 공격 방식(근거리/원거리/만능) 결정
- 전설 변신은 게임 내 최상위 전투력 차이를 만드는 핵심 요소

**3) 마법인형 시스템**
- 변신과 유사한 등급 체계, 합성을 통한 상위 등급 획득
- 경험치 보너스, 아인하사드의 축복 소모량 감소 등 필수적 패시브 효과
- 상위 등급 마법인형의 유무가 장기적 효율 격차를 크게 벌림

**4) 변신/마법인형 카드 각성**
- 영웅 이상 클래스 카드를 강화하는 초월 시스템
- 남는 카드를 재료로 소비, 최대 12개 투입 가능
- 재료가 많을수록 성공 확률 상승

**5) 스킬 강화**
- 1단계 강화 66.7%, 2단계 33.3%, 3단계 16.67% 성공 확률
- 단계가 올라갈수록 확률이 절반씩 감소하는 전형적 구조

#### MMO 소셜/경쟁 성장
- 혈맹(길드) 시스템: 혈맹전, 공성전을 통한 집단 경쟁
- 필드 PK를 통한 사냥터 점유 경쟁 (원작 리니지1의 핵심 유산)
- 서버 내 정치/외교 시스템이 사실상 성장 효율에 영향

#### 성장 레이어 (약 7~8개)
레벨업 / 장비 강화 / 변신 등급 / 마법인형 등급 / 스킬 강화 / 각성(초월) / 아인하사드의 축복 관리 / 컬렉션

#### 유저 반응
- 극심한 P2W 구조: 전설 변신 1장의 가치가 수백만원대
- "확률형 아이템"의 대명사로 사회적 논란까지 확산
- 그럼에도 노스탤지어와 PK 경쟁의 중독성으로 높은 매출 유지
- 국내 모바일 게임 과금 모델의 "기준점"을 형성한 타이틀

### 2-2. 블레이드&소울 레볼루션 (B&S Revolution, 2018)

| 항목 | 내용 |
|------|------|
| **출시** | 2018년 12월 (한국) |
| **개발사** | 넷마블 (PC 원작: 엔씨소프트 IP) |
| **매출** | 출시 초기 구글플레이 매출 상위권, 이후 점진적 하락 |
| **장르** | 모바일 MMORPG (블레이드&소울 IP) |

#### 핵심 성장 시스템

**1) 무기 성장 (돌파) 시스템**
- 무기 진화/돌파를 통해 단계적으로 상위 무기로 성장
- 돌파에 필요한 핵심 재료가 유료 키트에 포함되어 있어 과금 유도
- 무기 성장 단계가 극도로 높아 무과금 유저의 성장 속도가 현저히 느림

**2) 장비 강화**
- 일정 강화 수치 이상에서 실패 확률 존재
- 넷마블 특유의 다층 재료 요구 시스템

**3) 보패(Soulshield) 시스템**
- PC 블소의 핵심 시스템을 모바일에 이식
- 보패 8부위 세트 효과 + 개별 옵션 랜덤성

#### MMO 소셜/경쟁 성장
- 비무(PvP) 및 전장 시스템
- 문파(길드) 협동 컨텐츠
- 실시간 액션 기반 전투로 컨트롤 요소 존재

#### 성장 레이어 (약 6~7개)
레벨업 / 무기 돌파 / 장비 강화 / 보패 / 기공패 / 스킬 강화 / 장신구

#### 유저 반응
- "과금을 강제하는 아이템 파밍 구조"라는 비판
- 유료 이벤트에 10만원 이상 투입해도 불완전한 장비만 획득
- 무기 성장 진행이 지나치게 느려 무과금 유저 이탈 심각
- PC 블소 IP의 힘으로 초기 유입은 많았으나, 성장 벽으로 유저 유지 실패

### 2-3. 검은사막 모바일 (Black Desert Mobile, 2018)

| 항목 | 내용 |
|------|------|
| **출시** | 2018년 2월 (한국), 2019년 12월 (글로벌) |
| **개발사** | 펄어비스 (Pearl Abyss) |
| **매출** | 글로벌 누적 수천억원대 매출, 2018~2020 한국 모바일 매출 상위권 |
| **장르** | 오픈월드 모바일 MMORPG |

#### 핵심 성장 시스템

**1) 장비 강화 (Enhancement)**
- PC 검은사막의 강화 시스템을 계승
- +15 이후 "프리" / "듀오" / "트리" / "테트" / "펜" 등급으로 확장
- 강화 실패 시 강화 수치 하락 + 내구도 감소 시스템
- "스택 쌓기": 실패 횟수에 따라 다음 성공 확률 보정 (천장 시스템의 원형)

**2) 잠재력 돌파**
- 장비의 등급 자체를 상승시키는 시스템
- 일반 -> 고급 -> 희귀 -> 유일 -> 전설 -> 신화 -> 태초 -> 혼돈 등급 확장
- 돌파 단계마다 새로운 성장 천장이 생성됨

**3) 공명 시스템**
- 착용 장비 중 가장 낮은 강화 등급을 기준으로 추가 효과 부여
- 강화단계 21부터 효과 발동
- "하나만 높이면 안 되고 모든 장비를 균일하게 올려야 한다"는 구조
- 과금 유도와 동시에 "밸런스 있는 성장"을 강제

**4) 각성 시스템**
- 레벨 60 달성 시 각성 무기 해금
- 장비와 장신구도 별도 각성 가능
- 영지(거점) 내 무기공방에서 재료 소모

**5) 연금석 시스템**
- 추가 전투력 상승 요소
- 연금석 등급(심연 등급 추가)에 따른 극단적 전투력 격차
- 연금석 증폭 컨텐츠로 기존 연금석 가치 폐기 -> 유저 반발

**6) 마력 각인, 수정, 기술 교본**
- 다층 부가 성장 요소들이 전투력 수치에 복합적으로 기여

#### MMO 소셜/경쟁 성장
- 거점전/점령전: 길드 단위 영토 경쟁
- 투기장(Arena): PvP 랭킹
- 월드보스: 협동 레이드
- 영지 시스템: 개인 거점 성장 (생활 컨텐츠 연동)

#### 성장 레이어 (약 10~12개)
레벨업 / 장비 강화 / 잠재력 돌파 / 공명 / 각성 / 연금석 / 마력 각인 / 수정 / 기술 교본 / 지식 / 영지 레벨 / 가문 명성

#### 유저 반응
- **"무한 성장"의 대표적 사례**: 성장에 끝이 없어 과금 유저와의 격차가 무한히 벌어짐
- **40:1 무쌍 문제**: 핵과금 유저가 거점전에서 40명을 혼자 제압하는 극단적 P2W
- "성장해서 강해져도 즐길 수 있는 컨텐츠가 한정적"이라는 근본적 비판
- 계정 구매가 과금보다 효율적인 아이러니한 상황 발생
- 그러나 비주얼 퀄리티와 오픈월드의 자유도로 일정 유저층 유지

### 2-4. 메이플스토리M (MapleStory M, 2018)

| 항목 | 내용 |
|------|------|
| **출시** | 2018년 7월 (글로벌) |
| **개발사** | 넥슨 (Nexon) |
| **매출** | 글로벌 서비스, 아시아 시장 중심 수천억원대 누적 |
| **장르** | 2D 사이드스크롤 모바일 MMORPG |

#### 핵심 성장 시스템

**1) 스타포스 강화**
- PC 메이플스토리의 핵심 시스템을 모바일에 이식 (단, 주문의 흔적/주문서작은 제거하고 간소화)
- 장비에 부여된 별(스타포스)의 합산이 캐릭터 스타포스 수치
- 확률에 따라 성공(스타 증가) / 유지 / 하락 / 파괴 4가지 결과
- 파괴 시 "장비의 흔적"만 남으며, 전승 기능으로 스타포스와 잠재능력 이전 가능

**2) 잠재능력 시스템**
- 장비에 랜덤 옵션(잠재능력)을 부여
- 레어 / 에픽 / 유니크 / 레전더리 등급
- 큐브를 사용하여 잠재능력 재설정 (가챠형 시스템)

**3) 장비 등급**
- 노말 -> 레어 -> 에픽 -> 유니크 -> 레전더리 -> 미씩(Mythic)
- 합성/강화를 통한 등급 상승

#### MMO 소셜/경쟁 성장
- 길드 컨텐츠 (길드전)
- 무릉도장/엘보스 등 랭킹 컨텐츠
- 원작의 사회적 커뮤니티 감성 계승

#### 성장 레이어 (약 6~7개)
레벨업 / 스타포스 강화 / 잠재능력 / 장비 등급 합성 / 스킬 강화 / 링크 스킬 / 하이퍼 스탯

#### 유저 반응
- PC판 대비 간소화된 시스템에 대해 "접근성이 좋다" vs "깊이가 없다" 양분
- 스타포스 파괴의 스트레스는 PC판과 동일하게 존재
- 잠재능력 큐브의 가챠 특성이 P2W 요소로 작용
- 2D 횡스크롤이라는 독특한 포지셔닝으로 차별화 성공

### 2-5. 라그나로크 M: 영원한 사랑 (Ragnarok M: Eternal Love, 2018)

| 항목 | 내용 |
|------|------|
| **출시** | 2018년 10월 (동남아), 2019년 (글로벌) |
| **개발사** | Gravity / XD Inc. |
| **매출** | 동남아 시장 중심으로 글로벌 누적 수천억원대 |
| **장르** | 3D 모바일 MMORPG (라그나로크 IP) |

#### 핵심 성장 시스템

**1) 장비 강화 (Enhancement)**
- 베이스 레벨 수치까지만 강화 가능 (레벨 제한)
- 제니(골드)만 소모, 특별 재료 불필요
- 실패 페널티 없음 -> 접근성 높은 기초 시스템

**2) 제련 (Refine)**
- 프론테라 NPC "홀그렌"을 통해 진행
- +4까지 100% 성공, 이후 단계별 확률 감소
- 무기: 오리데오콘, 방어구: 에르늄 소모
- 원석 5개 합성 -> 광석 1개 제작

**3) 티어 업그레이드**
- 제작 장비의 등급 자체를 상승시키는 시스템
- 추가 효과와 스탯 향상

**4) 인챈트 (Enchant)**
- 게펜에서 이용 가능한 특수 효과 부여 시스템
- 랜덤 옵션 부여

**5) 소켓 인챈트**
- 동일 장비를 재료로 카드 슬롯 확장
- 카드 시스템(몬스터 카드 장착)과 연동

#### MMO 소셜/경쟁 성장
- 전통 라그나로크의 파티 플레이 시스템
- 길드전 (WoE) 계승
- MVP 보스 경쟁
- 피로도(스태미나) 시스템으로 일일 성장량 제한

#### 성장 레이어 (약 7~8개)
베이스/잡 레벨 / 장비 강화 / 제련 / 티어 업그레이드 / 인챈트 / 카드 소켓 / 룬 시스템 / 요리/음식 버프

#### 유저 반응
- 제련 +4까지 안전 구간이 있어 초보자 친화적이라는 평가
- 피로도 시스템이 과금 격차를 어느 정도 완화

- 특정 장비를 사용할수록 해당 분야 숙련도 상승

- 풀룻(Full Loot): PvP 사망 시 장비 전체 드랍 -> 장비가 소모품

**3) 경제/제작 시스템**
- 유저 간 거래 중심 경제
- 모든 장비를 유저가 제작
- 자원 채집 -> 가공 -> 제작의 풀 체인

#### MMO 소셜/경쟁 성장
- **길드 영토전 (GvG)**: 영토 점유가 경제적 이익과 직결
- 풀룻 PvP: 사망 시 모든 장비 드랍
- 동맹/적대 관계의 정치적 역학
- 블랙존/레드존의 위험 단계별 컨텐츠

#### 성장 레이어 (약 5~6개)
명성(숙련도) / 장비 티어 / 아이템 퀄리티 / 스펙 빌드 / 은화(경제력) / 길드 시즌 포인트

#### 유저 반응
- **"Pay to Progress"이지 "Pay to Win"은 아니다**라는 커뮤니티 합의
- 프리미엄 구독(명성 50% 보너스, 은화/루트 50% 보너스 등)이 성장 속도 차이 발생
- 프리미엄을 인게임 은화로 구매 가능 -> 무과금도 유지 가능 (단, 난이도 증가 추세)
- 풀룻 PvP로 인해 장비 가치가 "영구 자산"이 아닌 "소모품" -> 과금 장비의 가치가 상대적으로 낮음
- 한국형 MMO와 정반대의 성장 철학: "장비 축적"이 아닌 "스킬 축적"

아이템 레벨(재련) / 상급 재련 / 초월 / 엘릭서 / 보석 / 각인 / 카드 / 아크 패시브 / 트라이포드 / 어빌리티 스톤 / 악세사리 품질 / 장비 세트 효과

#### 유저 반응
- **아티산 에너지(천장 시스템)가 업계 표준을 제시**: "100% 실패해도 언젠가는 성공"
- 그러나 천장까지의 비용이 막대하여 사실상 "확률형과 다를 바 없다"는 비판도
- 초월/엘릭서 등 과도한 성장 레이어 누적에 대한 피로감 -> 결국 삭제 결정
- **"성장 시스템 레이어를 너무 많이 쌓으면 결국 삭제해야 한다"는 교훈**
- 레이드 중심 컨텐츠가 강화/성장의 목적성을 부여하는 데 성공
- 모바일 MMO들이 "천장 시스템", "재련 경험치", "계승" 등의 개념을 참고

## 3. 한국형 MMO 강화 시스템의 핵심 패턴

### 3-1. 강화 확률 메커니즘 유형 분류

| 유형 | 대표 게임 | 특징 | 유저 스트레스 |
|------|-----------|------|--------------|
| **파괴형** | 리니지2M, 리니지M | 실패 시 장비 파괴(결정화) | 극상 |
| **하락형** | 검은사막M, 미르4 | 실패 시 강화 수치 하락 | 상 |
| **유지형** | 메이플M(일부), 로스트아크 | 실패해도 수치 유지, 재료만 소모 | 중 |
| **보장형** | 천애명월도M | 실패해도 일정 성장 보장 | 하 |
| **천장형** | 로스트아크(아티산), 미르4(향로) | 실패 축적 -> 확정 성공 | 중하 |

### 3-2. 강화 확률의 일반적 패턴

**안전 구간 (Safe Zone)**
- 대부분의 게임이 초반 +3~+6 구간에서 100% 성공률 제공
- 유저에게 "강화의 맛"을 먼저 보여주는 설계 의도
- 리니지2M: +3까지 안전, 오딘: 안전 강화 4개 단계

**위험 구간 (Risk Zone)**
- 안전 구간 이후 30~70% 확률대로 전환
- 리니지2M: +4부터 약 66.7% 성공(33.3% 실패)
- 리니지M 스킬: 1단계 66.7% -> 2단계 33.3% -> 3단계 16.67%

**극한 구간 (Extreme Zone)**
- +10 이상에서 성공률 10% 이하
- 리니지2M: +16 이상에서 확률 급감
- 검은사막M: 펜(PEN) 이상 구간

### 3-3. 한국 MMO 특유의 "장비 파괴" 문화

한국형 MMO에서 강화 실패 시 장비가 파괴되는 시스템은 1990년대 후반 리니지1에서 시작되어 20년 이상 지속된 전통이다. 이 시스템의 설계 의도는 다음과 같다:

1. **경제적 소비 구조**: 장비 파괴 -> 재획득 필요 -> 게임 내 경제 순환 촉진
2. **도박적 쾌감**: 고위험-고보상 구조에서의 성공 쾌감 극대화
3. **과금 유도**: 축복 주문서(파괴 방지) 등 보조 아이템 판매
4. **서버 경제 제어**: 최상위 장비의 희소성 유지

**최근 변화 추세**: 2023~2024년부터 "장비 파괴 없는 강화"를 선언하는 게임이 증가하고 있다.
- 아이온2: "확률형 아이템 미판매" 선언
- 천애명월도M: 슬롯 강화로 파괴 리스크 제거
- 블소 NEO: 간소화된 성장 + 구독형 과금

## 5. P2W vs F2P 격차 관리 전략

### 5-1. P2W 심각도 스펙트럼

```
극심한 P2W <<<------------------------------------------->>> F2P 친화적
검은사막M > 리니지2M > 리니지M > 오딘 > 리니지W > 타워오브판타지 >
B&S레볼루션 > 그랑사가 > 니노쿠니CW > 미르4 > 세븐나이츠2 >
메이플M > 라그M > 로스트아크 > 천애명월도M > 알비온 온라인
```

### 5-2. P2W 격차 관리 방식별 분류

**A. 무제한 격차 허용 (Unlimited Gap)**
- **대표**: 검은사막 모바일, 리니지M/2M/W, 오딘
- 과금량에 비례하여 전투력이 무한히 증가
- 핵과금 유저가 40:1 무쌍을 찍을 수 있는 극단적 구조
- 한국 시장의 전통적 모델

**B. 천장 있는 격차 (Capped Gap)**
- **대표**: 로스트아크, 타워오브판타지
- 아티산 에너지/돌파 상한선 등 일정 수준의 천장 존재
- 시간이 지나면 무과금도 도달 가능 (단, 매우 오래 걸림)

**C. 시간 기반 자연 균형 (Time-Based Equalization)**
- **대표**: 라그나로크M, 메이플스토리M
- 피로도/스태미나 시스템으로 일일 성장량 제한
- 과금으로 시간을 압축할 수 있지만, 일일 한도 존재

**D. 구조적 평등 (Structural Equality)**
- **대표**: 알비온 온라인
- 풀룻 PvP로 장비가 소모품화 -> 과금 장비의 영구적 가치 없음
- 프리미엄은 "속도 보너스"이지 "능력치 우위"가 아님

**E. 컨텐츠 분산형 (Content-Distributed)**
- **대표**: 천애명월도M
- 성장 요소를 전투 외 다양한 컨텐츠(하우징, NPC 호감도, 사진 등)에 분산
- 과금 -> 스펙의 직선적 구조를 의도적으로 약화

### 5-3. 최근의 "착한 과금" 전환 트렌드 (2023~2025)

한국 게임 업계에서 "착한 과금"이라는 키워드가 부상하고 있다:

1. **확률형 아이템 미판매**: 아이온2가 선언, 후속 타이틀에도 영향
2. **구독(멤버십)/패스 중심**: 블소 NEO의 소울 멤버십, 월정액 보너스
3. **코스메틱(외형) 위주**: 능력치에 영향 없는 외형 아이템 판매
4. **수직 성장 약화, 수평 성장 강화**: 스펙 경쟁보다 컨텐츠 경험 중심

이 전환의 배경:
- 기존 P2W 모델에 대한 유저 피로감 누적
- 규제 환경 변화 (확률형 아이템 규제 논의)
- 글로벌 시장 확대에 따른 서구권 유저 반감 고려
- RPG 서브장르 내 MMO 비중 하락 (방치형, 서브컬처 성장)

## 6. 유저 수용도 분석: 만족 vs 불만 시스템

### 6-1. 가장 많은 유저 불만을 야기한 시스템 TOP 5

**1위: 장비 파괴 (리니지2M)**
- 수백만원 가치의 장비가 한 번의 실패로 결정화
- "수개월간의 노력이 한 순간에 사라진다"는 절망감
- 그러나 역설적으로, 이 도박성이 과금 동기의 핵심

**2위: 무한 성장 + 극단적 PvP 격차 (검은사막 모바일)**
- 40:1 무쌍으로 인한 무과금 유저의 극단적 박탈감
- 강해져도 즐길 컨텐츠 부족 -> "성장의 목적성" 상실
- 연금석 등급 추가로 기존 투자 가치 폐기

**3위: 변신/마법인형 합성 (리니지M)**
- 전설 변신 합성 확률 11%의 가챠적 본질
- 전설 변신의 유무가 전투력의 결정적 차이
- 확률형 아이템의 사회적 논란 촉발

**4위: 무기 돌파 A6 강제 (타워오브판타지)**
- A3 이상이어야 실전 운용 가능한 구조
- 가챠를 반복하지 않으면 컨텐츠 접근 자체가 제한
- "원신 대비 과금 압박이 심하다"는 직접 비교 피해

**5위: 과금 유도형 연금석 시스템 (검은사막 모바일)**
- 새로운 등급(심연) 추가로 기존 연금석 가치 폐기
- "과금하지 않으면 도태"되는 강제적 구조
- 유저들이 "지르지 않고는 못 배기게" 설계된 시스템

### 6-2. 유저 만족도가 높은 시스템 TOP 5

**1위: 아티산 에너지/천장 시스템 (로스트아크)**
- "실패해도 언젠가는 반드시 성공한다"는 심리적 안전망
- 업계 표준으로 자리잡은 천장 메커니즘
- 미르4의 "신비로운 향로"도 유사한 개념

**2위: 슬롯 강화 + 파괴 없음 (천애명월도M)**
- 성장 스트레스 최소화
- 장비 교체 시 재강화 불필요
- "성장의 만족감"에 집중한 설계

**3위: 공명 시스템 (검은사막 모바일)**
- 균형 있는 성장을 유도하는 구조
- "하나만 과금해서 올리면 된다"는 편법 방지
- 전체적인 장비 수준 향상을 목표로 설정

**4위: 숙련도 시스템 (세븐나이츠2)**
- "사용할수록 강해진다"는 직관적 보상
- 과금 없이 시간 투자만으로 성장 가능한 채널
- 영웅석 부가 보상으로 초월 재료 확보

**5위: 풀룻 PvP + 장비 소모품화 (알비온 온라인)**
- 과금 장비에 영구적 가치를 부여하지 않는 근본적 P2W 방지
- 스킬(명성)이 진짜 자산 -> 시간 투자 유저의 가치 인정
- 장비 손실 리스크가 오히려 긴장감과 몰입감을 높임

## 7. 10년간의 변화 추세 및 결론

### 7-1. 2014~2017: 단순 이식기
- PC MMO의 강화 시스템을 모바일에 그대로 이식
- 리니지M이 "변신/마법인형"이라는 모바일 전용 성장 축 추가
- 확률형 강화 + 장비 파괴가 표준 모델

### 7-2. 2018~2019: 확장과 차별화 시도
- 검은사막M: 오픈월드 + 무한 성장이라는 새로운 공식
- 메이플M/라그M: PC 원작 시스템 간소화 이식
- 리니지2M: 장비 파괴의 극대화로 과금 모델 극한까지 밀어붙임
- "전투력"이라는 단일 수치가 모든 성장의 결과를 대표하기 시작

### 7-3. 2020~2021: 레이어 팽창 + 블록체인 실험
- 성장 레이어가 8~12개까지 팽창
- 오딘, 검은사막M, 리니지2M이 다층 강화 구조의 극한을 시도
- 미르4: Play to Earn(블록체인) 모델로 성장의 의미 자체를 "현금화"로 전환
- 그랑사가: 애니 비주얼 + MMORPG 성장의 시도 (운영 실패로 종료 예정)

### 7-4. 2022~2024: 반성과 전환기
- **과잉 레이어의 폐해 인식**: 로스트아크가 초월/엘릭서 삭제 결정
- **착한 과금 전환**: 확률형 -> 구독/패스/코스메틱
- **성장 간소화 트렌드**: 블소 NEO, 천애명월도M
- **P2W에서 P2P(Pay to Progress)로**: 알비온 모델의 재평가
- **MMO 비중 하락**: 방치형 RPG, 서브컬처 게임이 MMO 매출 비중을 잠식

### 7-5. 핵심 교훈 및 결론

**1. 성장 레이어의 최적점이 존재한다**
- 7~8개가 "충분히 할 것이 많으면서도 압도당하지 않는" 적정 수준
- 10개 이상은 유저 피로감 유발, 결국 시스템 삭제로 이어짐

**2. 파괴형 강화는 양날의 검이다**
- 단기적으로 과금 극대화에 효과적
- 장기적으로 무과금 유저 이탈 + P2W 오명으로 시장 축소
- 2024년 이후 신작에서는 파괴형 강화를 채택하지 않는 추세

**3. 천장 시스템은 필수가 되었다**
- 로스트아크의 아티산 에너지가 업계 표준을 제시
- 어떤 형태로든 "실패해도 누적되는 보상"이 없으면 유저 이탈

**4. P2W 격차 관리가 게임 수명을 결정한다**
- 검은사막M의 40:1 무쌍은 게임 생태계를 파괴하는 극단적 사례
- 알비온의 풀룻 모델이나 천애명월도M의 분산형 모델이 대안으로 부상

**5. 한국형 MMO의 성장 시스템은 "확률형 과금 극대화"에서 "다양한 경험 제공"으로 전환 중이다**
- 리니지식 BM의 시대가 저물고 있음
- 패스/구독/코스메틱 중심의 새로운 모델이 실험 중
- 그러나 기존 과금 유저층의 이탈 없이 전환하는 것이 최대 과제
