# 액션 RPG / 로그라이트 모바일 게임 성장 시스템 연구 (2014-2024)

## 연구 목적
2014년부터 2024년까지 주요 액션 RPG 및 로그라이트/로그라이크 RPG 모바일 게임의 성장(progression) 시스템을 분석한다. 메타 프로그레션 설계, 런 기반 vs 영구 성장의 밸런스, 모바일 환경에 최적화된 세션-보상 구조를 중심으로 장르의 진화 과정을 추적한다.

### 1. Honkai Impact 3rd (붕괴3rd) - 2016

| 항목 | 내용 |
|------|------|
| 개발사 | miHoYo (현 HoYoverse) |
| 출시 | 2016년 (중국), 2018년 (글로벌) |
| 매출/다운로드 | 누적 매출 약 6억 달러+, 다운로드 4,500만+ |
| 장르 | 액션 RPG (가챠 기반) |

**핵심 성장 시스템:**
- **발키리 레벨링**: HP, ATK, CRT, DEF, SP 등 기본 스탯 상승. 레벨 캡이 존재하며 소재로 돌파
- **발키리 진화**: 프래그먼트 수집으로 랭크 승급 (B->A->S->SS->SSS). 각 랭크마다 스킬 해금
- **성흔(Stigmata) 시스템**: Top/Mid/Bot 3슬롯 장비. 세트 효과 존재. 강화/승급/정련 3단계 업그레이드
- **무기 시스템**: 가챠 획득, 강화/진화 가능
- **기지(Base) 업그레이드**: 사령부, 무기별 캐빈 등 시설 업그레이드로 스탯 보너스

**성장 레이어 깊이:** 5~6레이어 (캐릭터 레벨 + 랭크 진화 + 성흔 장착/강화 + 무기 강화 + 기지 업그레이드 + 스킬 해금)

**유저 평가:** 깊이 있는 액션과 풍부한 캐릭터 커스터마이징을 높이 평가. 다만 후반부 가챠 의존도와 성흔 세트 파밍의 부담이 지적됨. miHoYo의 라이브 서비스 모델의 원형을 확립한 작품.

### 2. Soul Knight (소울나이트) - 2017

| 항목 | 내용 |
|------|------|
| 개발사 | ChillyRoom |
| 출시 | 2017년 |
| 매출/다운로드 | 누적 다운로드 수억 회 이상 (정확한 매출 비공개) |
| 장르 | 모바일 로그라이트 던전 슈터 |

**핵심 성장 시스템:**
- **영구 캐릭터 해금**: 다양한 클래스(기사, 도적, 마법사, 엔지니어 등) 젬으로 구매
- **정원(Garden) 시스템**: 런 사이에 식물 재배, 휴식 콘텐츠 겸 소재 획득
- **무기 제작/해금**: 설계도 수집으로 새 무기 영구 해금
- **런 내 성장**: 각 방 클리어 시 랜덤 스킬/아이템 획득, 무기 교체
- **멀티플레이**: 협동 플레이로 난이도 조절

**런 vs 영구 성장 밸런스:** 영구 성장은 캐릭터 해금과 무기풀 확장에 집중. 런 내 성장은 랜덤 무기와 스킬 조합이 핵심. 영구 스탯 인플레이션은 최소화하여 매 런의 신선함을 유지하는 방향.

**모바일 최적화:** 한 손 조작 가능한 간결한 UI. 런 당 10~15분 세션. 로그라이트의 핵심을 모바일에 성공적으로 이식한 초기 사례.

**유저 평가:** 모바일 로그라이트의 레퍼런스로 높은 평가. 과금 없이 즐길 수 있는 공정한 모델, 다양한 무기와 캐릭터 조합의 리플레이성이 강점.

### 3. Brawl Stars (브롤스타즈) - 2018

| 항목 | 내용 |
|------|------|
| 개발사 | Supercell |
| 출시 | 2018년 12월 (글로벌) |
| 매출/다운로드 | 연간 수억 달러 매출, 다운로드 5억+ |
| 장르 | 멀티플레이어 액션 (RPG 성장 요소 포함) |

**핵심 성장 시스템:**
- **트로피 로드(Trophy Road)**: 트로피 누적으로 브롤러, 파워 포인트, 코인 해금. 전체 진행도의 척도
- **파워 레벨**: 파워 포인트+코인으로 파워 1~11까지 업그레이드. 레벨 7에서 가젯, 9에서 스타파워, 11에서 하이퍼차지 해금
- **브롤 패스**: 시즌별 보상 트랙. 무료/유료 트랙 분리
- **마스터리 시스템**: 특정 브롤러 반복 사용 시 코인, 파워 포인트, 크레딧 보상

**성장 레이어:** 4레이어 (트로피 진행 + 파워 레벨 + 가젯/스타파워 해금 + 시즌 패스)

**RPG 성장과 PvP의 균형:** 파워 레벨이 전투력에 직접 영향을 미치지만, 스킬 비중이 높아 저레벨도 고레벨을 이길 수 있는 구조. 레벨 차이가 있되 결정적이지 않은 밸런스가 핵심.

**유저 평가:** 간결한 3분 매치와 다양한 모드가 장기 리텐션의 핵심. 성장 시스템은 꾸준한 업데이트로 레이어가 추가되어 왔으며, F2P 친화적이라는 평가와 후반부 성장 병목에 대한 비판이 공존.

### 4. Punishing: Gray Raven (펑크레이) - 2019

| 항목 | 내용 |
|------|------|
| 개발사 | Kuro Games |
| 출시 | 2019년 12월 (중국), 2021년 (글로벌) |
| 매출/다운로드 | 누적 매출 1.6억 달러+, 다운로드 600만+ |
| 장르 | 모바일 액션 RPG |

**핵심 성장 시스템:**
- **컨스트럭트(Construct) 레벨링**: EXP 포드로 레벨업, 스탯 상승
- **프로모션(승급)**: 조건 충족 시 자원 투자로 영구 스탯 증가, 랭크별 흰 별 추가
- **스킬 트레이닝**: 25단계 스킬 레벨업 (스탯, 스킬 데미지, 쿨다운, 궁극기 데미지)
- **장비 강화**: 무기와 메모리(성흔 유사) 장착 및 강화
- **서포터 시스템**: 서포터 레벨링/진화, 레어도별 효과 차등

**성장 레이어:** 5레이어 (캐릭터 레벨 + 승급 + 스킬 트레이닝 + 장비 강화 + 서포터)

**유저 평가:** 전투 액션의 품질이 높고, F2P 친화적 가챠 시스템으로 호평. 성장 시스템은 Honkai Impact 3rd와 유사하나 더 직관적이라는 평가. 다만 콘텐츠 소진 속도가 빠르다는 지적 존재.

### 6. 궁수의 전설 (Archero) - 2019

| 항목 | 내용 |
|------|------|
| 개발사 | Habby |
| 출시 | 2019년 5월 |
| 매출/다운로드 | 첫 달 $8.5M/1000만 DL, 누적 $2.6억+/9600만+ DL |
| 장르 | 모바일 로그라이트 액션 |

**핵심 성장 시스템:**

**(1) 런 내 성장 (임시):**
- 레벨업 시 3개 중 1개 어빌리티 선택 (바운스 화살, 멀티샷, 리코셰 등)
- 악마 거래 시스템 (HP 교환으로 강력한 능력 획득)
- 장비 효과의 런 내 시너지

**(2) 메타 프로그레션 (영구):**
- **탤런트 시스템**: 골드로 업그레이드. 첫 탤런트 'Glory'가 런 시작 시 어빌리티 1개 부여. 장비 보너스 강화, 레벨업 시 힐 등 영구 패시브
- **장비 시스템**: 가챠/드롭으로 획득. 무기/방어구/반지/스피릿 6슬롯. 등급별 최대 레벨 차등. 합성으로 등급 상승
- **히어로 성장**: 재화로 히어로 업그레이드, 스탯과 퍼크 제공 (2019년 10월 추가)

**런 vs 영구 성장 밸런스:**
새 챕터 진입 시 통상 전투력 부족 -> 탤런트/장비 강화 + 패턴 학습 병행 -> 스탯과 스킬 동시 성장. 이 구조가 "다음 한 판만 더"의 동기를 부여. 다만 후반 챕터에서 RNG 의존도가 높아지면서 그라인드 필요성이 증가하는 것이 약점.

**성장 레이어:** 4레이어 (런 내 어빌리티 + 탤런트 + 장비 + 히어로 성장)

**유저 평가:** 모바일 로그라이트의 상업적 성공 공식을 확립. 짧은 세션(5~10분)과 명확한 진행감이 강점. "하이브리드 캐주얼" 장르를 개척. 다만 후반부 과금 유도와 광고 의존 모네타이제이션에 대한 비판 존재.

**역사적 의의:** Archero는 모바일 로그라이트의 상업화 공식을 정립한 게임. 매출의 60%가 인앱결제, 40%가 광고에서 발생하는 하이브리드 모네타이제이션 모델을 증명. Habby를 2019년 매출 $1M에서 $181M으로 성장시킨 원동력.

### 7. Dead Cells (모바일) - 2019

| 항목 | 내용 |
|------|------|
| 개발사 | Motion Twin |
| 출시 | 2019년 (모바일 포트) |
| 매출/다운로드 | 프리미엄 가격, PC/콘솔 합산 1000만+ 판매 |
| 장르 | 로그라이트 메트로바니아 |

**핵심 성장 시스템:**

**(1) 런 내 성장:**
- 셀(Cells) 수집: 적 처치 시 획득, 던전 구간 종료 시에만 사용 가능 (사망 시 소실)
- 무기/스킬 랜덤 드롭 및 선택
- 스크롤로 스탯 분배 (Brutality/Tactics/Survival)

**(2) 메타 프로그레션:**
- **셀 투자**: 영구 업그레이드 구매 (HP 포션, 새 무기 해금 등)
- **블루프린트**: 던전 내 발견 -> 구간 클리어 시 수집 -> 새 무기/스킬 영구 해금 풀에 추가
- **룬(Runes)**: 엘리트 적 처치로 획득. 새로운 이동 능력 영구 해금 (메트로바니아 요소)

**런 vs 영구 성장 밸런스:**
Dead Cells의 메타 프로그레션은 "파워 인플레이션"이 아닌 "옵션 확장"에 초점. 새 무기를 해금하면 드롭 풀에 추가되어 다양한 빌드가 가능해지지만, 직접적인 스탯 인상은 제한적. 룬은 새 경로를 열어 탐색 범위를 넓힘. 이 설계가 "숙련도 기반 성장"을 유지하면서도 매 런의 다양성을 보장.

**성장 레이어:** 3레이어 (런 내 셀/스크롤/장비 + 블루프린트 해금 + 룬 영구 해금)

**모바일 이식 평가:** 프리미엄 가격 모델. 터치 조작에 대한 평가는 엇갈리나, 게임 본연의 품질은 높이 평가. 모바일에서 "PC급 로그라이트 경험"을 제공한 선구적 사례.

### 8. Pascal's Wager - 2020

| 항목 | 내용 |
|------|------|
| 개발사 | TipsWorks |
| 출시 | 2020년 1월 |
| 매출/다운로드 | 프리미엄 가격 ($6.99), 모바일 소울라이크 시장 개척 |
| 장르 | 모바일 소울라이크 액션 RPG |

**핵심 성장 시스템:**
- **본즈(Bones) 시스템**: 경험치 겸 화폐. 사망 시 일부만 소실 (다크소울 대비 관대)
- **속성 포인트 분배**: 레벨업 시 직접 스탯 배분
- **트로피 시스템**: 패시브 보너스 제공, 진행에 따라 트로피 강화 가능

**성장 레이어:** 4레이어 (레벨/스탯 분배 + 탤런트 트리 + 트로피 장비 + 캐릭터별 무기/어빌리티)

**유저 평가:** 모바일에서 소울라이크를 구현한 도전적 시도로 높이 평가. 사망 패널티 완화(본즈 일부만 소실)가 모바일 환경에 적합하다는 평가. 다만 조작감과 카메라에 대한 아쉬움이 있음.

## Part 2: 핵심 분석 - 성장 시스템의 진화와 설계 원칙

---

### A. 메타 프로그레션 설계의 진화 (2016-2024)

#### 1세대: 단순 스탯 인플레이션 (2016~2018)
- **대표작**: Honkai Impact 3rd, Soul Knight
- **특징**: 캐릭터 레벨 + 장비 강화 + 스킬 레벨의 직선적 성장
- **메타 프로그레션**: "더 강해지기" 위한 수직적 파워 증가
- **문제점**: 레벨 캡 도달 후 콘텐츠 소진, 가챠 의존도 증가

#### 2세대: 옵션 확장형 메타 프로그레션 (2019~2020)
- **대표작**: Archero, Dead Cells, Hades
- **혁신**: 영구 성장이 "파워 증가"가 아닌 "선택지 확대"로 전환
- Dead Cells: 새 무기 해금 = 드롭 풀 확장 (파워가 아닌 다양성)
- Hades: Mirror of Night = 플레이스타일 커스터마이징 (리스펙 자유)
- Archero: 탤런트/장비가 기반 파워를 올리되, 런 내 어빌리티 선택이 핵심

#### 3세대: 서바이벌라이크의 이중 루프 (2021~2023)
- **대표작**: Magic Survival, Vampire Survivors, Survivor.io, 20 Minutes Till Dawn
- **혁신**: "런 내 기하급수적 파워 커브" + "런 간 선형적 메타 성장"
- 런 내에서 무력한 캐릭터가 30분 만에 화면을 뒤덮는 파워 판타지
- 메타 프로그레션은 시작점을 미세하게 끌어올려 "다음 런의 기대감" 조성
- 핵심 원칙: 메타 스탯 < 런 내 빌드 선택

#### 4세대: 복합 레이어 + 라이브 서비스 (2022~2024)
- **대표작**: Diablo Immortal, Torchlight Infinite, Survivor.io (후기)
- **특징**: 5~6개 이상의 성장 레이어 적층
- 시즌 리셋 + 영구 성장의 공존
- 배틀 패스, 일일 보상, 길드 콘텐츠 등 라이브 서비스 요소와 결합
- 월드 파라곤 등 "사회적 진행도 조절" 메카닉 등장

### B. 런 기반 vs 영구 성장 밸런스의 스펙트럼

```
[퓨어 로그라이크] ←————————————————→ [퓨어 RPG]
  Brotato    Dead Cells   Hades   Vampire S.   Archero   Survivor.io   Diablo I.   Honkai 3rd
  (메타 무)   (옵션 확장)  (커스텀)  (미세 스탯)  (장비+탤런트) (장비 중심)   (파라곤+젬)  (가챠+강화)
```

**최적 밸런스의 핵심 원칙들:**

1. **"제로 진행 런 제거" 원칙 (Vampire Survivors)**
   - 실패한 런에서도 반드시 무언가를 가져가야 한다
   - 골드, 해금 진행, 업적 등 최소 하나의 영구 보상

2. **"메타 < 런 내 선택" 원칙 (Soulstone Survivors, Hades)**
   - 메타 프로그레션으로 얻는 파워가 런 내 빌드 선택보다 작아야 한다
   - 이것이 깨지면 초반만 어렵고 후반은 무의미해지는 단조감소 곡선 발생

3. **"옵션 확장 > 스탯 인플레이션" 원칙 (Dead Cells)**
   - 새로운 무기/스킬/경로를 열어주는 메타가 직접적 스탯 증가보다 건강
   - 리플레이성과 다양성을 동시에 보장

4. **"시작점 미세 조정" 원칙 (Vampire Survivors, 20 Minutes Till Dawn)**
   - 메타 PowerUp의 개별 증가분은 미미하지만 누적은 유의미
   - 그라인드 감이 적으면서도 장기 성장의 만족감 제공

### C. 모바일 세션 길이 vs 진행 보상 밸런스

### E. 매일 복귀시키는 요인 - 로그라이트 RPG의 일일 리텐션 설계

#### 1. 진행감의 보장 ("시간 낭비 없음")
- 실패한 런에서도 재화/경험/해금 진행이 발생
- 메타 프로그레션이 "다음 런은 조금 더 쉬울 것"이라는 기대감 조성
- Archero: 골드 -> 탤런트, Vampire Survivors: 골드 -> PowerUp

#### 2. 새로움의 약속 ("다음 런은 다를 것")
- 랜덤 빌드 조합이 매 런 새로운 경험 보장
- 언락 시스템: "다음에는 새 캐릭터/무기를 써볼 수 있다"
- Hades: 매 런 다른 신의 분 조합
- Soulstone Survivors: 글로벌 트리에서 새 메카닉 해금

#### 3. 짧은 커밋먼트 ("한 판만 더")
- 10~15분 세션이 재시도 장벽을 낮춤
- 모바일 환경에서 "자투리 시간 활용" 포지셔닝
- Survivor.io: 챕터 하나만 -> 자연스럽게 다음 챕터로

#### 4. 라이브 서비스 요소 (후기 작품)
- 일일 보상 / 출석 보상
- 시즌 / 배틀 패스 진행도
- 이벤트 던전 / 한정 콘텐츠
- Survivor.io, Diablo Immortal: 일일 퀘스트와 한정 보상

#### 5. 사회적 요소
- 길드 / 클랜 콘텐츠
- 랭킹 / 리더보드
- Brawl Stars: 팀 매치, Diablo Immortal: 레이드

### F. 성장 시스템의 복잡도와 상업적 성공의 상관관계

| 복잡도 | 게임 | 매출 규모 | 타겟 |
|--------|------|-----------|------|
| 낮음 (1~2 레이어) | Vampire Survivors, Brotato | 중간 (프리미엄/인디) | 캐주얼~미드코어 |
| 중간 (3~4 레이어) | Archero, Survivor.io, Soul Knight | 높음 ($1억~5억+) | 하이브리드 캐주얼 |
| 높음 (5~6+ 레이어) | Diablo Immortal, Honkai Impact 3rd | 최고 ($5억+) | 미드코어~하드코어 |

**핵심 인사이트:**
- 성장 레이어가 많을수록 과금 포인트가 증가하여 매출 잠재력이 높아짐
- 그러나 복잡도가 높을수록 진입 장벽이 상승하여 DAU 확보가 어려워짐
- **최적점**: 3~4 레이어로 직관적이면서도 깊이가 있는 시스템 (Archero, Survivor.io의 성공 공식)

## Part 3: 종합 결론

### 모바일 액션 RPG / 로그라이트의 성장 시스템 진화 요약

1. **2016~2018 (태동기)**: 가챠 기반 액션 RPG가 주류. 성장 = 숫자 올리기. Soul Knight가 모바일 로그라이트의 가능성을 증명.

2. **2019 (분기점)**: Archero가 "하이브리드 캐주얼 로그라이트"를 창시. 짧은 세션 + 메타 프로그레션 + F2P 모네타이제이션의 삼위일체 확립. Hades가 "스토리 드리븐 메타 프로그레션"의 교과서를 제시.

3. **2021~2022 (서바이벌라이크 혁명)**: Magic Survival(한국)에서 시작된 서바이벌라이크 장르가 Vampire Survivors로 글로벌 히트. "이동만으로 플레이 + 런 내 기하급수 성장 + 미세한 메타 프로그레션"의 새 공식. Survivor.io가 이를 F2P 모바일에 최적화하여 $5억+ 달성.

4. **2022~2024 (복합화)**: Diablo Immortal이 PC ARPG의 깊이를 모바일에 이식하며 $5억+ 달성 (논란 수반). 서바이벌라이크 후발 주자들(Soulstone Survivors, Halls of Torment, 20 Minutes Till Dawn)이 메타 프로그레션의 다양한 변주를 실험.

### 차세대 설계를 위한 핵심 교훈

1. **"실패해도 보상받는" 메타 프로그레션은 필수** - 모바일 유저의 이탈 방지 핵심
2. **메타 프로그레션은 "옵션 확장"에 우선순위** - 스탯 인플레이션은 최소화
3. **런 내 빌드 선택의 임팩트 > 메타 파워** - 숙련도와 전략의 가치 보존
4. **10~15분 세션에서 완전한 감정 아크 완성** - 모바일 골든 타임
5. **3~4 성장 레이어가 최적** - 직관성과 깊이의 균형점
6. **라이브 서비스 요소와 로그라이트 코어의 조화** - 일일 리텐션과 런 자유도의 양립
