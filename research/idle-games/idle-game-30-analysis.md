### 방치 유형별 성공률
- 순수 방치 = $5M 천장 (법칙 30)
- 방치+가챠 = $100~500M
- 방치+가챠+PvP+소셜 = $1B+ 가능
- 방치+로그라이트 = $100M+ (Capybara Go 검증)
- 방치+P2E = **전멸** (2022년 이후)

### 실패 패턴
1. P2W + 클론 = 15개월 이내 종료 (Valkyrie Rush)
2. P2E = 토큰 하락 시 게임 즉사 (MIR4, Idle Mystic)
3. 차별화 없는 방치RPG = 시장 포화로 매몰
