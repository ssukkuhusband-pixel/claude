# 방치형 게임 설계 원칙 조사

> 조사일: 2026-03-30
> 출처: GDC(Pecorella 2016), Deconstructor of Fun, Sensor Tower, Naavik, Gamigion

## 핵심 원칙 요약

### 코어 루프
- 오프라인 보상 상한: 6~12시간이 최적 (하루 2~3회 복귀 유도)
- 복귀 트리거 5가지: 보상 상한 알림, 시간 제한 이벤트, 진행 차단, 소셜, 데일리
- 프레스티지: 잃는 것 최소화, 획득은 쌓이기, 첫 1~3일 내 경험

### 수학 모델 (Pecorella GDC 2016)
- 비용: 지수 함수 (n^x)
- 생산: 선형/다항 (x^k)
- 이 갭이 벽을 자연 생성

### BM 수익 결정 공식 (법칙 30)
```
수익 = 가챠 깊이 × PvP 긴장감 × 소셜 레이어
```
- 순수 방치 = $1~5M
- 방치+가챠+PvP+소셜 = $1B+

### 성장 쾌감 유지 4도구
1. 순차적 레이어 해금
2. 빠름-느림-빠름 리듬
3. 빌드 다양성
4. 질적 변화 (외형/능력이 바뀌어야)

### 소셜/PvP
- 비동기 방어 AI, 스냅샷 랭킹, 비동기 습격, 시즌 리더보드
- 길드 보스 참여 유저 D30 리텐션 2.3배 (Tap Titans 2)

### 실패 패턴
1. "너무 방치적" = 유저가 게임 존재 망각
2. 성장 인플레이션 = 숫자가 무의미
3. 시장 포화 = 차별화 없으면 매몰
4. P2W 과도 = 무과금 이탈 → 생태계 붕괴

### 하이브리드 트렌드
- 방치+수집RPG: AFK Arena $3B
- 방치+로그라이트: Capybara Go $109M
- 방치+건설: Cookie Run Kingdom $500M+
- 방치+MMORPG: MIR4 (P2E 붕괴로 실패)
