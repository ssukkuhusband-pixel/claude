## 1. Diablo III

| 항목 | 내용 |
|------|------|
| **스킬 총 개수** | 클래스당 약 25개 활성 스킬 × 7 클래스 = 약 175개. 각 스킬당 5개 룬 변형 → 실질 변형체 약 875개 |
| **캐릭터당 활성 스킬 수** | **6개 슬롯** (Primary, Secondary + 클래스별 4개) |
| **스킬 복잡도** | 중간. 룬 적용 시 단순 데미지 변환(원소 교체)부터 완전한 메커니즘 변경(단일 타겟 → AoE)까지 다양 |
| **스킬 트리/분기 구조** | **자유형 선택제** — 트리 없음. 레벨업으로 스킬 잠금 해제 후 자유 배치. 레벨 30에 전 스킬 개방, 레벨 60에 모든 룬 효과 해금 |
| **등급/희귀도 시스템** | 스킬 자체에 등급 없음. 룬(5종)이 변형 역할 수행. 패시브 스킬 따로 존재 (4개 슬롯) |
| **스킬 간 차별화 수준** | **높음** — 룬 하나로 동일 스킬이 완전히 다른 플레이 스타일 유도. 예: 위자드 텔레포트 룬에 따라 순간이동형/공격형/생존형으로 분화 |
| **핵심 성공 요인** | 진입장벽 제거 (스킬 트리 폐지) + 즉각적 실험 가능성. 단, 후기 메타에서 실질 빌드 다양성 부족이 비판받음. "이론상 조합 수 >> 실제 생존 가능 빌드 수" 문제 존재 |

**레퍼런스 포인트**: 6슬롯 제한 + 룬 변형 시스템은 모바일 "스킬 슬롯 제한 + 스킬 강화 분기" 설계의 원형임다.

## 2. Diablo IV

| 항목 | 내용 |
|------|------|
| **스킬 총 개수** | 클래스당 약 30~40개 활성/패시브 노드. 7개 클러스터 구조. 총 최대 71 스킬 포인트 투자 가능 |
| **캐릭터당 활성 스킬 수** | **6개 슬롯** (Primary 1 + Secondary 1 + 나머지 4) |
| **스킬 복잡도** | 3단계 — 기본 스킬(단순) → Enhanced 업그레이드 → 2갈래 분기 업그레이드 |
| **스킬 트리/분기 구조** | **분기형** — 각 활성 스킬에 기본 강화(Enhanced) 1개 + 고급 분기 2개 중 택1. 7개 클러스터가 순차 잠금 해제 |
| **등급/희귀도 시스템** | 스킬 자체 등급 없음. Key Passive (최강 패시브 1개 선택제) 별도 존재. 시즌별 Paragon Board 보드 시스템 추가 |
| **스킬 간 차별화 수준** | **높음** — 분기 선택에 따라 동일 스킬이 공격형/방어형/군중제어형으로 완전 분화. Necromancer의 Bone Spear는 분기에 따라 단일 관통형 또는 연쇄 파편형으로 플레이 스타일 자체가 달라짐 |
| **핵심 성공 요인** | 명확한 업그레이드 경로 시각화 + 분기 선택의 정체성 부여. 단, 출시 초기 빌드 다양성 부족 비판 있었으나 이후 패치로 개선됨 |

**레퍼런스 포인트**: 스킬당 2갈래 분기 업그레이드는 모바일 "스킬 강화 방향 선택" 시스템으로 직접 활용 가능함다.

## 3. Path of Exile (1)

| 항목 | 내용 |
|------|------|
| **스킬 총 개수** | **활성 스킬 젬 276개** + 서포트 스킬 젬 178개 = 총 454개 젬. 패시브 트리 1,325개 노드 |
| **캐릭터당 활성 스킬 수** | 장비 링크 슬롯에 의존. 일반적으로 메인 스킬 1~2개 + 서브 스킬 4~6개. 실질 활용 스킬 **6~8개** |
| **스킬 복잡도** | **최고 수준** — 활성 젬 1개 + 서포트 젬 5개 링크 = 단 하나의 스킬이 6개 모디파이어로 변형. 예: Fireball에 Added Cold, Spell Echo, Controlled Destruction, Concentrated Effect, Fire Penetration 조합 시 완전히 새로운 스킬 행동 |
| **스킬 트리/분기 구조** | **자유형 웹 구조** — 1,325 노드 패시브 트리는 3개 시작점(클래스)에서 출발하지만 어디든 이동 가능. Ascendancy 시스템으로 클래스별 4개 특화 트리 추가 |
| **등급/희귀도 시스템** | 젬 자체는 레벨 1~21, 품질 0~23% 수치 강화. Vaal Gems는 타락된 강화 버전. Awakened Support Gems는 DLC 한정 강화판 |
| **스킬 간 차별화 수준** | **업계 최고** — 동일 활성 젬도 서포트 조합에 따라 완전히 다른 플레이 스타일. 분당 100회 히트 빠른 타격 빌드 vs 10초마다 1회 초대형 피해 빌드가 모두 Cyclone 하나로 구현 가능 |
| **핵심 성공 요인** | "젬이 아이템"이라는 발상 — 거래 가능, 레벨업 가능, 조합 실험 가능. 빌드 자유도 = 플레이 수명 극대화. 단 진입장벽이 업계 최고 수준 |

**레퍼런스 포인트**: 서포트 젬 = 스킬 강화 조합 시스템의 원형. 모바일에서 "스킬 강화 재료 조합"으로 변형 가능함다.

## 4. Path of Exile 2

| 항목 | 내용 |
|------|------|
| **스킬 총 개수** | Early Access 기준 활성 젬 약 200+종 (지속 확장 중). 패시브 트리 별도 |
| **캐릭터당 활성 스킬 수** | **9개 활성 스킬 젬 슬롯** (기본). Ascendancy에 따라 추가 가능 |
| **스킬 복잡도** | PoE1 대비 단순화. 젬을 장비가 아닌 스킬 전용 창에 배치. 서포트 젬은 스킬당 최대 5개 (Jeweller's Orb로 확장) |
| **스킬 트리/분기 구조** | PoE1 패시브 트리 계승 + Dual Weapon Specialization 추가. 무기별로 다른 스킬 세트 운영 가능 |
| **등급/희귀도 시스템** | 서포트 젬 캐릭터당 고유 1개 제한 (같은 서포트 젬 2개 장착 불가) → 빌드 선택의 트레이드오프 강화 |
| **스킬 간 차별화 수준** | **높음** — PoE1 복잡도 유지하면서 UX 개선. 지원 젬 제한으로 오히려 빌드 정체성 더 명확 |
| **핵심 성공 요인** | PoE1의 복잡성은 유지하되 "장비 소켓 색깔 맞추기" 같은 비직관적 요소 제거. 신규 유저 진입장벽 대폭 낮춤 |

**레퍼런스 포인트**: "서포트 젬 캐릭터당 1개 제한" = 스킬 강화 재료에 희귀성/선택 비용 부여하는 모바일 BM 연계 설계 아이디어임다.

## 5. Dark Souls 시리즈 (DS1~3 통합)

| 항목 | 내용 |
|------|------|
| **스킬 총 개수** | DS3 기준: 소서리 38개 + 미라클 38개 + 파이로맨시 30개 = **마법 106개**. 무기 아츠(Weapon Arts) 약 85~98개 (DLC 포함) |
| **캐릭터당 활성 스킬 수** | 무기 아츠 **1개** (장착 무기에 종속) + 마법 슬롯 **최대 10개** (Attunement 스탯 의존: 10→1슬롯, 50→10슬롯) |
| **스킬 복잡도** | 낮음~중간. 무기 아츠는 대부분 단일 행동. 마법은 소서리(데미지)/미라클(힐·버프)/파이로맨시(불 계열) 역할 명확 분리 |
| **스킬 트리/분기 구조** | **트리 없음** — 스탯 투자(Attunement, Intelligence, Faith, Dexterity 등)로 사용 가능 범위 결정. 무기 아츠는 무기에 고정 |
| **등급/희귀도 시스템** | 마법/아츠 등급 없음. 무기 강화 단계(+1~+10)만 존재 |
| **스킬 간 차별화 수준** | **매우 높음 (무기 단위)** — 무기마다 다른 아츠로 완전히 다른 전투 감각 제공. 단, 동일 클래스 내 스킬 선택지는 제한적 |
| **핵심 성공 요인** | 무기 자체가 스킬의 정체성 = 무기 수집 동기 극대화. "이 무기 쓰면 이런 전투" 명확한 판타지 부여. 높은 학습 비용 자체가 마스터리 보상감 제공 |

**레퍼런스 포인트**: "무기 = 스킬 정체성" 구조는 모바일에서 "무기/장비가 스킬 세트를 결정"하는 BM 연계 설계의 레퍼런스임다.

## 6. Elden Ring

| **스킬 복잡도** | 중간~높음. Ashes of War는 단순 공격부터 버프/스탠스 변경/召唤까지 다양. 일부는 다단계 조건 보유 |
| **스킬 트리/분기 구조** | 트리 없음. Ashes of War는 자유롭게 무기에 탈부착 가능 (DS3 대비 혁신). 스탯 투자로 마법 사용 여부 결정 |
| **등급/희귀도 시스템** | Ashes of War 등급 없음. Sacred/Magical/Lightning 등 무기 친화도(Affinity) 옵션으로 스킬 속성 변경 가능 |
| **스킬 간 차별화 수준** | **매우 높음** — 같은 칼에 다른 Ash of War 장착 시 완전히 다른 전투 스타일. Moonveil(마법 빔 발사)과 Giant Hunt(거대 적 후리기)는 사용 무기가 같아도 체감 전혀 다름 |
| **핵심 성공 요인** | DS3 고정 무기 아츠에서 "자유 탈부착"으로 진화. 실험 비용 낮아져 빌드 다양성 폭발적 증가. 오픈월드 탐험 동기(새 Ash of War 발견)와 스킬 시스템 시너지 |

**레퍼런스 포인트**: "스킬을 아이템처럼 수집·장착·교체" = 모바일 스킬 카드/스킬 런 시스템 설계 아이디어임다.

## 7. Monster Hunter: World (+ Iceborne)

| 항목 | 내용 |
|------|------|
| **스킬 총 개수** | MHW 기본 약 130종 스킬 + Iceborne 추가 포함 **약 160~170종**. 모두 패시브 스킬 |
| **캐릭터당 활성 스킬 수** | 엄밀한 의미의 "활성 스킬 슬롯" 없음. 장비 세팅으로 패시브 스킬 자동 발동. 특수행동 아이템(섬광탄, 덫 등) 별도 |
| **스킬 복잡도** | 낮음~중간. 대부분 "+X% 공격력", "회피 성능 +1" 등 수치형. 일부 Set Bonus는 특수 메커니즘 발동 (2세트/4세트 보너스) |
| **스킬 트리/분기 구조** | 트리 없음. **세트 빌딩** 구조 — 방어구 5부위 + 장식주(Decorations)로 스킬 레벨 합산. 동일 스킬 여러 장비에서 스택 가능 |
| **등급/희귀도 시스템** | 스킬 등급 없음. 장식주는 Level 1~4 크기로 슬롯 적합성 결정. 일부 희귀 장식주(예: 달인의 장식주 Lv4)는 드롭률 극저 |
| **스킬 간 차별화 수준** | **무기 간 높음, 스킬 간 중간** — 14종 무기 타입이 완전히 다른 플레이 스타일 제공. 스킬 자체는 강화 수치이지만 세트 조합으로 빌드 방향성 결정 |
| **핵심 성공 요인** | 스킬 = 아이템 획득 보상의 자연스러운 결과물. 몬스터 사냥 → 소재 획득 → 장비 제작 → 스킬 활성화 루프가 완벽하게 설계됨. 스킬이 "플레이 스타일 정의"가 아닌 "플레이 강화"에 집중 |

**레퍼런스 포인트**: 스킬 = 장비 보상, 세트 시너지 = 수집 동기. 모바일 세트 효과/장착 시너지 시스템의 완성형 레퍼런스임다.

## 8. Devil May Cry 5

| 항목 | 내용 |
|------|------|
| **스킬 총 개수** | Nero 약 40개 + V 약 35개 + Dante 약 80개(12개 무기 × 스타일 조합) = **총 155개 이상**. 전 캐릭터 해금 비용 약 11,204,300 Red Orbs |
| **캐릭터당 활성 스킬 수** | Nero: 버스터 암 포함 **즉시 사용 가능 수십 개** 콤보. Dante: 4개 스타일 × 12개 무기 조합으로 **60+ 무브** 동시 보유 가능 |
| **스킬 복잡도** | 중간. 스킬 설명은 입력 커맨드 중심. 단, Dante의 스타일 시스템(Swordmaster/Gunslinger/Trickster/Royal Guard)이 동일 무기를 4가지 방식으로 사용 |
| **스킬 트리/분기 구조** | **선형 구매형** — Red Orbs로 스킬 순차 구매. 분기 없음. 모든 스킬 최종 구매 가능 (시간의 문제) |
| **등급/희귀도 시스템** | 등급 없음. Dante의 스타일 레벨(Lv1~3)로 추가 무브 해금 |
| **스킬 간 차별화 수준** | **최고 수준** — Dante의 4 스타일은 플레이 패러다임 자체를 전환. Royal Guard(적 공격 흡수·반격)는 다른 게임에서 유사 시스템 찾기 어려운 유니크 메커니즘 |
| **핵심 성공 요인** | Style 시스템 = 동일 입력이 장착 스타일에 따라 완전히 다른 효과. 숙련도가 쌓일수록 조합 가능성 기하급수적 증가. Style 점수 시스템(D~SSS)이 스킬 활용 동기 강화 |

**레퍼런스 포인트**: 동일 버튼 입력이 상황에 따라 다른 스킬 발동 = 모바일의 "스킬 체인/콤보 연계" 시스템 레퍼런스임다.

| **스킬 간 차별화 수준** | **직업 간 매우 높음** — Magick Archer(마법 화살), Mystic Spearhand(마법 창술), Warfarer(혼합) 등 전투 감각 완전히 다름. 동일 직업 내 스킬 선택도 4슬롯 제한으로 선택의 고통 존재 |
| **핵심 성공 요인** | Warfarer의 크로스 직업 스킬 이식 = 빌드 자유도 극대화. Pawn(AI 동료) 직업도 독립적으로 육성 = 사실상 2캐릭터 동시 운영 |

**레퍼런스 포인트**: 직업 레벨링으로 스킬 해금 = 모바일의 "캐릭터/클래스 육성형 스킬 해금" 시스템임다.

## 모바일 RPG 기획 적용 핵심 인사이트

### 1. 활성 스킬 슬롯 "황금 구간"
- 대부분의 성공작: **4~8개** 슬롯 수렴
- 6개 슬롯 (D3, D4, PoE): 충분한 조합 + 과부하 없음
- 모바일 권장: **4~6개** (터치 UX 고려 시 4개 적합)

### 2. 스킬 차별화의 핵심 = "플레이 스타일 변환"
- 성공한 시스템들은 스킬 선택이 수치 강화가 아닌 **플레이 방식 자체**를 바꿈
- D3 룬: 단일→AoE 전환 / PoE 서포트 젬: 빠른 타격→느린 초대형 전환
- Hades 신 라인: 번개 체인 vs 독 누적 vs 넉백 중심

### 3. 등급 시스템의 역할
- Hades Boon 4등급 / Dead Cells 2등급 / RoR2 아이템 6등급
- 등급이 **수치 강화**만이면 BM용. 등급이 **행동 변형**까지 하면 게임 심도 향상
- 모바일 권장: 등급이 스킬 변형 효과를 가져야 의미 있슴다

### 4. 빌드 다양성 보장 구조
- 최고 평가 게임들의 공통점: **제한**이 오히려 다양성 만듦
  - Last Epoch 5개 특화 제한 → 선택의 의미 강화
  - Hades 슬롯당 Boon 1개 제한 → 시너지 전략 필요
  - PoE 서포트 젬 1개 제한 (PoE2) → 트레이드오프 강화

### 5. 스킬 획득 경로 = 컨텐츠 동기
- Elden Ring: 탐험하면 새 Ash of War 발견
- MHW: 몬스터 사냥 → 소재 → 스킬
- RoR2: 도전 달성 → 대체 스킬 해금
- 모바일 권장: 스킬 해금이 **다양한 경로**(퀘스트/던전/이벤트)에 연결되어야 함다
