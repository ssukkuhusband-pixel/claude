# 소셜/PvP 시스템 설계 원칙 조사

> 조사일: 2026-03-29 | 조사 대원: 참치김밥 (Sonnet 에이전트)
> 출처: GDC Vault, Kongregate, Deconstructor of Fun, Naavik, GameRefinery

## 1. 소셜 시스템 핵심 데이터

### 길드 효과 (Kongregate GDC Europe 2013)
- 길드 멤버 수익: 비멤버 **10~20배**
- 소셜 기능 도입 후 참여/리텐션 **2~2.5배**
- 미국 Top 100 게임 중 **70%** 길드 보유

### 해금 타이밍 표준
- Day 1: 친구 목록, 초대 코드
- Day 1~3: 길드 가입
- Day 3~7: 길드 채팅, 기부
- Day 7+: 길드 전쟁, 레이드

### 소셜 압력 양면성
- 긍정: 복귀 유인, 팀 결제 심리
- 위험: 일일 의무 과중 → 번아웃
- 원칙: 기부 부담 1분 미만, 참여=선택적 보너스

### 친구 초대
- 일반 UA 대비 **30% 전환율**
- 양측 보상 + 지속성 조건이 최적

## 2. PvP 핵심 데이터

### 비동기 PvP 우위
- Skillz 연구: 비동기가 6개월 리텐션 **+30%**
- Match Masters: 구독 도입 후 RPD **+95%**
- 매칭 대기 3분+ → 이탈 급증

### P2W 붕괴 사이클
무과금 지속 패배 → 이탈 → 매칭 풀 감소 → 저과금 이탈 → 생태계 붕괴

## 3. 로그라이트 소셜 진화

### Habby 3세대 진화
- Archero: 솔로 우선 → 소셜 후발 (약점)
- Survivor.io v2.0.1: 클랜 추가 (개선)
- Capybara Go: 소셜 카지노 DNA → 코어 루프 통합 (최선)

### Coin Master 슬롯+PvP 모델
- 슬롯 랜덤성 → P2W 격차 심리 희석
- PvP(공격/레이드) → 스핀 소모 자연 촉진
- 운수좋은날 직접 적용 가능

### 로그라이트 최적 소셜 도입 우선순위
1. 데일리 챌린지 랭킹 (쉬움, 효과 즉각)
2. 빌드/결과 공유 버튼 (바이럴)
3. 길드 + 채팅 + 기부
4. 비동기 PvP (빌드 vs 빌드)
5. 길드 전쟁/협동 레이드

## 4. BM 연계
- 길드 기부 → "팀을 위해" 결제 심리
- PvP 미션 → 배틀패스 연동
- 코스메틱: 30시간+ 유저 47%가 구매
- 협동 → 역할 다양성 → 가챠 수요

## 5. 실패 패턴 5가지
1. 소셜 강제 → 솔로 유저 이탈
2. 길드 의무 과중 → 번아웃
3. P2W 격차 방치 → 생태계 붕괴
4. 매칭 3분+ → 이탈
5. "나중에 추가" → 기존 유저 무관심
