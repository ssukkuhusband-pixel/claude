## 2. 게임별 상세 분석

---

### 1. Archero

**장르**: 하이브리드 캐주얼 / 로그라이트 (운수좋은날 직접 비교군)

**핵심 메커닉 요약**
- 이동하면 공격 멈춤 / 정지하면 자동 공격
- 스테이지 클리어마다 스킬 3개 중 1개 선택
- 로그라이트 구조: 죽으면 처음부터

**FTUE 설계 특징**
- 튜토리얼 텍스트 없이 첫 스테이지에서 바로 투하
- "움직이면 안 쏜다"는 규칙을 직접 플레이로 즉시 이해
- 이동 조작 → 자동 공격 확인 → 적 처치 → 스킬 선택 → 스테이지 클리어 → 보상의 플로우가 3분 안에 완성
- 첫 선택(스킬 픽)은 1-2분 내에 등장해 의사결정 재미를 즉시 제공

**강점**
- 핵심 조작 규칙(이동/정지)이 10초 안에 이해됨
- 첫 보상(스킬 선택)까지 빠름 → D1 도파민 루프 완성

**약점**
- 로그라이트 전략 깊이는 3~5판 이후에야 체감
- 에너지(에너지 시스템) 노출이 첫 경험을 방해할 수 있음

---

### 2. Survivor.io

**장르**: 하이브리드 캐주얼 / 로그라이트 (운수좋은날 직접 비교군)

**핵심 메커닉 요약**
- 조이스틱 이동만으로 자동 공격
- XP(초록 결정) 수집 → 레벨업 → 스킬 선택
- 15분 생존 목표

**FTUE 설계 특징**
- 첫 화면 즉시 인게임 진입, 좀비 물량으로 압박감 즉시 전달
- 레벨업 → 스킬 선택 루프를 1분 30초 안에 최초 경험
- 텍스트 없이 시각적 UI(XP 바, 레벨업 팝업)로 규칙 전달
- 성장 곡선이 가파름: 초반 5분에 여러 스킬 습득 → 파워 판타지 빠른 실현

**강점**
- "자동 공격 + 이동만" 인터페이스가 가장 단순한 수준
- 좀비 물량 = 시각적 긴장감이 즉시 몰입 유도

**약점**
- 메타게임(장비 강화, 스킬 트리)은 후반에 드러남 → 초반 깊이 부재

### 3. Capybara Go!

**장르**: 하이브리드 캐주얼 / 로그라이트 (텍스트 기반)

**핵심 메커닉 요약**
- 텍스트+이미지 기반 자동 진행
- 매 라운드마다 랜덤 이벤트+스킬 선택
- 전투 직접 조작 없음, 메타게임(장비/스킬 빌드)이 핵심

**FTUE 설계 특징**
- 첫 "날짜" 선택지가 30초 이내 등장 → 로그라이트 선택의 재미 즉시 전달
- 텍스트 게임이지만 이미지+애니메이션으로 시각적 피드백 보완
- 선택 → 결과 → 보상의 루프가 매우 짧음 (1분 이내)
- 강제 튜토리얼 없음에 가까움 → 자연스러운 온보딩

**강점**
- 조작 없음 → 진입장벽 제로
- 텍스트 RPG 특성상 첫 선택이 즉시 이루어짐

**약점**
- 텍스트 기반 → 시각적 재미 체험이 느림
- 핵심 빌드 전략은 여러 판 이후에야 체감

### 4. Vampire Survivors

**장르**: 하이브리드 캐주얼 / 로그라이트

**핵심 메커닉 요약**
- 이동만으로 모든 전투 (완전 자동 공격)
- 시간이 지날수록 적 증가, 파워업으로 생존
- 30분 생존 목표 또는 Death Reaper 처치

**FTUE 설계 특징 (무튜토리얼 철학)**
- 튜토리얼 완전 없음 — 첫 로드 즉시 게임 시작
- "이동만 하면 된다"는 단 하나의 규칙을 플레이 자체로 즉시 전달
- 첫 30초 안에 공격/피격/레벨업 루프가 자연스럽게 형성
- 규칙이 단순해서 설명 없이도 작동하는 설계

**강점**
- 가장 짧은 "첫 조작까지 시간": 2초 이내
- 학습 비용 제로 → 즉각적인 재미 체험
- PC(스팀) 기준 D1 리텐션 55-65%로 업계 최고 수준

**약점**
- 빌드 시너지의 깊이는 수십 판 이후에야 드러남
- 목표 불명확 → 일부 유저는 "뭘 하는 게임인지 모름" 초반 이탈

**참신한 메커닉 교육 인사이트**: "설명이 필요 없을 만큼 단순하게 만들면 튜토리얼이 필요 없다"

### 5. Brawl Stars

**장르**: 하이브리드 캐주얼 / 멀티플레이어

**핵심 메커닉 요약**
- 듀얼 스틱 조작 (이동 + 조준/공격)
- 다양한 게임 모드 (3v3, 배틀로얄 등)
- 브롤러 성장+수집

**FTUE 설계 특징**
- 첫 로드 후 즉시 강제 튜토리얼 (Shelly 조작 교육)
- 5초 안에 첫 조작 (좌측 조이스틱 이동)
- 듀얼 스틱 기본기 → 슈퍼 어빌리티 → 게임 목표 순으로 점진적 교육
- 봇 상대로 연습 매치 1회 이후 실전 PvP 투입
- 강제 튜토리얼 3분 이내 → 빠른 핵심 재미 진입

**강점**
- 복잡한 멀티 게임임에도 3분 만에 기본기 완성
- 봇 매치로 "내가 이긴다"는 첫 승리 경험 보장

**약점**
- 다양한 모드/브롤러 → 초반 콘텐츠 복잡성으로 방향 상실 가능

### 6. Genshin Impact

**장르**: 수집형 오픈월드 RPG

**핵심 메커닉 요약**
- 오픈월드 탐험 + 실시간 전투
- 캐릭터 수집(가챠) + 원소 반응 시스템
- 방대한 스토리/세계관

**FTUE 설계 특징**
- 오프닝 영상 1분 30초 → 첫 조작 (스킵 불가 구간 존재)
- 강제 스토리 진행: 자유 탐험까지 45~60분 필요
- 첫 과금 노출은 30분 이후 (초심자 배너)
- 풀 보이스/시네마틱으로 스토리 몰입도 극대화
- 전투 교육은 스토리 흐름 안에서 자연스럽게 진행

**강점**
- 스토리+그래픽 품질이 첫 인상을 강렬하게 형성
- "나중에 이 캐릭터를 얻을 수 있다"는 욕망 즉시 자극

**약점**
- 자유도 진입까지 45분 이상 → 이탈율 높은 긴 강제 튜토리얼
- 원소 반응 시스템 교육이 충분히 강조되지 않아 초반 전략성 미달
- 가챠 과금 구조 이해까지 시간 소요

### 7. NIKKE: Goddess of Victory

**장르**: 수집형 슈팅 RPG

**핵심 메커닉 요약**
- 자동/수동 전환 가능한 3인칭 슈팅
- 캐릭터 수집(가챠) + 커버 시스템
- 미소녀 캐릭터 스토리

**FTUE 설계 특징**
- 오프닝 영상 60초 → 첫 전투 (스킵 불가)
- 강제 Chapter 1-4 진행: 15~25분 필요
- 스토리 스킵 가능이지만 전투는 강제 진행
- 10분 이후 첫 10연뽑 → 가챠 재미 즉시 전달
- 자동 스킬, 자동 에임은 Chapter 1 완료 후 해금

**강점**
- 슈팅 조작이 직관적 → 10초 안에 이해
- 첫 가챠(10연뽑)가 15~25분 이내 제공 → 수집 판타지 즉시 체험

**약점**

**약점**

- Genshin Impact 계열 HoYoverse 작품

**FTUE 설계 특징**
- 오프닝 영상 90초 → 첫 조작 (스킵 불가 구간)
- 강제 Chapter 1 진행: 30~60분 필요 (스킵 불가 스토리 많음)
- 전투 교육은 스토리 안에서 자연 진행
- 첫 가챠 노출 20분 이후 (전도 선물 등)
- 방대한 스토리/세계관이 HoYo 팬에게는 강점, 신규 유저에게는 부담

**강점**
- 연출/그래픽 품질 최상위 → 첫 인상 강렬
- 스토리 팬에게는 긴 튜토리얼이 오히려 콘텐츠

**약점**
- 첫 전투까지 90초+ → 모바일 표준 대비 가장 느림
- 강제 스토리 = 비전략적 유저 초반 이탈 원인

## 3. 핵심 인사이트: 참신한 메커닉을 가진 게임이 어떻게 교육하는가

### 3-1. Balatro의 포커 메커닉 교육 방법론

Balatro는 "포커를 모르는 유저"를 가정하고 설계된 가장 정교한 메커닉 교육 사례임다.

**핵심 전략: "지식을 사전에 요구하지 않는다"**

```
기존 방식 (잘못된 예):
"포커 족보를 먼저 외우세요" → 5분 텍스트 강의 → 게임 시작

Balatro 방식 (올바른 예):
즉시 게임 시작 → 카드 선택 시 실시간으로 족보 표시 →
점수 올라가는 것 확인 → 자연 학습
```

**세부 기법**
1. **실시간 족보 표시**: 카드 선택할 때마다 "이 패는 Two Pair입니다 (Chips x2, Mult x2)" 즉시 표시 → 외울 필요 없음
2. **낮은 첫 허들**: 첫 목표가 300칩으로 매우 낮음 → 실패 없는 첫 경험 보장
3. **조커의 유레카 모멘트**: 첫 조커 획득 시 점수가 갑자기 5~10배 올라감 → "조커가 핵심"임을 말이 아니라 숫자로 이해
4. **규칙 위반의 점진적 허용**: 튜토리얼 → 기본 포커 룰 이해 → 조커로 룰 깨기 → 더 강한 룰 파괴

**운수좋은날 적용 인사이트**:
- 슬롯머신 족보(같은 심볼 3개 등)를 사전에 설명하지 않아도 됨
- 릴이 돌아갈 때 실시간으로 "이 조합은 OOO입니다" 표시
- 첫 전투 허들을 매우 낮게 설정 → 반드시 승리하는 첫 슬롯 경험 보장

---

### 3-2. Slay the Spire의 덱빌딩 교육 방법론

Slay the Spire는 "튜토리얼 없이 UI가 가르친다"는 철학의 정점임다.

**핵심 전략: "카드 텍스트가 완전한 룰북이다"**

```
기본 덱 (Strike x5, Defend x4, Bash x1):
- Strike: 6 데미지. (3 에너지로 6 데미지)
- Defend: 5 방어력 획득.
→ 두 카드만으로 공격/방어 개념 완전 전달
```

**세부 기법**
1. **최소 시작 덱**: 처음 10장만으로 시작 → 선택 과부하 없음
2. **텍스트의 완전한 명확성**: 카드 효과가 모호하지 않음 → 텍스트 = 튜토리얼
3. **첫 카드 선택 경험**: 전투 승리 후 즉시 3장 중 1장 선택 → "내 덱이 바뀐다"는 핵심 재미 즉시 전달
4. **실패 학습 허용**: 로그라이트이므로 죽어도 재도전 → 실패가 교육 과정

**운수좋은날 적용 인사이트**:
- 슬롯 심볼 효과 텍스트를 완전히 명확하게 작성 → 별도 튜토리얼 최소화 가능
- 첫 런은 단순 심볼로만 구성 → 2~3런부터 복잡한 시너지 심볼 도입

---

### 3-3. 장르별 교육 전략 패턴 분류

| 패턴 | 대표 게임 | 핵심 원리 | 적용 조건 |
|------|----------|-----------|-----------|
| **즉시 투하** | Vampire Survivors, Slay the Spire | 메커닉이 충분히 단순함 → 설명 없이도 이해 | 조작이 1~2개인 게임 |
| **실시간 안내** | Balatro, Candy Crush | 게임 진행 중 즉시 피드백 제공 | 새로운 규칙이 있는 게임 |
| **점진적 공개** | Clash Royale, Clash of Clans | 7매치/10분에 걸쳐 복잡성 단계별 추가 | 전략적 깊이가 있는 게임 |
| **강제 스토리** | Genshin, Honkai, NIKKE | 스토리 몰입이 핵심 가치인 게임 | IP/캐릭터 기반 게임 |
| **즉시 승리 보장** | AFK Arena, Candy Crush | 첫 전투/레벨 반드시 성공 | D1 리텐션이 핵심인 게임 |

## 4. 운수좋은날을 위한 FTUE 설계 원칙 도출

### 4-1. 슬롯머신 메커닉 교육 전략 (Balatro 방식 적용)

**핵심 원칙**: "슬롯머신을 모르는 유저도 10초 안에 이해"

1. **첫 룰**: "릴을 돌리면 심볼이 나온다" → 버튼 1개, 즉시 체험
2. **심볼 조합 교육**: 같은 심볼이 나올 때마다 실시간으로 "OOO 조합! +X 데미지" 표시
3. **첫 전투 반드시 승리**: 첫 던전은 극도로 낮은 허들 → 슬롯 결과에 관계없이 클리어 가능
4. **첫 특수 심볼**: 2~3분 이내에 특수 심볼 등장 → "기본과 다른 것"의 재미 즉시 체험

### 4-2. 핵심 FTUE 타이밍 목표

| 이벤트 | 목표 시간 | 근거 (레퍼런스) |
|--------|----------|----------------|
| 첫 조작 | 5초 이내 | Candy Crush, Brawl Stars 기준 |
| 첫 전투 | 30초 이내 | Archero, Survivor.io 기준 |
| 첫 보상 (스킬 선택) | 2분 이내 | Archero, Balatro 기준 |
| 핵심 메커닉 이해 | 3분 이내 | Vampire Survivors 기준 |
| 첫 과금 노출 | 7-10분 이후 | AFK Arena, Clash Royale 기준 |

### 4-3. 절대 하면 안 되는 것 (레퍼런스 실패 사례 기반)

| 금기 | 실패 사례 | 이유 |
|------|----------|------|
| 오프닝 영상 90초+ 스킵 불가 | Honkai: Star Rail | 모바일 유저의 15초 인내심 초과 |
| 첫 레벨에서 실패 가능 | (일반적 실수) | 첫 경험이 패배 = 즉시 이탈 |
| 텍스트 강의 2분+ | 많은 구형 게임 | 유저는 읽지 않음 |
| 모든 시스템 동시 노출 | NIKKE 초반 UI | 정보 과부하 → 이탈 |
| 에너지(스태미나) 첫 3분 이내 노출 | 일부 캐주얼게임 | 첫 경험에서 벽을 만들면 안 됨 |

| 평균 | 26-39% | Archero, Survivor.io, Genshin Impact |

- 하이퍼캐주얼: D1 50%+ 기대

- 전략 SLG: D1 35-45% 양호
