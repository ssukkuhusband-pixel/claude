## 핵심 인사이트

### 참신한 메커닉 교육 방법론

**Balatro (포커 교육 — 최고 사례):**
- 카드 선택하면 즉시 "Two Pair" 표시 → 족보 암기 불필요
- 첫 목표 허들(300칩) 매우 낮게 → 반드시 성공
- 첫 조커 획득 시 점수 5~10배 급등 → 텍스트 없이 숫자로 이해

**Slay the Spire (덱빌딩 교육):**
- 기본 덱 10장만으로 시작 → 선택 과부하 없음
- Strike/Defend 2개로 공격/방어 개념 전달
- 첫 전투 승리 후 즉시 카드 선택

### D1 리텐션 벤치마크

| 등급 | D1 리텐션 | 해당 게임 |

| 상위 | 40-54% | Royal Match, Candy Crush, Brawl Stars, Clash Royale |
